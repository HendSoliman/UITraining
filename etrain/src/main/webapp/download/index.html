Dematic iQ Mobile

Download

Select the installer from the list below to begin the installation. For Apple devices ensure you can reach http://www.apple.com. You can also download the Apple IPA file and manually add the file to iTunes. This is helpful if you can’t access this page on your iPad.

To download, touch the image for your device below, or scan the appropriate QR code.

Start Dematic iQ Mobile

On successful download and installation an additional icon appears on the desktop of your device:

Touch the icon to start the application. The Dematic iQ Mobile login screen shows up.

 

Login

Upon login screen you select the warehouse and enter logon parameters to access the Dematic iQ Mobile web application:

  • Warehouse contains access parameters to Dematic iQ Mobile web application
  • Modify, Create, Remove: Enter Warehouse access paramters
  • User the login user name
  • Password the login password (this may need to be entered explicitly depending on warehouse configuration).

Touch the OK Button to connect and authenticate to Dematic iQ Mobile web application.

Enter Access parameters

You may create, modify or delete the access parameters of a Warehouse:

  • Name: Name of the Warehouse
  • Server: Server name or IP-address
  • Port: Server port number
  • Project: Project name of the web application
  • User the login user name
  • Password the login password (warehouse configuration determines whether this can be saved locally)
