Replenishment Thresholds

Purpose

A Replenishment Threshold defines SKU specific triggers and configuration for the creation of replenishment orders of the replenishment types "fix quantity", "fix quantity effective", "total quantity" and "total number of load units".

      
Screen

      
Sub-Screens
  • Detail Dialog Replenishment Threshold.
      
Special Buttons

Button

Description

Manually check for immediate replenishment, according to Threshold Configurations. If the SKU's Inventory quantity fell below the configured threshold, a Replenishment Order is created immediately.

 INFO 

This function does not support the replenishment type "lookahead"!

The selected Replenishment Thresholds are reset to values proposed by the system based on the replenishment threshold blueprint definitions of the Replenishment Supervised Zone and the SKU Load Unit Capacities.
