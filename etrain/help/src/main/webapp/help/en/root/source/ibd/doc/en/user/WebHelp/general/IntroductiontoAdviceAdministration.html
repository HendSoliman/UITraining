Introduction to Advice Administration

Advice

An Advice announces the goods to add to Dematic iQÂ® controlled Inventory. It represents a delivery from a defined source at a defined time. It consists of an Advice header and a set of Advice Lines; and for each Advice Line, a set of Advice Line Load Units.

The possible lifecycle of Advices is described in the related description of the status values.

      
Advice Lines

Advice Lines specify the SKU, inventory attributes and quantities within the Advice. Advice Lines can be split. For example, a delivery for the Advice Line consists of the same SKU with various batches. The set of inventory quantities is delivered on a set of identified pallets, which may be specified in the Advice Line Load Units, if known. Advised Load Units are expected to contain only Inventory of one SKU.

The possible lifecycle of Advice Lines is described in the related description of the status values.

 INFO 

Splitting of Advice Lines is currently not used in the standard!
     
Advice Line Load Units

Advice Line Load Units represent yet received or preliminarily advised Load Units. They are:

  • created during Host import if concrete Load Units are advised directly with the Advice.
  • created during Receiving process for each received and currently created Load Unit, if no concrete Load Units are advised.
  • created in combination of both cases.

The possible lifecycle of Advice Line Load Units is described in the related description of the status values.

      
Rules for Advices

The following validation rules are applied for acceptance of Advices while receiving:

  • Target quantity must be greater 0.
  • If unlimited overdelivery is not allowed, the current quantity must be less than the allowed maximum due to the tolerance of the target quantity.
  • If there are Advice Line splits for an Advice Line, the sum of the target quantity of the Advice line splits must equal the target quantity of the Advice Line.
  • The current quantity of an Advice Line must be greater or equal to the sum of the current quantity of the Advice Line Load Units.
  • SKU ID and Receiving Quantity Unit are mandatory.
  • The Load Unit Code, Parent Load Unit Code and Base Load Unit Code must not define a cycle.
  • A Load Unit Code can be used in multiple Advice Lines, but must not belong to more than on Advice.
