Location Management

Overview

The following main business objects provide the basic part of the Location Management:

  • Facilities and Areas (incl. Groups (Aisles) and Subgroups (Levels)).
  • Locations with Location Types (supporting discrete layouts or dynamic slotting), Storing Conditions
  • Weight Limiters
  • Zones and Dispo Zones
  • Pick Controller.
