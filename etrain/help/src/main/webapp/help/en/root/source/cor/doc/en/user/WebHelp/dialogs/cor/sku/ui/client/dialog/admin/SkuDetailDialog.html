Fields

Field

Description

SKU ID

Uniquely identifies a Stock Keeping Unit within a Client.

Identification

Description

Description of the Stock Keeping Unit.

Owner/Status

Owner

If the SKU is created by a host or locally by the LMS.

Client

Owner of the Inventory of this Stock Keeping Unit.

Status

Degree that a SKU can be used to manage Inventory.

  • Incomplete (SKU Definition is incomplete and cannot be used)
  • Complete (SKU Definition is complete and can be used)
  • Marked for delete (SKU is marked for deletion. No new Inventory may be created for this SKU. When no Inventory remains, the SKU is deleted).

Product 

Supplier

Company that delivers Inventory of this SKU by default.

Product Group ID

Identification of the product group, according to the host.

Product Code

Identification of the product, according to the host.

Product Code Type

Type of product code used for this SKU. For example, an EAN13 or ISBN.

Details

Units

Base Quantity Unit

All Inventory of this Stock Keeping Unit is managed related to the Base Quantity Unit.

Preferred Load Unit Type

Load Unit Type preferably used to store Inventory of this SKU. For Inventory of this SKU using a different packaging level, the preferred Load Unit Type for this SKU and this packaging level overrules (if defined) the configuration that only depends on the SKU.

ERP

Batch Mandatory For Host

If a batch is mandatory for this SKU according to the host. If set to 'true', a batch is mandatory for inventory of this Stockkeeping Unit, regardless of the SKUBlueprint for the SKU. If set to 'false', the SKUBlueprint defines if a batch is mandatory. May be overruled by a host plant configuration, if one exists.

Statistics

Shelf Life

Number of days between the date of production and the best before date of the inventory.

Weighted Access Frequency

Weighted absolute access frequency (considering accesses in different time periods) of the SKU as result of the last Velocity Class Analysis.

Handling

Classifications

Velocity

How often and how much Inventory of this Stock Keeping Unit is ordered by a Customer. May be overruled by a host plant configuration, if it exists.

For further details see Velocity Class Analysis.

Hazardous

If this Stock Keeping Unit is a hazardous substance the Hazard Classification has to be maintained.

Main Product Range ID

Main product range, according to the host.

Product Range ID

Range of the product, according to the host.

Palletizing Mode

Default mode for palletizing operations of this SKU:

  • None (SKU will not be planned for palletizing)
  • RAPIDPAL (SKU is planned only for an automatic palletizing operation at a RapidPal Station)
  • ERGOPAL (SKU is planned only for an manual palletizing operation at a ErgoPal Station)
  • Both (SKU is planned for both palletizing operations at RapidPal and ErgoPal Stations).

Internal

Usage Type

Type of use typically destined for this SKU:

  • Default (SKU is used for normal handling)
  • Packaging (SKU is used as packaging material)
  • Empty LU (Empty Load Unit provides an Inventory Unit (e.g. for retrieval strategies))

 INFO 

For the basics of the supply of Workstations with empty Load Units, see the Empty Container Supply  section of the Retrieval Overview.
  • Anonymous inventory (Inventory of unspecified SKUs that is identified by specific Inventory Unit attributes (e.g. special inventory mark/reference, Load Unit)).

Blueprint

The Blueprint this SKU is assigned to.

Weight Tracking

Specifies whether weight tracking is required for Inventory Units of this SKU.

Stocktaking 

Quantity Determination Method

Method to determine quantity while Stocktaking (Cycle Counting).

  • Counting
  • Weighting
  • Metering
  • Estimating

Cycle Counting Threshold

When during a Picking process the quantity of the Source Inventory Unit equals or decreases the Stocktaking Threshold, an immediate nearly zero-cross check operation is requested. If the cycle counting threshold is undefined, no picking related cycle counting is requested.

Cycle Counting Exclusion Period

Period in days another cycle counting is not requested after a counting has been done.

Stocktaking Excluded

If SKU should be considered for Stocktaking. Packing Material or empty Load Units are typically excluded.

In-/Outbound 

Handling Hint

Hint for handling the given material at picking or I-point. For example: "Use gloves", "Handle with extreme care".

Service Qualifier

Special handling hint for the DCD. Project specific.

Maximum Units Per Pick

Maximum quantity which can be picked with one Pick. Larger quantities are split during Picking operations with respect to this value.

 INFO 

If in Goods-To-Person Picking screens the different Picks from the same Source Load Unit are merged, this parameter will be ignored anyway.

Split Case Allowed

If a handling unit can be opened to pick remaining quantities which cannot be accomplished with a full handling unit.

Dispatch Packing Group

Different dispatch packing groups must not be packed together into the same Dispatch Load Unit.
