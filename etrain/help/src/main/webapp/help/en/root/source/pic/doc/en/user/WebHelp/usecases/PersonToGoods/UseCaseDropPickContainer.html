Use Case Drop Pick Container

Description

A drop of all Pick Containers of the current Pick Walk is required if all Pick Containers of current Pick Walk finished due to finishing the last Pick of related Pick Walk.

      
Triggers and Actors

The following persons must perform this after Pick Container is finished:

  • Picker operating in Person-to-Goods Pick Zone.
      
Prerequisites
  • All required Picks of all Pick Containers of current Pick Walk are finished.
      
Work Flow

Step

Description

1.

Open the screen RF-Picking and perform the Use Case: Pick Operation for all active Picks of current Pick Walk.

2.

The screen Drop Pick Container displays.

3.

Scan a valid Destination Location.
"Enter" is activated implicitly with scan.

3a.

If an invalid Destination Location is scanned, an error message appears. Proceed with step 3.

4.

The screen Next Pick Walk displays.

5.

Press "Enter".

Stop picking operations

5a.

To stop picking operations, press "F4".

6.

The operational mode "Picking" finishes and the main menu launches.
     
Result

All Picks and Pick Missions of related Pick Walk proceed to "finished" status. All Pick Containers of Pick Walk are booked on required Location.
