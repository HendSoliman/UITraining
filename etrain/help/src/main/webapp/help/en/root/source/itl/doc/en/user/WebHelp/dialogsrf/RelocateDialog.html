Dialog Overview

RF-Terminal Relocating

Functions
  • Spontaneous Relocating of a whole Load Unit to another Location.

 INFO 

Relocation of a Load Unit is not allowed, if

- Inventory or Load Unit have an existing reservation

- an active Transport exists

- the destination location is blocked (for inbound).
     
Dialog Overview

Dialog state

Description

Relocate

Determination of the Source Load Unit to be relocated is the initial action.

Relocation Target

Determination of the Target Location for the relocating operation.
