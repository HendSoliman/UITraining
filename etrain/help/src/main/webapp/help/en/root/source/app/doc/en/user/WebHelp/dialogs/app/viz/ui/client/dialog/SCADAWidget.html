Visualization Screen

Purpose

The Visualization Screen provides information about the pre-configured conveyor systems and informs the user of state transitions of each element. Predefined colors indicate the current state of an element. A small set of controls represents the elements of a warehouse with context-based functionality. There are operative as well as non-operative elements. The user is able to interact with the system by three supported user-interactions. The Visualization provides several zoom levels. Each zoom level shows more detailed information of each element (if supported)..
