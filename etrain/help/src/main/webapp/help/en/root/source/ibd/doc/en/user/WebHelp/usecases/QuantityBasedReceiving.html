Overview

Quantity based Receiving

Purpose

This section provides an overview of quantity based goods receipt in automated or manual areas at Receiving Workstations or with support of RF-Terminals based on Advices or Deliveries.

      
Overview

Use Case

Description

Quantity based Receiving

The Quantity Based Receiving in automated or manual areas at a Receiving Workstation ordered to given Advices provides all required functionalities for deposit the delivered Inventory of an Advice in single parts onto Load Units and initiate the concluding inbound transports by using the "Receiving Dialog".

Quantity based Receiving with RF-Terminal

The Quantity Based Receiving in automated or manual Areas using a RF-Terminal ordered to given Advices provides all required functionalities for deposit the delivered Inventory of an Advice in single Inserts onto Load Units and initiate the concluding inbound transports by using the "RF-Terminal Receiving".

Unloading General

Describes the unloading processes based on Deliveries with the basic steps only.

Unloading with RF-Terminal

The quantity and Receipt based Receiving in automated or manual Areas ordered to given Shipments provides all required functionalities for deposit the delivered Inventory of a Delivery in single Inserts onto Load Units and initiate the concluding Inbound Transports using the RF-Terminal Unloading.
