Depalletizing Workstations

Overview

This screen is for configuring a Depalletizing Workstation.
A Depalletizing Workstation receives aggregated units, which may be fully or partly segregated, and outputs a set of segregated units and optionally a remainder of the input.
The supported types are: depalletizing into trays or totes or directly to belt or conveyor.

 INFO 

This is a manually operated workstation.
Automatic depalletizing (e.g. in AMCAP scenarios) is done with Depalletizer Stations.
