Overview

Offline Picking

Brief Description

Offline Picking in Person-to-Goods Pick Zones occurs using the screen Pick Walk where multiple Pick Walks can be created and separately finalized. With activation, the configured Pick List prints on the Pick Zone's default list printer to support the Pick Walk.
Finalization occurs for a single Pick Walk with confirmation of each single Pick using different Confirmation Codes and preparation of all used Pick Container.

Use Case

Description

Create Pick Walk

Starts Pick Walk with activation of one or multiple Pick Walks and prints configured default Pick List.

Finalize Pick Walk

Finalizes Pick Walk with confirmation of single, differing Picking results of a Pick Walk.
