Dispatch Load Units

Purpose

A Dispatch Load Unit is the Order related Shipping equivalent of the Load Unit to support Shipping processes and to represent a shipped unit within an Order, even if the related Load Unit and the Inventory were deleted. A Dispatch Load Unit is created during a Shipping Order Fulfillment process, usually when the Pick Container is created or during creation of Full Load Unit Retrievals, and is related to an existing Load Unit. The Dispatch Load Unit is associated to Orders for which it is used and may exist when the corresponding Inventory and Load Unit were already deleted in the Logistics Manager.

Fields

DLU Order Lines

Description

Client

Client of the Order Line.

Order ID

Order assigned to the Dispatch Load Unit.

Order Line

Order Line assigned to the Dispatch Load Unit.

Quantity

Part of the Order Line, in Base Quantity Units, that is assigned to the Dispatch Load Unit.

Sequence

Order that the Inventory requested by the Order Line must be packed into the Dispatch Load Unit. Inventory assigned a lower sequence must be packed before that with a higher sequence. Inventory without sequence must be packed last.

Batch

If required inventory must be of this specific Batch. If empty, it is not considered, i.e. can be any with respect to Retrieval Strategies.

SKU

SKU of the Order Line assigned to the Dispatch Load Unit (see screen Order Lines).

Best Before Date

Best before date of the Order Line assigned to the Dispatch Load Unit (see screen Order Lines).

External Order Type

Customer order type of the Order assigned to the Dispatch Load Unit (see screen Orders).

Service Qualifier

Service qualifier of the Order assigned to the Dispatch Load Unit (see screen Orders).

Route/Order

Route of the Order assigned to the Dispatch Load Unit (see screen Orders).
