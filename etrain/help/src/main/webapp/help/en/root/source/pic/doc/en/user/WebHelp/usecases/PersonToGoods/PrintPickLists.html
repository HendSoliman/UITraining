Print Pick Lists

Description

If Offline Picking, the Pick List provides single instructions for Pick operations during a Pick Walk. Each Pick of the Pick Walk is represented as a single list item. The List prints automatically during Pick Walk creation (see Create Pick Walks). This describes the reprinting of a Pick List.

      
Triggers and Actors

If a Pick List reprint is required, the following persons must perform this:

  • Control Center Supervisor
  • Picker.
      
Prerequisites
  • At least one Pick Walk is available.
  • Printing configuration is complete.
  • A default list printer for the related Zone is configured (Pick Zones).
      
Work Flow

Step

Description

1.

Open the Pick Walk screen.

2.

Select one or more Pick Walk(s) from the Pick Walk grid.

3.

Click the "Print Pick Lists for the selected Pick Walk(s)".
     
Result

For each selected Pick Walk, a Pick List prints on the configured List Printer of the related Pick Zone.
