Tracked Case Conveyor

Purpose

Provides information about the current allocation of a conveyor.

 

      
Functions

Supports the user by the following tasks:

  • check the current sequence auf load units
  • navigate between previous and next conveyor
  • change current view of conveyor
  • refresh data and display current allocation of conveyor

 

      
Screen

 

      
Areas

Screen is diveded into five areas:

  • Button area with label of current location
  • area providing additional information about load units
  • frontal view of conveyor and load units (with product id) and position on conveyor
  • current conveyer direction
  • topview of conveyor and load units

 

INFO: The alignment of the load units can be left, right and middle (in conveyor direction). By default alignment is set to middle.

 

      
Fields

Button

Description

navigate to previous conveyor (left conveyor from users point of view)

refresh view (request current allocation of conveyor)
reset fault button (not supported)

change flow direction (change users point of view)

clear load unit (not supported)

navigate to next conveyor (right conveyor from users point of view)
