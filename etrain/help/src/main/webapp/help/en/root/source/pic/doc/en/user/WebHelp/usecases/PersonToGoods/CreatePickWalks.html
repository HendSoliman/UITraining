Create Pick Walks

Description

With the start of Offline Picking, a Pick List prints with single instructions for pick operations during the Pick Walk. The Pick Walk turns to "active" status.
Upon Pick Walk activation, a Pick List prints, if appropriately configured. Reprinting of a Pick List and layout are described in "Print Pick List(s)". 

      
Triggers and Actors

If workload preparation in an Offline Picking Zone is required, this must be performed by a supervisor or picker.
If exceptional Offline Picking is required, e.g. due to a Picking equipment malfunction.

      
Prerequisites
  • Configuration of related Pick Zone is complete (Pick Zones).
  • At least one Pick Mission in Pick Zone of type "Person-to-Goods" in "released" status is available.
  • Printing configuration is complete.
      
Work Flow

Step

Description

1.

Open the Pick Walk screen.

2.

Click the "Create Pick Walks" button.

3.

Define the Picking Zone for the new Pick Walks to create. Optionally limit the number of Pick Walks to be activated.

4.

The system groups the next (configured number of) Pick Missions to new Pick Walks until the defined number of Pick Walks are created.
     
Result

For each created Pick Walk, the configured Pick List prints on the configured default printer. Pick Walks proceed to "active" status.
