Starting translator failed

Symptom

One or more of the translators failed while starting up.

      
Brief description

The translator applicaton went to state failed and is (are) shown as red node in the Runtime State Administration dialog.

      
Categories
  • EAI
  • Translator
      
Possible errors

In case of a failed start, the following reasons are possible:

  • The Database connection is not available.
  • The Java Class File used by the XSLT Transformer is not accessible.
      
Detail description

The Virtual Machine (VM) or one of the translator applications has been started within the Runtime State Administration dialog or on system level.

The SC and the VM are up and in state running, but one of the applications stopped in state failed.
