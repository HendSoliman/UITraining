Units of Measure

Purpose

Unit of Measure configuration converts values expressed in a Unit of Measure to another Unit of Measure. A Unit of Measure is SKU independent and cannot be used to represent quantities of Inventory directly, but it can be used to convert the quantity of Inventory to a Unit of Measure. When the Basic Quantity Unit or an alternative Quantity Unit of the SKU is at the same time a Unit of Measure, the quantity can be displayed as another unit of measure of the same unit of measure dimension.

Typical Units of Measures are: centimeter, kilogram or liter.

Each Unit is assigned to a dimension, which describes one physical measurement of the Inventory. For a certain dimension, several units can be given, which can be converted to each other using specific factors. For each dimension, there can be only one base unit, to which the given factors are related.

 INFO 

 Attention: In case it is desired to change the base Unit of Measure for a certain dimension it must be ensured, that all values for this dimension which are transferred from an external or underlying system are converted accordingly.

 INFO 

For configuration purposes refer to the related topic in the Warehouse Administration Guide.
