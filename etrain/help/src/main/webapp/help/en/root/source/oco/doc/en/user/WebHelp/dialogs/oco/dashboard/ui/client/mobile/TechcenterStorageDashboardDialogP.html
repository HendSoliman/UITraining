Storage Overview

Purpose

This screen displays information on all storage areas with special regard to the capacity, movements, housekeeping and equipment availability.

 

      
Screen

 

      
Fields

Information based on the occupation status of the different areas.

KPI

Description

Min. Residual Capacity for High Load Units

Minimal residual capacity of the whole storage areas for Load Units of a "high" height classification.

Actual % occupied

Bin occupation ratio using reservation capacity and total capacity for each single area or aisle.

Available space

Number of available (or locked for output) bin positions in the current area for each possible Load Unit Type.

Available space for high LUs

Available Space of bins with a height classification "high".
     
Availability Automatic Areas

Information based on the current availability of the equipment in automated storage areas.

KPI

Description

Inbound Exceptions in last hour

Number of exceptions that occurred with an Inbound transport within the last hour.

Outbound Exceptions in last hour

Number of exceptions that occurred with an Outbound transport within the last hour.

Equipment Availability per Area

Ratio to the sum of all available bins:

  • Inbound: bins that are not "locked for input" or "locked for input and output"

  • Outbound: bins that are not "locked for output" or "locked for input and output".
     
Housekeeping

Information based on the currently not optimal located Load Units.

KPI

Description

Mislocated

Number of stored Load Units having a mislocation factor and thus eligible for a housekeeping operation.

Mislocation Issue

Number of stored Load Units grouped by their mislocation issue.
     
Current Work Movements

Information based on the current transport movements.

KPI

Description

Inbounds

Number of released or active transports having different source and destination areas and a destination area of type stock.

Outbound

Number of released or active transports having different source and destination area, and a source area of type stock.

Shuffle

Number of released or active transports having the same source and destination area of type stock.
     
Completed Work Movements

Information based on the finished transport movements.

KPI

Description

Inbound, Outbound, Shuffle (/h)

Number of finished storage, retrieval and shuffle transports (for the last day with a 1h granularity).
