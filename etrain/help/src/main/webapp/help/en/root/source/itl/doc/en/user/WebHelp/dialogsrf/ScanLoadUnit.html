Scan Load Unit

Purpose

In case of required by configuration each new Base Load Unit to be counted must be verified by a scan.

      
Screen

      
Prerequisites
  • Stocktaking Zone configuration requires the scan of the Load Unit to be counted (defined by "Scan Load Unit Handling"). 
      
Fields

Field

Description

Counting Order

Unique identifier of the currently executed Counting Order.

Base Load Unit

Base Load Unit which is to be scanned.

Base Load Unit (Scan)

Scanned Base Load Unit.
     
Special Buttons

Button

Description

OK
Key: Enter

Confirm the currently scanned Base Load Unit and switch to "Counting / Recounting".

If scan of SKU is required, the dialog switch to "Scan SKU".

Exit
Key: ESC

Cancel counting operations and switch to main menu.
