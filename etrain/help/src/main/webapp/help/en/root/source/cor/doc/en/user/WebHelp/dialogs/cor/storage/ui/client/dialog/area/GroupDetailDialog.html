Detail Dialog Group

Purpose

This screen opens when creating, viewing, or modifying Groups (Aisles).

For an overview of all elements used to group locations in a warehouse, see the Warehouse Structure Description.

      
Screen

      
Fields

Field

Description

Facility

Selected Facility. Read only field.

Area

Selected Area. Read only field.

ID

Unique ID of the Aisle.

Configuration Status

If Aisle configuration was accomplished and the Aisle can therefore be used by the system.

Group Type

The group represents a group of locations which could be

  • Aisle (see Warehouse Structure Description)
  • Conveyor (This type usually groups transport points with respect to the PLC).

Handling 
Lock Status

If the Aisle was locked for specific operations. Meant as a long term lock set by an user.

 INFO 

All current Inventory reservations within this Aisle will be cancelled with an immediate new reservation process.

 INFO 

This Lock Status can only be changed with use of the related function of the overview screen.
Values are identical to those of the Technical Lock Status.
Lock Status Reason

Reason for the lock. 
Technical Lock Status

If the Aisle was locked for specific operations for a short term by a underlying transport subsystem (only for project specific use):

  • available (all transport operations will be executed)
  • locked for input (inbound operations would not be assigned to the Aisle automatically)
  • locked for output (outbound operations would not be assigned to the Aisle automatically)
  • locked for input and output (inbound and outbound operations would not be assigned to the Aisle automatically)

 INFO 

During inbound or outbound operations, technical lock status can be enabled or disabled within the strategies.
Technical Lock Status Reason

Reason for the lock 

Statistics 
Capacity

Overall number of storable Load Units within this Aisle. Calculated as sum of the appropriate values of all Locations of the Aisle less the positions locked for input.
Reservation Capacity

Number of Load Units which can still be disposed within this Aisle. Counter is maintained on each disposition operation with this Aisle and calculated as a sum of the appropriate value of all Locations of the Aisle. 
Fill Level

Used capacity of this Aisle in percent. 
Last Dispo Date

Date / Time of the last Inbound Disposition of this Aisle.
