Rich client login failed

Symptom

An error message appears after trying to login into the rich client.

      
Overview (short description)

Login is not possible.

      
Possible reasons

In case of a failed login, the following reasons are possible:

  • The application server is not reachable (network problem).
  • The application server is not running.
  • The login name is wrong.
  • The password is wrong.
  • The user has expired.
      
Error description (detail)

A message box appears which shows a detailed error message. The user will not be logged in.

Reason 4:

Step

Action

1

Please make sure that your network connection is ok. If you are not sure how to do this, contact your system administrator.

2

If your network connection is ok but the problem still exists, make sure, that the application server is running. If not, please restart the application server.

3

If the application server is running but the problem still exists, make sure that the database is up and the application server is correctly connected to it. If not, please restart the database and after that restart the application server.

Try to login again.
