Flex Locations

Purpose

The dialog visualizes the occupation of a storage bin which uses dynamic slotting and provide navigation action to neighbored bins. Displayed are:

  • Static properties of a bin (width / depth)
  • Case position is displayed as rectangle
  • rectangles for each stored case.

 

      
Screen

A number of states can be displayed by this screen. The background represents the current state of this bay. Possible values are:

  • Empty (green) 
  • Partially full (grey)
  • Full (blue)
  • Locked (yellow)
  • Error (red)

In case of a locked or error state an additional field will show the lock reason or error code.

The context menu contains an entry leading to the Clarification Bin Dialog

 

Blocked slots within the bay are represented by a rectangle with diagonal stripes. Upon selection these can be unblocked.
