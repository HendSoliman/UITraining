ZUSO Monitor

Purpose

ZUSO (aka Sort Lane) is a technique for sorting on a conveyor. It is used for sequencing in a captive Multi Shuttle area. This dialog monitors the sort lanes.

 

      
Functions

Operators can perform the following tasks:

  • Monitoring Sort Lanes
  • Start/Finish Clearing Sort Lanes

 

      
Overview

The dialog contains a master/detail grid showing the Sort Lanes as master data and the Transport Orders as details.
