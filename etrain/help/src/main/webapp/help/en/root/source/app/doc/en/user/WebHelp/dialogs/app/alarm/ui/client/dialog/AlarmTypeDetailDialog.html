Alarm Type Details

Purpose

This screen opens when creating viewing or modifying Alarm Types.

 

      
Screen

 

      
Fields

Field

Description

Id

 The Id of the Alarm Type
Alarm Domain The Alarm Domain the Alarm Type belongs to

Default Message

 The Default Message for this Alarm Type (language independent)

Category

 The Alarm Category: Information, Warning, Error
Life Cycle Type The Alarm's Life Cycle Type: Acknowledgment, Acknowledgement/Gone, Gone, Expire, see Alarm State transition
Expiration (ms) The expiration time in milliseconds for Alarms with Life Cycle Type "Expire"
Counted Flag indicating that Alarms of this type will be counted. Raising n- Alarms only create only one instead of n- Alarm notifications
Counted Period (ms) Period in milliseconds while Alarms of this type are being counted
Disabled Flag indicating notification for Alarms of this type is disabled
