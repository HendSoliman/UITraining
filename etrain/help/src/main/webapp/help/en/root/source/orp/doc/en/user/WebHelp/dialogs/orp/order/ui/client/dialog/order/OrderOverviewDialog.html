Special Buttons

Button

Description

Create new Order (Owner = Local) using:

  • Manual Creation of a Shipping Order
  • Creation of a Stocktaking Order.

Modify Order. Only possible for Orders that are not yet "complete".

Delete Order. Only possible for local Orders that are not yet "released" (owner is system or operator).

Modify Specific Settings for all selected orders on the Settings for Selected Orders. Only possible for Orders that are not yet "complete".

Release Orders for start of fulfillment. Only possible for Orders in "pending" status which are not "paused" (see fields in Detail Dialog Order). With Order Release, Retrieval Strategies are used to allocate the optimal required Inventory for the given Order Line criteria. As a result, Full Load Unit Retrievals and Pick Orders are created for subsequent Order Fulfillment, as described in Order Fulfillment overview.

 INFO 

If paused, Order should be released again. "Paused" flag must be changed manually before using the Modify Specific Settings button.

Cancel Orders. See Process: Order cancellation.

The concrete detailed conditions and subsequent operational processes with Order Cancellation could be different with project specific use. Cancellation considers the configuration topic "orp-order:orderClient.maxStatusForCancellation".

 INFO 

If Order Cancellation refused, cancel single Order Lines as described in Order Lines).

Abort Orders. See Process: Order cancellation. with respect to the special "Abort" notes.
If Order Abort refused, abort single Order Lines as described in Order Lines).

The concrete detailed conditions and subsequent operational processes with Order Abort could be different with project specific use.

Route

 

Release Route. Releases all Orders of the same Route as the selected Orders (see Release Button above).

 INFO 

Only possible for Shipping Orders.

 

Cancel Route. Cancels all cancelable Orders of the same Route as the selected Orders (see Cancel Button above) provided that no order of the route is already finished except configuration topic "orp-order:OrderCustomer.isCancellationAfterLoadingAllowed" is enabled.

Quick Order Builder. Creates simple new Order with Order Lines on the Quick Order Creation (see Manual "Quick & Easy" Creation of an Order).
