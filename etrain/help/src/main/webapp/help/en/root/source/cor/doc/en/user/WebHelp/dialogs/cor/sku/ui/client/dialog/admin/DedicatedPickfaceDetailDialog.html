Dedicated Pick Faces

Purpose

The "Dedicated Pick Faces" tab in the lower part of the screen Stock Keeping Units lists the Locations used for Picking, if this is configured accordingly in the Order Management.

Prerequisite: to use Dedicated Pick Faces, the related Disposition Zone must be configured with reservation type "aggregated".

 INFO 

In "Auto Disposition" areas only "Bins" are allowed as Dedicated Pick Face in order to avoid positioning issues. In Areas with Dynamic Slotting (positioning method is set to dynamic with use of Slots), Dedicated Pick Faces are not allowed at all.

 INFO 

For configuration purposes refer to the related topic in the Warehouse Administration Guide.
