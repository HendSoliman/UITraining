Inventory Movement Types

Purpose

Inventory Movement Types represent a classification of reasons for host-relevant inventory changes. An Inventory Change indicates that goods are added to or left the Inventory controlled by a Client within the Dematic iQ®. Any host-relevant inventory change initiated inside the Dematic iQ® (not ordered by the host) requires an Inventory Movement Type. For some hosts, an Inventory Movement Reason is also required.

 INFO 

Any change of the total quantity of an Inventory Unit within Dematic iQ® is sent to the configured external host system automatically in the standard.

An inventory movement type can have different movement directions. For each direction, the following default inventory movement type exist: in, standard delivery, out, Customer Order, from Client 'production' to Client 'sales' and internal.

 INFO 

For configuration purposes refer to the related topic in the Warehouse Administration Guide.
