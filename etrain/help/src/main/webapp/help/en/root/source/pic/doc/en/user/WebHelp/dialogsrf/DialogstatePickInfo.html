Pick Info

Purpose

"Pick Info" appears after push the key "F11" from every other dialog state that displays this key and provides all relevant data of the current pick operation for information purposes.

      
Screen

The single screens are viewable through scrolling by Page Dow/Up buttons or key F3/F4.

      
Fields

Field

Description

Pick Walk

The unique identifier of the current Pick Walk in process.

Source Location

The unique identifier of the current Pick Face (Source Location).

Source Load Unit

The unique identifier of the current Source Load Unit.

SKU

The SKU id uniquely identifies a Stock Keeping Unit within a Client.

Description

The description contains a textual description of the Stock Keeping Unit.

Client

The unique identifier of the Client.

Batch

The batch of the Inventory Unit to be picked.

Best before date

The best before date of the Inventory Unit to be picked.

Quantity

The remaining quantity to be picked into the Pick Container.

Unit

The quantity unit of the Pick operation. This will be the "Pick Quantity Unit" In case of exactly one Default Pick Quantity Unit is set for the quantity units of the SKU. Otherwise the Base Quantity Unit is displayed.

Pick Container

The Pick Container collects all Picks of one Pick Mission and contains the Inventory Units that are already picked into.

Put Location

The Alias represents the name of the Put Location on the Pick Cart for easy identification. The alias is only textual information and will not be validated from the system.

Customer

The name is unique per Client and could be used for display purposes.

Service Qualifier

The service qualifier is an optional attribute which describes how the Pick Order has to be fulfilled.

Destination

The unique identifier of the Destination Location for delivery of Pick Container.

Order Line Text

Free text of related Order Line.

Number of finished Picks

The number of already finished Picks within the current Pick Walk.

Number of outstanding Picks

The number of outstanding active Picks within the current Pick Walk.

List of Pick Container

The list of currently booked Pick Container on Put Locations of Pick Cart.
     
Special Buttons

Function/Button

Description

Scroll

Key "F3/F4":

Scrolling to the next or to the previous page.

Back

Key "Enter":

Close the Pick-Info screen and return to the previous step.
