Overview

Application starting errors

Introduction

The System Controller (SC) manages one or more VMController (VC). Each VMController is a Java Virtual Machine (JVM) with one or more applications.

It is possible, that an application is not able to start because any necessary information or configuration files are not available.

This section will describes and give solution for all errors which can be occur during the starting process of applications.

      
Overview

This section is structured as follow:

Chapter

Description

Starting SAP connector failed

Cause of error: Starting of application failed, which is responsible for SAP connection.

Starting translator failed

Cause of error: An application for processing incoming (from host) messages failed.

Starting ServiceBusTranslator failed

Cause of error: The application for processing outgoing (to host) messages failed.

CleanUP service failed

Cause of error: The application responsible for cleaning EAI specific database tables can't start.
