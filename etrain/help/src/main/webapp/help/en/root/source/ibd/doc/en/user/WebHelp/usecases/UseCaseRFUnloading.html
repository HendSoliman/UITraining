Quantity based Receiving with RF-Terminal

Description

Quantity Based Receiving in automated or manual Areas using an RF-Terminal ordered to given Advices provides all required functionality to deposit the delivered Advice Inventory in single Inserts onto Load Units and initiate the concluding Inbound Transports.

      
Triggers and Actors

This is started when:

  • delivered goods are available for goods receipt and
  • delivery receipts are available and the Receiving process of the delivered goods should be started with an RF-Terminal.
      
Prerequisites
  • Advice must be successfully transmitted from host to local system. It is available in "released" or "active" status for processing and can be checked in Advices screen
  • Load Unit, respectively the product, is labeled with an appropriate GS1-Label.
      
Required Resources
  • RF-Terminal Receiving.
      
Work Flow

Step

Description

1.

Open RF-Terminal Receiving from the main menu.

2.

Step Scan Dock displays.

3.

Scan the Dock.
"Enter" key activates with the scan.

3a.

If an invalid Dock is scanned, an error message appears. Proceed with step 3.

4.

If one Client exists in the system, it is automatically selected.

4a.

If more than one Client is available the step Select Client displays.

4b.

Select a Client and press "Enter".

5.

Next step Identify Advice displays.

6.

Enter a supplier and press "Enter". Or simply press "Enter" to proceed without selecting a Supplier.

6a

If no open Advice is found, proceed with step 6 or press "ESC" to abort the Receiving process.

7.

Next step Select Advice displays.

8.

Select an Advice using the "up" / "down" keys and press "Enter".

9.

Next step Scan Load Unit displays.

10.

Scan the appropriate barcode of the Load Unit where the Inventory to be received is deposited.

10a.

Press ESC to proceed with a different Advice.

11.

Next step Check LU-Type displays.

12.

Select a Load Unit Type if a previously incorrect entry or the type is yet unknown. Press "Enter".

13a.

If no Advice Line can be uniquely identified, the step Identify Product displays.

13a.1

Scan the product (GS1 - GTIN).

13b

If an Advice Line can be uniquely identified, the step Check Product displays.

14.

Enter the quantity to receive. Or simply press "Enter" to acknowledge the advised quantity.

15.

Screen proceeds with step 9.
     
Result

Successful processing of goods receipt due to one Advice Line occurs. Received Load Units with deposited goods have an Inbound Transport Request and are in transit to their Storage Locations. After arrival and Advice finalization, the Inventory is available for disposition processes.
