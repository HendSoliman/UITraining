Overview Goods-to-Person Picking

Purpose

Goods-to-Person Picking is supported for many Load Unit types and many configurations of material handling equipment. Goods-to-Person Picking deals with all picking processes, special within automated areas, where goods are transported to fixed Pickstations for pick operation and subsequently conveyed back to the storage area.

The Pick Controller for Goods-to-Person Picking includes a specific collaboration with transport management for supply with Source and empty Load Units, as well as Load Unit release at a stationary Pickstation when a pick is complete. Using containers, cases or cartons instead of pallets differs mainly with respect to the transport handling. The different Picking Dialogs support external trigger for automatic process flow with signaling of the conveyed Load Units by the PLC and external signals for Pick Confirmations (e.g. by additional Put-to-Light application).

Description

Basic Processes

The Pick Controller for Goods-to-Person Picking includes a specific collaboration with transport management for supply with source and empty pallets and pallet release at a stationary Pickstation when a pick is complete. Using containers, cases or cartons instead of pallets differs mainly with respect to the transport handling. Picking is done at Pickstations which are Workstations for picking purposes that contain all properties to fulfill Goods-to-Person requirements and represent their own workload of assigned Pick Missions.

Use Cases

All use cases which can be performed in a Goods-to-Person Pick Controller. They are valid for specific functions and configurations of Goods-to-Person Picking with respect to special tasks for collaboration with transport management.

Screens

All screens for Goods-to-Person Picking processes and configurations. It provides the Workstation screen, used to configure Pickstation(s), different operating Picking Screens for Goods-to-Person picking, and a special Analysis Dialog for the analysis of actual Pickstation situations.
