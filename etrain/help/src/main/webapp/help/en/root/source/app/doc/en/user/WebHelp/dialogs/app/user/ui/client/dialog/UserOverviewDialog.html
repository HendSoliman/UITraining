Users

Purpose

This screen is used for Administrating users.

 

      
Functions

Operators can perform the following tasks:

  • Create, delete, editing users
  • Assign users to groups
  • Clear logged on flag of a user.

 

      
Screen

 

      
Sub-Screens
  • User Detail Dialog

 

      
Fields

Field

Description

User Name

Unique Username

Full Name

Full name of the user, e.g. first name and last name

Description

Description of the user, e.g. department

Locked

Yes / No - Can the user log on?

Logged On

Yes / No - IIs the user logged on currently?

Password Changeable

Yes / No - Can the user change his/her password?

Password Is Initial

Yes / No - Is the password still initial?

End of Privacy Period

Date and time when the user can be notified again, e.g. by alarms.

Time Of Last Login

Date and time of the last log on

Time Of Last Logout

Date and time of the last log off

Failed Login Attempts

Number of failed log on attempts

Terminal Id

Terminal id where the user logged on.

Host Name

 Host name where the user logged on.

Email Address

Email Address of the user

Telephone Number

Telephone number of the user

Mobile Telephone Number

Mobile Telephone number of the user

Password Expiration Date

Date and time when the password gets invalid.

User Expiration Date

Date and time when the user cannot log on to the system anymore

Locale

Locale of the user.

Description
