Marshalling Grids

Purpose

A Marshalling Grid is a special zone within a warehouse for the possibly sequenced staging of units before shipping. The Marshalling Grid Administration Dialog is used to configure the Marshalling Grids and provides the basic management functions for related operational steps as closing or opening and assignment of Docks or Loads.

Special Buttons

Button

Description

Open the selected Marshalling Grid to accept the assignments of Loads.

Close the selected Marshalling Grid to prevent further assignment of Loads.

Release the selected Marshalling Grid and switches the "blocking status" accordingly.

Lock the selected Marshalling Grid to prevent the progress of any standard operations.

Assign the selected Marshalling Grid to a Dock (see Assign Marshalling Grid to Dock)

Assign Loads to the selected Marshalling Grid (see Assign Loads to Marshalling Grid)
