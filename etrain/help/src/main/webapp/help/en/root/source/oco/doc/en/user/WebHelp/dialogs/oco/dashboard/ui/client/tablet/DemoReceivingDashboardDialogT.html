Receiving Overview

Purpose

This screen displays information on main receiving concerns with a special regard on pallets or totes and automatic or manual areas.

 

      
Screen

      
Fields

Provides information based on open Advices.

KPI

Description

Missing Deliveries

Number of released and not yet active Advices with an outdated expectation date.
     
Special Buttons

Provides information based on the outstanding workload distinguished between pallets and totes.

KPI

Description

Expected Pallets

Estimates the number of pallets indicated by not yet received Advice Lines:

  • Advice Line Load Units with Load Unit Type "EP" are counted directly.

  • SKUs with a defined Load Unit capacity for type EP calculates by using the announced quantity .

  • Otherwise one pallet per Advice Line is assumed.

Expected Totes

Estimates the number of totes indicated by not yet received Advice Lines:

The same calculation is done with Load Unit Type "TO".

Deliveries (/h)

Number of released Advices with an expected shipment within the next 24 hours (1h granularity).
     
Completed Work: Deliveries

Provides information based on the yet completed Advices.

KPI

Description

Completed (/h)

Number of completed Advices in status "completed", "to be confirmed" and "finished" (for the last day with a 1h granularity).
     
Completed Work: Received Load Units

Provides information based on the already received Advice Line Load Units.

KPI

Description

Completed (/h)

Number of completed Advice Line Load Units in status "completed" and "receiving completed" grouped by 1 hour periods for the last 3 days.

Advice type

Number of completed Advice Line Load Units in status "completed" and "receiving completed" grouped by the Advice Type for the last 3 days.
     
Active Operators

Provides information based on the currently active receiving operators distinguished between manual and automated areas.

KPI

Description

Active operators

Number of currently working operators in receiving areas which have received a Load Unit within the last 10 minutes.
