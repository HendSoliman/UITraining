Overview

Frame

Description

2D Pallet View

Show 2D appearance of all packages on the base pallet. This is also the arrangement area for constructing current pallet. 

Mouse clicking (left/ right key) on a package on 2D view, can take turns its status by following orders: active, half-active and inactive.

  • Active: Package with yellow/ green surface, and its detailed information will be displayed in the information section under 2D and 3D view. Package position will be controlled by moving mouse pointer at this time.
  • Half-active: Package with grey surface, and all detailed information will also be displayed. But we can NOT move package by mouse in this status.
  • Inactive: Package with white surface, its detailed information will not be shown, and also can't be moved.

 INFO 

1. Active package with yellow color can only be moved in the space between other packages of the same horizontal level, and be restricted within the available range of the pallet. We can extends the available range of the pallet by change the configuration oversizeLengthMax and oversizeWidthMax in config/dcd/pac-palletizerloadunit.xml.

2. Active package with green color is allow to do overlap move. It means when position of the active package overlaps the position of other package(s), then this active package will be put upon the highest one.

3. Move active package (GREEN color) onto trash bin area and click it, then package will get removed.

3D Pallet View

Show 3D appearance of all packages on the base pallet.

 INFO 

The current active (yellow/green color in 2D view) /half-active (grey color in 2D view) package will be highlighted with red lines.

 

Stored SKUs

List all SKUs(SKU Quantity Unit & Quantity) stored in aisle 01 of the location of current workstation. Clicking box in the list to add a new package in 2D&3D views.

If the quantity of the SKUs run out, then the SKU will be hidden in the list.

All SKUs

List all SKUs(SKU Quantity Unit) stored in whole warehouse. Clicking box in the list to add a new package in 2D&3D views.
