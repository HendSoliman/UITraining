Pick Walk

Purpose

A Pick Walk contains one or more Pick Missions that must be processed within one entire Pick Walk in case of Person-To-Goods Picking only.

This screen provides the functions to activate and finalize "Person-to-Goods" Pick Walks and supports the paper based Offline Picking.
With Pick Walk activation, the Pick List prints if configured appropriately.
With Pick Walk confirmation, each Pick must be confirmed with its picked quantity, used Pick Container and possibly the measured weight, Function Code and Reason if an exceptional result.
A Pick can also be split into another Pick Container due to completed container weight or volume.
