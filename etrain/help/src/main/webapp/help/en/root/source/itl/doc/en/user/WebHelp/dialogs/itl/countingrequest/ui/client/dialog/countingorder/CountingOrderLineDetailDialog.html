Detail Dialog Counting Order Line

Purpose

This screen opens when creating, viewing, or modifying a Counting Order Line.

 INFO 

For configuration purposes refer to the related topic in the Warehouse Administration Guide.

Overview

Handling & Result

Times

      
Fields

Field

Description

Counting Order

Unique ID of the Counting Order.

Counting Order Line ID

Unique ID (usually a running number) of the Counting Order Line.

Status

Current execution state of the Counting Order Line:

  • pending (Counting Order Line is awaiting release for execution)
  • released (Counting Order Line is released for execution)
  • active (Counting Order Line execution already started)
  • complete (Counting Order Line execution finished and the Counting Order Line is fully completed)
  • cancelled (Counting Order Line was cancelled without confirmation of any counting result).

Overview

Invalid

If this Counting Order Line is invalid. After Counting Order Line creation for recounting operation based on an original Counting Order Line, the original Counting Order Line becomes invalid to denote the existence of a related Recounting operation.

Sequence

For Person-To-Goods Counting Orders for each Counting Order Line the processing sequence within the Counting Order is given by this field.

Order Line

Stocktaking Order Line of this Counting Order Line (including the values for the Client and Order ID).

Inventory

Base Load Unit

ID of the Base Load Unit where the Load Unit to be counted is stacked onto.

Load Unit

ID of the Load Unit to be counted.

SKU ID

Stock Keeping Unit of the Inventory Unit to be counted.

Description

Detailed SKU information.

Batch

Batch in which the inventory was produced.

Best Before Date

Date before the inventory should be consumed.

Quantity

Total quantity of the related Inventory Unit expressed in the Base Quantity Unit of the SKU.

Unit

Unit of the quantity of the SKU.

Handling & Result

Counting Result
Error Code

Last error which occurred with this Counting Order Line:

  • OK (successfully completed)

  • load unit missing (counting not possible, because no Load Unit found)

  • load unit not available (counting not possible, because the mentioned Load Unit is currently not available for counting)

  • inventory not available (counting not possible, because the mentioned Inventory Unit is currently not available for counting).

Number of Counts

Number of count attempts on this Counting Order Line.

Expected Quantity

Quantity, in Base Quantity Units, which was expected for this Counting Order Line.

Counted Quantity

Quantity, in Base Quantity Units, which was counted through the last counting attempt.

Quantity confirmed by operator

Enabled if the stocktaking operator confirms the actual counted quantity manually.

Difference Quantity

Difference between Expected and Counted Quantity.

Recounting

Max. Number of Recounts

Maximum number of requested recount attempts if the recount tolerance is exceeded.

Recount Tolerance

Percentage of deviation from the expected quantity while recounting result confirmation by the operator. Leads to not automatically accept - forces operator to accept explicitly.

Non-automatic tolerance

Percentage of deviation from the expected quantity while counting result confirmation by the operator. Leads to not automatically accept - forces operator to accept explicitly.

Stocktaking Executor

Unique Identifier of the operator who executed (counted) the Counting Order Line.

Confirming operator

Unique Identifier of the operator who confirmed the Counting Order Line.

Execution Location

Location where the counting operation took place.

Times

Counting Start Time

Actual start time of the booking operation of this Counting Order Line.

Counting End Time

Actual end time of the booking operation of this Counting Order Line.

Counting Timestamp

Timestamp of the last counting operation of this Counting Order Line.

Inventory Unit

Last Stocktaking Date

Last Stocktaking date of the Inventory Unit before booking of this counting operation.

Load Unit

Last Stocktaking Date

Last Stocktaking date of the Load Unit before booking of this counting operation.

Location

Last Stocktaking Date

Last Stocktaking date of the Location before booking of this counting operation.
