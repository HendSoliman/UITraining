Replenishment Threshold

Purpose

This topic describes configuration of the Replenishment Threshold.

     
Configuration

A Replenishment Threshold defines an inventory quantity based condition (threshold) when creating a request for replenishment (replenishment order).

Attributes of the Replenishment Threshold are predefined by the Default Configuration.  They control business logic and can be reconfigured. See user guide screen for details.
