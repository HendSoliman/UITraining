Preferred Destinations

Purpose

A Preferred Destination specifies a set of Locations preferably used for Inventory storage for a defined SKU. The Putaway Strategy tries to find a valid bin in the set of Preferred Destinations. Another bin is selected only if no Preferred Destination Location is available.

The set of Locations is specified by the "from location" and the "to location". All Locations whose ID is sequenced between the ID of the "from location" and the "to location" belong to the set of Locations. Several preferred destination definitions are possible per SKU. They are ordered for selection processes by field sequence.

 INFO 

For configuration purposes, see the related topic in the Warehouse Administration Guide.
