Workstation - Detail

Screen

      
Fields

 INFO 

For configuration purposes refer to the related topic in the Warehouse Administration Guide.

Field

Description

ID

Unique ID of the Workstation.

Description

Description of the Workstation.
Terminal

Terminal

ID of the assigned Terminal (each client (PC) is represented by a terminal).

Workstation Type

Type of Workstation:

  • Receiving (for quantity based or Full Load Unit receiving)
  • Repacking (for repacking of Inventory Units onto other Load Units that is not picking)
  • Clearing (for all kind of clarification purposes at special Clarification Stations)
  • Depalletizing (for manual depalletizing into totes or onto conveyor)
  • Counting (for counting operation for stocktaking purposes)
  • Picking (for Goods-to-Person Picking operations at special Pickstations)
  • RapidPick A1 (special Pickstation for 1 Pick Container in automatic mode)
  • RapidPick A6 (special Pickstation for 6 Pick Container in automatic mode)
  • RapidPick M6 (special Pickstation for 6 Pick Container in manual mode)
  • RapidPal (special station AMCAP Station)
  • ErgoPal (special station for manual mixed case palletizing)
  • AMCAP Depalletizing (for specific AMCAP related automatic depalletizing)
  • Manual Depalletizing (for specific AMCAP related manual depalletizing).

Status

Status

Current state of the Workstation:

  • available (ready to work and get workload assigned)
  • paused (no new workload will be assigned)
  • closed (no workload will be assigned).
Operation Mode

The current mode of operation of related equipment this workstation is in:

  • automatic (Relevant equipment that influences the processing of the workstation is in automatic mode)
  • manual (Relevant equipment that influences the processing of the workstation is in (exceptional) manual mode).

Active

If the Workstation is active.

 INFO 

If inactive, not be automatically supplied with operational tasks, such as Pick Missions for example.
Be aware: only used for automatic assignment of workload not for the possibility to launch to the related operational screen.

Location

Location

Location ID that the Workstation is assigned to. This Location defines the place of the Workstation within the Warehouse. In terms of transport issues this must be the related Conveyor Location.

 INFO 

If the workstation needs to be assigned to a single aisle for a dedicated business process (e.g. in AMCAP or GTP-iAT scenarios) this location must represent the virtual coordinates of the group (aisle).

 INFO 

The specific Locations for operational purposes are used within the specific business processes by the related operational screen.

 INFO 

If any transport error occur, the destination for the required and initiated clarification transport is set to the Default Destination of the Location.

Handling

Load Unit Type

Load Unit Type used by default when Load Units are created automatically at the Workstation.

Queue Limit

Maximum number of possible Queue Entries for this Workstation for assignment of single task references. The impact strongly depends on the related business process.

Request Size

The Request Size defines the number of Queue Entries that must be assigned and requested for activation simultaneously as a group. The impact strongly depends on the related business process.

E.g. for picking operations this is the number of available and enabled Put Locations at best.

Request Threshold Counter

If the request for activation of yet assigned tasks fails because of insufficient number of assigned (occupied) Queue Entries this value defines the number of failed attempts so far. After a configurable number of failed attempts the request will be initiated anyway.

Load Unit Grid Orientation

Orientation that Load Units with geometrical stacked partitions (compartments) display at this Workstation.
Load unit grid figure must be turned clockwise by the defined value.

Empty Load Unit Destination

When a Source Load Unit becomes empty it is forwarded to the Empty Load Unit Destination.

Load Units Enroute Threshold

Lower threshold for source transport enroute to this station.

 INFO 

To disable this parameter set to 0.
