Counting Orders

Purpose

A Counting Order is a collection of counting, weighing, metering or estimating instructions at the fulfillment level based on a Stocktaking Order. One Counting Order instruction always refers to a concrete Inventory Unit, respectively Location, inside the considered Stocktaking Zone and defines the details for quantity determination of the considered Stocktaking object.

 INFO 

For configuration purposes, see the related topic in the Warehouse Administration Guide.

Special Buttons - Counting Order

Button

Description

Release Counting Orders for start of execution.
Only possible for Counting Orders in "pending" status (see Status field in the Detail Dialog Counting Order).

 INFO 

For Person-To-Goods Offline counting orders the release function also activates the counting order.

Open the Report Counting Results to enter the counting results for the Counting Order.
Only available for Counting Orders of appropriate Stocktaking Type (Person-to-Goods).

Close Counting Order with the appropriate bookings according the Stocktaking Zone configuration.
Only possible for Counting Orders in "to confirm" status.

Print Difference List. All lines of the Counting Order display with the following basic information:

  • SKU with description
  • Area
  • Load Unit ID
  • Excepted, counted and difference quantity
  • Executor ID with time stamp
  • Status.

Print Manual Counting List.
Used for counting operations in manual warehouse areas. Only displays lines in

"pending", "released" and "active" status with the following basic information:

  • SKU ID
  • Location
  • Load Unit ID.

 INFO 

The expected quantity does not display.
     
Special Buttons - Counting Order Line

Button

Description

Open the Report Counting Result to enter the counting result for the Counting Order Line.
Only available for Counting Orders of appropriate Stocktaking Type (Person-to-Goods).

Recount Counting Order Lines.
Only available for Counting Order Lines in "complete" status and Counting Order in "to confirm" status.
With this action, Counting Order Lines become invalid and new Counting Order Lines are created for recount operation.

Confirm counted quantity of the selected Counting Order Lines.
Only available for Counting Order Lines in "complete" status and Counting Order in "to confirm" status.
This explicit confirmation is required for Counting Order Lines with appropriate result deviations.
