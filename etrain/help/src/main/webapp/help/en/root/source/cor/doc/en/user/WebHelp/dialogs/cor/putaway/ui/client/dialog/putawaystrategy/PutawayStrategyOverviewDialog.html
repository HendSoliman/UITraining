Putaway Strategies

Purpose
  • Configure and enable required Putaway strategies for classic Areas (non-Multishuttle Areas) with definition of
  • valid and applicable facility (for Area Selector) or respectively Area (for Aisle and Bin Selectors)
  • required priority/sequence of considered strategies
  • enabling / disabling of specific strategies.
  • Configuration of the Putaway strategies for Multishuttle areas with definition of
  • the valid and applicable area
  • the required weight of considered strategies (all strategies with weight > 0 are considered concurrently with weighted single result sets).

 INFO 

For configuration purposes, see the related topic in the Warehouse Administration Guide.
     
Screen

      
Sub-Screens
  • Detail Dialog Putaway Group Strategy to configure Putaway Group Strategies (e.g. for aisles) for classic or Multishuttle Areas
  • Detail Dialog Putaway Level Strategy to configure Putaway Level Strategies for Multishuttle areas
  • Detail Dialog Putaway Bin Strategy to configure Putaway Bin Strategies for classic Areas only.
      
Fields

Field

Description

Order By

Strategy criteria used to sort the matching Locations. Entries with descending sortation are separately listed otherwise ascending is default.

Description

More detailed description of the Putaway Strategy configuration with the given Order By definition.

Enabled

Classic Strategies only!
This Putaway Strategy Configuration is only used when this is activated. Otherwise the strategy will be ignored.

Sequence

Classic Strategies only!
All enabled entries are used within the order by clause with respect to the related sequence. Lower sequence defines a more important "order by".

Weight

 Multishuttle Strategies only!
The weight (weighting factor) define the weighting of the configured MS Putaway Strategy. All strategies with weight > 0 are considered and a priority list is calculated. The priority list represents the ranking of the destinations in terms of the specific strategy. In case of multiple strategies has to be considered each ranking is weighted with configured weighting factor and the overall result priority list will be sorted according to the mean of all single lists.

Facility

Limitation on specified Facility if area strategy is mentioned. Can specify different Putaway Area Strategies for different Facilities. Empty value denotes the availability for all Facilities, which do not have a specific set of strategies.

 INFO 

In case of a certain facility has at least one specified entry all strategies that should apply for this facility need to be set with the facility value as well.

Area

Limitation on specified Area if Aisle or Bin Strategy is mentioned. Can specify different Putaway Aisle and Bin Strategies for existing Areas. Empty value denotes the availability for all Areas, which do not have a specific set of strategies.

 INFO 

In case of a certain area has at least one specified entry all strategies that should apply for this area need to be set with the area value as well.
