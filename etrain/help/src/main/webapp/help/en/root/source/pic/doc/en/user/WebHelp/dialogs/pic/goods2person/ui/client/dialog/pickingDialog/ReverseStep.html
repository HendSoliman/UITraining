Reverse

Brief Description

The "Reverse" screen appears after an operator enables it from the "Picking" screen.
Here, operators can confirm the reverse picking operation with exchange of Source Load Unit and Pick Container.

      
Screen

      
Prerequisites
  • Operator pushed "F7: Reverse" button or "F7" key in Picking.
      
Fields

For field descriptions, see Picking Screen Details.

      
Functions / Buttons

Function/Button

Description

Enter "Quantity Reverse"

Enter the picked and counted quantity for subsequent Putaway and restoring.

Change to state "Short"

Related Button or "F2" key:

Confirm inventory shortage due to several configured shortage reasons.

See Short.

Upon confirmation, additional exchange of Source Load Unit and Pick Container is booked by the system and must be performed automatically by the System or manually by the operator. Pick Container remains empty and will be deleted.

Confirmation of inventory shortage is only possible with previous entered Quantity Reverse = 0.

Cancel:

Related Button or "Esc" key:

Cancel and return to former state without confirming changes.

Pick Confirmation as "ok" with "reverse":

Related Button or "Enter" key:

Requested pick operation is confirmed with additional exchange of Source Load Unit and Pick Container.

Confirmed quantity of Inventory Unit of displayed Source Load Unit is booked into the Pick Container, the Pick is booked and confirmed with Confirmation Code 'ok'.

Exchange is performed automatically by the System or manually by the operator.

If Quantity Reverse = 0, Pick Container remains empty and will be deleted.
