Routes

Purpose

A Route is the central business object carrying information on the Carrier of the Load, the assigned Dock where the loading is executed, departure and required staging times. Thus the Route is a concrete instance of an Itinerary.
If required, all stops of the Itinerary can be hit during the Route which can optionally be planned by an external Route Planning System. The Route Administration Dialog is used for the basic management of Routes.
