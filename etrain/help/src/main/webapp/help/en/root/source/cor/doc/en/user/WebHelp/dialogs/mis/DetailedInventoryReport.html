Detailed Inventory Report

General information

This report is a list of all Inventory Units which are available within the warehouse or a selected Area. The inventory is defined by the SKU, Bach and Best Before Date and separately listed for the different Locations and Load Units.

      
Screen

      
Input Parameters

Input Parameter

Description

Facility

List information for the specified Facility only.

Area

List information for the specified Area only.

Client

List only inventory for the specified Client.

SKU

List only inventory for the specified SKU.

Batch

List only inventory with the specified batch.

Best Before Date from/to

List only inventory where the best before date is within the specified range.
     
Fields

Field

Description

Lot/Batch

The batch (lot) of the Inventory.

Best Before Date

The best before date is the date before the Inventory should be consumed.

Area

The Area of the Load Unit Location.

Location

The Location of the Load Unit that contains the selected Inventory Unit.

Load Unit

The Load Unit that contains the selected Inventory Unit.

Base Quantity Unit

The Quantity Unit represents the unit of the quantity of the SKU.

Total Quantity

The quantity is the total quantity of this Inventory Unit expressed in the Base Quantity Unit of the SKU.

Quantity on Hold

The total quantity in case at least one Inventory Hold for the selected Inventory Unit exists. The quantity is expressed in the Base Quantity Unit of the SKU.

Quantity Free

The total free quantity that is available for disposition and not on hold. The quantity is expressed in the Base Quantity Unit of the SKU.

Subtotals

Subtotal SKU, Batch/Lot, Best Before Date

The subtotals for the specified SKU, Batch and Best Before Date within the specified area.

Subtotal SKU

The subtotals for the specified SKU within the specified area.

Subtotal Area

The subtotals within the specified area.
