Quality Assurance Handler

Purpose

For basic clarification at a QA Workstation (quality assurance).
This handler is used when the logistic error code of the scanned Load Unit is "QA Clarification".

      
Information

Following views are provided in case of existent related data:

  • Load Unit
  • Weight Information
  • Inventory Information with packaging data
  • Orders in which the current Load Unit is used as Pick Container or Dispatch Load Unit

 INFO 

In case of a Load Unit Stack also the ID of the Base Load Unit is shown.
     
Functions

Available Functions

Change Location (F6)

Book Load Unit onto a location, which must be previously selected and confirmed.

Print Label (F7)

Reprint Load Unit Label on label printer connected to this Workstation.

Change Quantity (F8)

Open a detail dialog with all available data related to the Inventory Unit in case of existing.
In case of more than one Inventory Unit is available one has to be selected first.

Unlock Load Unit (F9)

Use this function to unlock the Load Unit.

 INFO 

Only available, if the lock status is not free.
