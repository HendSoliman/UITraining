Overview

Picking at RapidPick Stations

Purpose

Picking at RapidPick stations (A1, A6 and M6) occurs on the RapidPick Screen. It provides all functionality for multi-order Goods-To-Person picking processes for containers at special RapidPick stations.

      
Overview

Use Case

Description

Rapid Pick Operations

Pick operations include the standard case for confirmation of the single pick:

  • without variation
  • with handling of inventory shortage
  • with further split operation due to a full Pick Container
  • with advised overdelivery.

Put new Pick Container

If additional Pick Containers are required at a Pickstation, this must be performed to supply all required Pick Containers.

Push away finished Pick Container

On M6 RapidPick Stations, the finished Pick Container must be pushed away manually.
