Detail Dialog Dock

Purpose

This screen opens when creating, viewing or modifying Docks.

      
Screen

      
Fields

Field

Description

ID

Unique identification of the Dock (usually a short abbreviation).

Description

Description of the Dock.

Dock Function

Type of allowed dock operations:

  • inbound (inbound operations only)
  • outbound (outbound operations only)
  • inbound_outbound (inbound or outbound operations)
  • unused (not used for any operations at the moment)

Location

Integrates the Dock into the warehouse and all related operations that deal with locations (e.g. transports).

Status 

Status

Current state of the dock:

  • open (accepts assignments of Loads or Marshalling Grids)
  • closing (accepts no further assignments of Loads, but is still occupied. After the active Load is finished, the status is automatically set to closed)
  • closed (is closed and assignments are not accepted)

Configuration Status

If Dock configuration was finished:

  • incomplete (not yet complete and the system prohibits usage)
  • complete (complete and the system allows usage)

Dock Lock Status

If the Dock was locked for specific operations:

  • available (not locked and available for the operations allowed with the defined Dock Function)
  • locked (locked and not available for any operation)

Lock Reason

Reason for the lock.

Dock Occupation Status

Information about the current occupation of the Dock:

  • free (currently not occupied by any assignment)
  • occupied (currently occupied by an assignment for an operation)
