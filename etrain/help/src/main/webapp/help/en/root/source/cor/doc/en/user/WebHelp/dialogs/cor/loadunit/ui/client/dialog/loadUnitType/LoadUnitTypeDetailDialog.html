Detail Dialog Load Unit Type

Purpose

This screen opens when creating, viewing, or modifying Load Unit Types.

 INFO 

For configuration purposes refer to the related topic in the Warehouse Administration Guide.

 INFO 

The Unit of the displayed dimensions are given by the appropriate Base Quantity Units defined by the "Dialog Unit of Measure".
     
Screen

Properties

Dimension

Stacking

      
Fields

Field

Description

ID

ID of the Load Unit Type for identification.

Properties

Owner

"Host" for Load Unit Types defined by the host. "Local" for types that are manually created here.

To Be Deleted

If a Load Unit Type is flagged to be deleted it will be deleted when the last Load Unit of this type vanished.

Description

Description of the Load Unit Type.

Transport

PLC Name

LU Type is translated to this Name for the communication with the underlying PLC.

Usage

Virtual

LU's of this type are virtual and represents no real existent Load Units.

Stackable

If Load Units of this type can be stacked.

Usagetype Empty Load Unit

If Load Units of this type can be used as special "Empty Load Units".

 INFO 

Load Units of this type can be requested as special "Empty Load Units" to Pickstations. If Load Units of this type are being created as empty or become empty, an Inventory Unit of the special SKU with usage type "EMPTY_LU" will be booked to this Load Unit. Thus, it can be requested to a Pickstation as a future Pick Container using the Retrieval Strategies.
Refer to the process description "Empty Container Supply".

Usable for RapidPAL

 If the Load Unit is allowed for the special RapidPAL process.

Height Classification

Height classification for a LU for a LU Type with fixed (not variable) height.

Variable Height

If LU of this type can be of different height.

Variable Basic Size

If LU of this type may vary in length and/or width.

Handling

Keep Empty Load Units

If Load Units of this type should not be automatically deleted if the Load Unit became empty.

 INFO 

Area must be configured appropriately as well.

Dimension

Length

Length of related LU's; measuring unit is project specific.

Width

Width of related LU's; measuring unit is project specific.

Height

Height of the loading device of related LU's; measuring unit is project specific.

Volume

Volume of related Load Units; measuring unit is project specific.

Empty Weight

Weight of related Load Units without goods on it.

Max Weight

Weight limit of the related Load Units.

Max Height

Height limit of the related Load Units.

Raster Size X

Size of the raster for the X-Dimension. If greater than 0, the Load Unit is rasterized and provides geometrical stacking. Stacked Load Units represent partitions (inserts) on a concrete 'Position On Parent' on the Base Load Unit.

Raster Size Y

Size of the raster for the Y-Dimension. See Raster Size X.

Stacking

Allowed Stack Load Unit Types

All Load Unit Types allowed for stacking on this Load Unit Type. If no Load Unit Type IS assigned, stacking of all available Load Unit Types is allowed. If at least one Load Unit Type is assigned, only assigned Load Unit Types are accepted for stacking on this Load Unit Type.
