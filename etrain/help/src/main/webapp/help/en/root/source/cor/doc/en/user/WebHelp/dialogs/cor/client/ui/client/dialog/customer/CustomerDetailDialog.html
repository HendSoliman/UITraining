Detail Dialog Customer

Purpose

This screen opens when creating, viewing, or modifying customers.

      
Screen

      
Fields

Field

Description

ID

Identifier of the Customer. Unique within the system.

Identification

Name

Name of the Customer.

Name 2

Free field. Used to display further details.

Name 3

Free field. Used to display further details.

Logo File

File name of the logo image.

Global Location Number

The Global Location Number (GLN) is used to identify the Customer in a GS1 barcode and is part of the GS1 system of standards. The GLN is a 13-digit number that comprises a GS1 Company Prefix, Location Reference and Check Digit.

Owner

Owner

Customer definitions sent from the host have the owner "host" and are not changeable. Customer definitions created on screen have the owner "local" and are changeable.

Client

Customer is related to this Client. Can be used for specific processes related to this Client.

Status

Lock Status

If the Customer was locked for specific operations.

To Be Deleted

True, if the Customer is foreseen for deletion.
