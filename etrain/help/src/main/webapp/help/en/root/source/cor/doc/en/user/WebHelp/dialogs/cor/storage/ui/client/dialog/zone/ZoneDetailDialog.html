Screen |  Fields

Detail Dialog Zone

Purpose

This screen opens when creating, viewing, or modifying Zones.

 INFO 

For configuration purposes refer to the related topic in the Warehouse Administration Guide.
     
Screen

      
Fields

Field

Description

ID

ID of the Zone for identification.

Description

Description of the Zone.

Configuration Status

If Zone configuration is finished and the Zone can be used by the system.

Zone Type

Type of Zone. Defines its purpose:

  • storage (optional partition of the warehouse for administration of specific storage purposes. Project-specific use only)
  • picking (optional partition of a Zone of type "Pick Controller". Used by the Picking Subsystem (Pick Controller))
  • pick controller (a Zone of this type is administrated by a specific Pick Controller)
  • shipping (all Locations of a Stocktaking Zone are grouped due to Shipping processes. Project-specific use only)
  • stocktaking (all Locations of a Stocktaking Zone are grouped due to Stocktaking processes).

Sequence Number

Order in which Zones are chosen if more than one Zone is available for a business specific purpose.

Zone Status

Current status of the Zone (active or inactive). If inactive, the Zone is not considered for specific business processes.

Fill Level

Unused. Project-specific use only.

Pick Controller

Pick Controller which serves this Zone with respect to Picking purposes.
