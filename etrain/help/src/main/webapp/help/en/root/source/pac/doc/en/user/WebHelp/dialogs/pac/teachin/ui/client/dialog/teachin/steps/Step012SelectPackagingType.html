Teach-In Step: Select Packaging Type

Purpose

In this and the previous step, the packaging variant is characterized with respect to its packaging type.

All Packaging Types available for characterization are listed in dialog Dialog Packaging Types.

For characteristics see also the detailed list of Packaging Types.

If the number of different Packaging Types required for a project exceeds a certain ergonomic limit, two screens are provided for choice, otherwise only a single one.

If two screens are used for the choice, the first allows selection of the Packaging Group and the second screen is used for selection of the Packaging Detail for the selected Packaging Group.
