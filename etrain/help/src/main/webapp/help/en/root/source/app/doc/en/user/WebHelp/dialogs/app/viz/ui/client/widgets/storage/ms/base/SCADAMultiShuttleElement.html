SCADAMultiShuttleElement

Purpose

SCADAMultiShuttleElement represents a physical multishuttle system. It consists of several shuttles, lifts, pick- and drop-stations. The number of aisles can also be specified. This element provides four states. Up to now only multishuttle captive is supported.

 

      
Overview

 INFO 

The shape of the multishuttle is a simple rectangle which includes several sub-elements representing shuttles, lifts etc. This type of element is also known as  multi-layer-element. These elements consists of several layer with each layer showing a more and more detailed view of the element.

 

      
States

State

Description

WHITE

 currently no state available for location

GREEN

location is working with no errors

BLUE

location was blocked by admin
RED location has errors
ORANGE no connection

 

      
Sub-Controls

Symbol

Description

Shuttle

Lift

Rack-In / Rack-Out Conveyor
Pick / Drop Station

 

 INFO 

All elements are static representations. Movements are not supported.

 

      
Event Handling
Each sub-control has its own state. If there is a StateChangedEvent for a sub-control, the event is propagated to the corresponding parent-element.
