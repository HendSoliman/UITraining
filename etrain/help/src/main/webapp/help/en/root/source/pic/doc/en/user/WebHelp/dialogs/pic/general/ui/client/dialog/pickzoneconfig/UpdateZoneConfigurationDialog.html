Update Pick Zone

Purpose

This screen provides the ability to copy a configuration of one Pick Zone to one or more other Pick Zones.

 INFO 

For specific configuration purposes refer to the related topic in the Warehouse Administration Guide.
     
Screen

      
Fields

For field descriptions, see the Detail Dialog Pick Zone Configuration.

      
Usage

Step

Description

1.

Select one entry on the main screen and click "Update".

2.

The "Update Pick Zone Configuration" screen displays.

3.

Select zone(s) from the list on the right.

4.

Mark attributes whose value should be copied to the selected Pick Zones.

5.

Click "OK".
