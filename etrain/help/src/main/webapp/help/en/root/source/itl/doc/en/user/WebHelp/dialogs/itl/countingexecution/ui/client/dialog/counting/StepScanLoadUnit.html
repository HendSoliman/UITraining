Scan Load Unit

Purpose

The "Scan Load Unit" step appears if in Stocktaking Zone Configuration "Always" is set, or "More than one" is set and currently several Load Units are located at the Workstation. If the scanned Load Unit is still transported it will ask to complete the Transport Order. The below screen shows a Load Unit already arrived at the Workstation, but not scanned yet.

      
Screen

      
Prerequisites
  • Stocktaking Zone configuration requires the scan of the Load Unit to be counted (defined by "Load Unit Scan").
  • Required Load Unit is en route to the Workstation or yet available at the Workstation.
