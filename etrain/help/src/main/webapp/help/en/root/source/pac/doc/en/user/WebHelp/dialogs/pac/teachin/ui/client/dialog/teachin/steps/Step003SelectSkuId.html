Teach-In Step: Select SKU

Purpose

This step allows identification of a product / SKU in cases where no product code is available or where it is not unique with respect to SKUs.

The step is entered only if no GTIN has been specified in step Scan Product Code or if multiple SKUs exist which have the same product code.

 INFO 

Using the same GTIN for different Trade Items or even for Trade Items of different SKUs is not in accordance with GS1.
Nevertheless, it may occur in exceptional cases.

Screen

 

      
Usage
  1. Enter filter criteria for the SKU ID and / or the SKU description
  2. Press <Enter> to see the result
  3. The screen will show as many lines for SKUs as fit on the screen
    If not all qualified SKUs can be shown a hint will be displayed
  4. If necessary, refine the filter to restrict the result set
  5. Touch on the correct SKU to select it.
  6. Press <Next> to continue. 

 

      
Fields

Field

Description

SKU-ID Filter 

In this field a filter criterion for the SKU ID can be entered. For unknown parts, a wildcard character ("*") can be used, for example "135*" 

Description Filter 

In this field a filter criterion for the SKU description can be entered. Typically, this will be part of the product name seen on a package.

For unknown parts, a wildcard character ("*") can be used. 

 INFO 

The filter criterion must be specified in the correct case.
