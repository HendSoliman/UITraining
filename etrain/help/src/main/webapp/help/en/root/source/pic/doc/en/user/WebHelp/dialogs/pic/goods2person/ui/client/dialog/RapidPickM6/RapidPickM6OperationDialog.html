RapidPick Dialog

Purpose

The RapidPick screen provides all functions for specific rapid picking operations at a special Goods-to-Person Pickstation for container (tote). Three Pickstation types are supported:

  • A1 Pickstation (with 1 automatically handled Pick Container)
  • A6 Pickstation (with 6 automatically handled Pick Container).
  • M6 Pickstation (with 6 manually handled Pick Container)

 A-Pickstations are automatically handled stations which don't require operator interaction for Pick Container supply. For a manual Pickstation, put and push operations are performed manually by the user and supported by this screen accordingly. Pick operations include the following standard cases for confirmation of the single pick:

  • without variation
  • with handling of inventory shortage
  • with additional split pick operation
  • with advised overdelivery

 INFO 

Screen is optimized for resolution: 1280 x 1024.

 INFO 

The operational Picking process is described in Picking at RapidPick Stations.

 INFO 

Any locked Put Location is not available for picking operations at a Workstation. Locked Put Locations are specifically marked (see screen "Locked Put Location").
     
Overview

Station structure

The main panel shows the Pickstation structure. E.g. for A6/M6 stations, one source location in the center and on the both sides, 3 locations for used Pick Containers. When using A1 stations, only one location is available for the Pick Container (left or right side; this is configurable).

The exception screen provides buttons on the right for increasing and decreasing the current quantities, which can be set to 0.1, 1 and 10. An arrow also indicates which direction the user must move to fulfill the current pick operation (only available for A6/M6 stations).

Information panel

Basic SKU information displays along the bottom, such as ID, description, Base Quantity Unit and if an image is available. This is only available in the normal screen and replaced by available function codes when the screen switches to the exception screen.

Button panel

4 buttons are available on the bottom right. Refer to the different states for a description of general and button functionality. For manually operated stations, a switch to the exception screen is not provided until all required put or push operations are fulfilled.

 INFO 

The filename of the SKU image is configured in the attribute "image name" of the SKU. The image should be available with this filename in the default misc folder.
