Keyboard Handling

Purpose

This chapter describes the available keyboard shortcuts.

 

In order to avoid conflicts with standard keyboard shortcuts which are evaluated directly by the browser, the UI uses the modifier Ctrl-Alt to resolve conflicts. Example: Ctrl-N is evaluated by the browser (in general opens a new browser window), Ctrl-Alt-N is evaluated by the UI frame. Both of them have the semantics "new".

Dialog

The following table describes all shortcuts which act on the currently active dialog.

Shortcut

 Description

F5

 Refreshes the data grid.
Info: do not use the refresh button of the browser to refresh data. It reloads the page and opens a new session. All entered data will be lost.
Enter
  • If the focus is on a selected row of a grid, a sub dialog is opened which displays detailed information of the selected data set. This corresponds to the view/modify action of the toolbar.
  • If the focus is on a filter text field, a refresh is performed using the text entered a a filter criterion.

Ctrl-Alt-N

Opens a sub dialog for creating a new data set. This corresponds to the new action of the toolbar.

Ctrl-Alt-P

Triggers the printing of the dataset. This corresponds to the print action of the toolbar.

Ctrl-Alt-Q

Closes the dialog.
Del Deletes the currently selected dataset. This corresponds to the delete button of the toolbar.
Context menu/ Window Opens the context menu for the currently selected dataset. This corresponds to a right click.

 

The following tables describes keyboard shortcuts which allow navigation within the data grid.

Shortcut

 Description

Arrow-up / Arrow-Down

 Moves the currently selected row in the data grid one line up / down.
Page-up / Page-down Scrolls the data grid one page up / down. The selected row(s) remain unchanged.

 

      
Subdialog

The following tables describes keyboard shortcuts which apply to subdialogs.

Shortcut

 Description

Enter

 Triggers the default button of the sub dialog.
Esc Closes the sub dialog without saving data. This corresponds to the cancel action.
Tab / Shift-Tab Moves the focus to the next / previous input field.
Ctrl- Alt <-/-> Displays the next / previous tab in the tab pane.
F1 F1 opens the online help of the sub dialog.
