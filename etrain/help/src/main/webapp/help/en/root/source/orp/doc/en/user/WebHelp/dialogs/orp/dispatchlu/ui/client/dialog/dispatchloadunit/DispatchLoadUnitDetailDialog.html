Detail Dialog Dispatch Load Unit

Purpose

This screen opens when creating, viewing or modifying a Dispatch Load Unit.

      
Screen

      
Fields

Field

Description

Load Unit ID

Unique ID of the related Load Unit which contains Order related Inventory. A Dispatch Load Unit is created during a Shipping Order Fulfillment process, usually when the Pick Container is created or during creation of Full Load Unit Retrievals, and is related to an existing Load Unit.

External ID

Unique ID of the Dispatch Load Unit. Represents the Order related Shipping equivalent of the Load Unit. Picking can be done directly into Dispatch Load Units. Can also represent the external ID for Shipping service providers, e.g. parcel services.

Status

Current execution state of the Dispatch Load Unit:

  • planned (Dispatch Load Unit is only planned and not yet used for the Order Line)
  • available (Dispatch Load Unit is just created and available but yet unused)
  • used (Dispatch Load Unit is used and assigned to an Order)
  • picked (Picking into the Dispatch Load Unit is finished)
  • packed (Packing of the Dispatch Load Unit is finished)
  • staged (Dispatch Load Unit has been staged)
  • loaded (Dispatch Load Unit has been loaded)
  • shipped (Shipment of the Dispatch Load Unit is started)
  • completion intended (Completion of the Dispatch Load Unit during Order Cubing intended)
  • completion required (Completion of the Dispatch Load Unit during Order Cubing required)
  • cancelled (The related Order has been cancelled. Prevents loading).

Dispatch Load Unit Type

Dispatch Load Unit Type represents the physical attributes of the Dispatch Load Unit as configured in the screen Dispatch Load Unit Types.

 INFO 

If the system is requested to create automatically a new Dispatch Load Unit the type will be defined by the type of the new Load Unit. In case of multiple DLU Types exists for the used Load Unit Type the system applies any without providing a specific selection logic. This is done for example if a Pick Container that is represented as a DLU as well is manually repacked in a new Load Unit.

Packing Group

Inventory to put into the Dispatch Load Unit. Order Lines that belong to different packing groups must not be put together into Dispatch Load Unit.

Shipping

Fulfillment Zone

Where fulfillment of the Dispatch Load Unit is planned.

Destination

Location where the Dispatch Load Units must be transported (e.g. Marshaling Grid Location).

Dimensions

Volume

Calculated or measured volume of the Dispatch Load Unit.

Weight

Calculated or measured weight of the Dispatch Load Unit.

Times

Staging Time

Point in Time when the DLU was staged.

Loading Time

Point in Time when the DLU was loaded.

Shipping Time

Point in Time when the DLU was shipped.
