Teach-In Step: Layer Simulation

Purpose

This step is used for verification of dimensions through a comparison between a physical layer layer calculated using the measured dimensions.

If the number of items obtained in the layer simulation differs significantly from the actual number of items in the physical layer, this can indicate that the underlying dimensions are wrong and that the data need to be verified.

Fields

Field

Description

LU Type

Load Unit Type of the source Load Unit.

LU Length

Length of the Load Unit. If length and width measured in step Pallet Structure are larger than the dimensions registered for the Load Unit Type, the measured values will be shown. Otherwise those of the Load Unit Type.

LU Width

Width of the Load Unit.

Pack Length

Length which is used for layer calculation.

The Pack Length is usually slightly larger than the measured Teach Length (For details see Teach Parameters Overview)

Pack Width

Width which is used for layer calculation.

The Pack Width is usually slightly larger than the measured Teach Width (For details see Teach Parameters Overview)

Items/Layer

Number of items per layer as specified for the SKU Quantity Unit (See SKU Load Unit Capacities for Load Unit Type *_LAYER)

Items/Layer (simulated)

Number of items per layer as obtained with a palletizing calculation for a layer using the given pallet dimensions as well as the given pack dimensions.
