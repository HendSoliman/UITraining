Configuration Parameters - Orientation Restriction

Purpose

This topic is used to configure the installed turning and toppling devices for AMCAP. Depended on this configuration the orientation of items are restricted.
Located in demo\target\configuration\pac-orientationrestrictions.xml.

      
DeWrapTopic

Key

Type

Dimension

Default

Description
orientationRestriction

String

 None

TURN_TOPPLE_TURN (all orientations are allowed)

Possible values are:

  • TURN (Only a pin is installed, allowed orientations are: LW and WL)
  • TURN_TOPPLE (A pin and a topple station is installed, allowed orientations are: LW, WL, HW and HL)
  • TURN_TOPPLE_TURN (A pin and a topple station and after that one more pin are installed, allowed orientations are: LW, WL, HW, WH, HL, LH)

 INFO 

TOPPLE_TURN is not supported, e.g. a topple station and after it a pin. Here the allowed orientations are depended on the initial orientation of the package.
