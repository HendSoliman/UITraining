SCADA Conveyor

Purpose

SCADAConveyor represents a physical conveyor location. This element provides five states as well as currently three context menu entries. The user can navigate to the Detailed Dialog of the Element and to the Status Dialog. There are no more detailed views of this element.

 

      
Screen

The conveyor element is a simple rectangle. The shape includes an arrow indicating the current conveyor direction. There is also a curved-conveyor Element available for changing the conveyor direction.
