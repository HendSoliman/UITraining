Picking

Purpose

When a Source Load Unit is available on the Pickstation, all Picks from it are loaded and display in this state.

 INFO 

Picks for Pick Container that are not yet available do not display.

This function is the normal standard operation. Requested pick operations are confirmed without any variation regarding the displayed data.
For the manual station (M6), push actions are also indicated. Pick Container Put Locations change the color and allow a manual push of the Pick Container. Simultaneously, picking from the Source Load Unit is possible.
The put operation is booked automatically or the operator confirms it with the Dialog Create Pick Container.
