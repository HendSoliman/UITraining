Using Dematic iQ Web Help Index and Search

Purpose

This topic describes the usage of the Index and Search tab of Dematic iQ Web Help.

Search

The Search tab allows searching the entire documentation for the occurrence of specific words.

The search function will find all documents containing words which are identical with a search criterion or which begin with a search criterion.
Search criteria are not case sensitive.

In Internet Explorer, the found occurrences will be highlighted when a document is opened.

The following search criteria are supported:

  • Search for a single word:
    "Client"
    will find all documents containing at least one word starting with "Client"
  • Searching documents containing two or more words:
    Individual terms must be separated by a blank:
    "Client user" or "Client AND user"
    will find all pages containing "Client" and "user"
    If "AND" is used, it must be uppercase.
  • Searching documents containing at least one of a set of words:
    "Client OR user OR dock"
    will find all documents containing at least one of the given words.
Index

The Index tab lists a set of index terms.

When characters are keyed in the edit field, the index list is positioned at the next match.

Double clicking on an entry or pushing <Display> with an entry selected, will show the selected document.
