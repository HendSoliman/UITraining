Source Put

Purpose

"Source Put" is the initial step of the De-palletizing screen. The screen depends on the De-palletizing Workstation configuration.

Scan Source (manually)
When the De-palletizing screen opens or there is no remaining quantity to pack on the source pallet, it checks if automatic source scanning is configured for the De-palletizing Workstation. If it is not configured, the operator is asked to scan the source pallet manually.

Wait Source (automatic)
When the De-palletizing screen opens or there is no remaining quantity to pack on the source pallet, it checks if automatic source scanning is configured for the De-palletizing Workstation. If it is configured and no source pallet arrived at the Source Location of the De-palletizing Workstation, the operator is asked to wait for the next source pallet.

Prerequisites
  • Valid configuration of De-palletizing Workstation with valid source and Destination Location.
  • If using operation mode "to conveyor", exactly one virtual Load Unit must be available at Destination Location.
  • There is no remaining quantity to pack from the last source pallet.

Special Buttons

Button

Description

Confirm manual scan

Related Button or "Enter" key:

Only available if manual Load Unit scanning is configured. If there are Destination Load Units to push away, a popup indicates this operation. After successful scanning, complete Source Load Unit data loads and displays in the "Source Panel".
If the operation mode of the De-palletizing Workstation is “to container” the next step is:

  • Confirm, if the Destination Load Unit is already available.
  • Destination Put ('Scan', if no automatic destination scanning is configured for this Workstation, otherwise 'Wait' for scan result)

If the operation mode of the De-palletizing Workstation is “to conveyor”, the next step is:

  • Confirm.

 INFO 

If the scanned Load Unit is still in transport, the operator can confirm the running transport:

With "OK", the system sends the related request to the underlying transport system (MFC) and the result displays immediately.

Update Source LU Queue and
try to load the next Load Unit
Related Button or "F5" key:

Available for manual and automatic scan. For manual scanning, only the "Source LU Queue" updates. For automatic scanning, it attempts load the next Load Unit after updating the queue.
If the operation mode of the De-palletizing Workstation is “to container”, the next step is:

  • Confirm, if the Destination Load Unit is already available.
  • Destination Put ('Scan', if no automatic destination scanning is configured for this Workstation, otherwise 'Wait' for scan result)

If the operation mode of the De-palletizing Workstation is “to conveyor”, the next step is:

  • Confirm.
