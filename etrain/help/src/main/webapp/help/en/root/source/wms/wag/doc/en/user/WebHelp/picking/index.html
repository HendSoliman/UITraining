Picking Related Resources

Purpose

This topic describes how to configure picking related resources in the Logistics Manager.

      
Overview

The following picking related resources must be configured in the Logistics Manager.

 INFO 

The resource order in the following table considers dependencies. Resources should be created in this order.

 

Resource

Short Hint

Description

Domain Values

optional

Describes configuration of picking related dynamic domain values.

Pick Zone Configuration

must be configured

Configures how picks are executed.

Confirmation Reason

optional

Assigns client-specific reasons for pick confirmation codes which are relevant in confirmation to the superior order fulfillment system and for reporting.

Pick Confirmation Handling

default is generated by system initialization

Assigns a handling configuration to pick zone and pick error code combination.

Workstation

must be configured if Goods-to-Person Picking

Workstation for Goods-to-Person picking.

Terminal Zone

optional

These assignments ensure that Pickers Terminal only gets Pick Jobs of configured Pick Zone regardless of configured Pick Zones of Picker which are overruled by this configuration.
