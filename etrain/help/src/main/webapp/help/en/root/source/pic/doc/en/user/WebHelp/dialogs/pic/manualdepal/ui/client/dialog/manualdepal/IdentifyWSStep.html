Identify Position

Purpose

This screen appears either at the dialog start or when the operator decide to change the physical position of the conveyor. Therefore the action “Change Position” is used in any step but active step.

      
Prerequisites
  • Start Manual Depalletizing Dialog
  •  Operator click "Change Position" button
