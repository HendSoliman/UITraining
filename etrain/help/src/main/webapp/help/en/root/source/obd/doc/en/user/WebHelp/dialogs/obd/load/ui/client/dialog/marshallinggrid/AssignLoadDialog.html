Assign Loads to Marshalling Grid

Purpose

The detail dialog is opened  by applying the "Assign Loads" button in the Marshalling Grid Administration dialog and provides the functionality to assign one or multiple Loads to a Marshalling Grid. The revoke of an existing assignment is possible as well.

 INFO 

If "Multiple Loads allowed" of the Marshalling Grid is disabled, only one Load could be assigned to the Marshalling Grid.
     
Screen

      
Fields

Field

Description

Available Loads

All available and not yet assigned Loads.

Assigned Loads

Already assigned Loads of the selected Marshalling Grid.
