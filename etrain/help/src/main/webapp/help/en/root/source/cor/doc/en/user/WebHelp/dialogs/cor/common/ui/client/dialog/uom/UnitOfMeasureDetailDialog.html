Detail Dialog Unit of Measure

Purpose

The detail dialog is opened when creating, viewing, or modifying Units of Measure.

 INFO 

 Attention: to change the base unit of measure for a certain dimension, all values for this dimension which are transferred from an external or underlying system must be converted accordingly.

 INFO 

For configuration purposes refer to the related topic in the Warehouse Administration Guide.
     
Screen

      
Fields

Field

Description

ID

Identification of the Unit of Measure itself; usually a short abbreviation.

Dimension

All Unit of Measures of one dimension can be converted to each other.

Description

Description of the Unit of Measure.

Conversion Factor

Value expressed in this Unit of Measure can be converted to the base Unit of Measure by multiplying the conversion factor.

ISO Abbreviation

Official abbreviation for the unit according to ISO standards.

Base Unit

If this Unit of Measure is the base Unit of Measure for this dimension.

 INFO 

Attention: to change the base unit of measure for a certain dimension, all values for this dimension which are transferred from an external or underlying system must be converted accordingly.
