Clients

Purpose

A Client represents the owner of products that are stored and handled within an Dematic iQ®-controlled warehouse. It is necessary to have one Client to work with Dematic iQ®.
If more Clients exist, some warehouse resources (SKU, Inventory) are distributed between Clients exclusively. Other Resources, such as Location and Load Unit, can be used for all Clients.
Client management occurs on the Client Administration screen. Clients can be assigned to different Host Systems by the EAI Interface.

      
Screen

The upper grid displays all Clients defined in the Dematic iQ®.
The middle grid provides the configured addresses and persons of the Client.
The lower tabs display the data of related objects, based on the Client selected in the master grid (addresses of the Client or a person of the Client):

  • Phone numbers
  • Email addresses
  • Addresses (only if persons)
