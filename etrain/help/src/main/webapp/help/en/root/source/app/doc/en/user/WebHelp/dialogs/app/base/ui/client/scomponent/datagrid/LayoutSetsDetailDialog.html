Layout Sets

Purpose

Data grid layouts may be saved by a given name within the dialog Layout-Sets.

 

      
Functions

The dialog Layout-Sets serves the following tasks:

  • Save current layout by a given name
  • Restore a saved layout when opening dialogs
  • Modify or delete saved Layout-Set
