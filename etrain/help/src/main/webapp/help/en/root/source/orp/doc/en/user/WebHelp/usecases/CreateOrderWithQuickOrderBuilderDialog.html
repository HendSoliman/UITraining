Manual "Quick & Easy" Creation of an Order

Description

Manual "quick & easy" Order creation occurs using the screen Quick Order Creation.

The Order is completed in two steps, with minimal user input, and can be released immediately. The Order can consist of one Order Line if Inventory Criteria is used, or of multiple Order Lines if Load Unit Criteria are used. Many Orders can be created in quick succession. This screen supports the following Order Types:

  • Shipping
  • Operator Retrieval
  • Stocktaking.
      
Triggers and Actors

A local user initiates this when:

  • a Shipping process for Inventory Units of one specific SKU should be executed.
  • a Shipping process for one or multiple Load Units should be executed.
  • no special properties of Order or Order Lines (Selection Criteria) are required.
      
Required Resources
  • Quick Order Creation. Can be launched from the screen Orders.
      
Work Flow

Step

Action

1.

Open the Orders screen and click the button
("Quick Order Creation") in the upper grid.

2.

The screen Set Order Header opens.

3.

Enter the desired header information and click "OK" or "Create LU Order Lines".

4a.

If "OK" was clicked, continue with assigning Inventory Criteria Order Lines to the Order.

4b.

If "Create LU Order Lines" was clicked, continue with assigning Load Unit Order Lines to the Order.

5.

Click "Create and stay" or "OK".

6a.

If "OK" was clicked, continue with Step 3.

6b.

If "Create and stay" was clicked, you stay on the current screen.
     
Result
  • Shipping Order for desired selection criteria SKU or Load Units is created and available for release.
  • Operator Retrieval Order for desired Load Units is created and available for release.
  • Stocktaking Order for desired selection criteria SKU or Load Units is created and available for release.

With release of the created Order, all relevant Inventory Units based on the selection criteria are identified by the system and considered for the subsequent order fulfillment process, which depends on the Order Type.
