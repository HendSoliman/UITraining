Warehouse Administration Guide

Purpose

The Warehouse Administration Guide describes administration tasks in a warehouse managed by Dematic iQ®. It includes resource administration ( e.g. work stations or load unit types) and business processes (eg. Putaway or Retrieval) with the appropriate strategies to be configured.

The purpose of the Warehouse Administration Guide is to:

  • enable a warehouse administrator to configure a warehouse
  • allow an efficient work flow for configuring and adjusting business processes
  • avoid wrong configurations
      
Target Group
  • Warehouse administrator
  • SW-Developer.
      
Prerequisites
  • Detailed know-how about Warehouse Logistics
  • Basic IT know-how
      
Overview

The Warehouse Administration Guide describes the configuration all Dematic iQ® resources with respect to dependencies. Resources should be configured in the order they appear in this document.
