Select Advice

Purpose

The dialog step "Select Advice" is next after the selection criteria have been entered and matching Advices are available. 

 INFO 

In case more Advices are available one Advice must be selected.
     
Screen

      
Prerequisites
  • Dock, Client and optionally a Supplier have been selected
  • At least one open Advice matching the defined criteria is available.
      
Fields

Field

Description

Client

The Id of the previously selected Client.

Dock

 The Id of the previously selected Dock.

Supplier

The Id of the previously selected Supplier or empty if no supplier was qualified.

Advice

The Id of the currently selected Advice.

Expected Delivery

Expected delivery time of the Advice.

# Advice Lines

The number of Advice Lines of the currently selected Advice.

# Palettes

The number of advised Load Units (Pallets) in the Advice.
     
Special Buttons

Button

Description

/\ and \/ (up and down)

Scrolling through the list of available Advices

Ok

Select the currently displayed Advice and switch to "Scan Load Unit"

Cancel

Cancel and switch to "Identify Advice"
