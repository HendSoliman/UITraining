Idle

Purpose

This screen appears for source location, which have no pallets on the way to it for manual depalletizing. As soon as workload is added the dialog change its state.

It is possible, that other source locations of current workstation have workload.

      
Prerequisites
  • workstation is enabled
  • the source location is not blocked
  •  no pallets is on the way to the source location
