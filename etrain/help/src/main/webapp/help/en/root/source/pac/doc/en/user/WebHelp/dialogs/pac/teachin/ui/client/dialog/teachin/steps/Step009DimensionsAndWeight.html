Teach-In Step: Dimensions & Weight

Purpose

In this step, dimensions and weight of an item are determined and checked for AMCAP compliance.

 

      
Step Sequence

Overview:

Teach-In Station dialog

Previous step:

Manage Pictures

Next step:

Layer Simulation (in case that a specific source Load Unit Type has been selected in step Pallet Structure)

Select Packaging Group (in case that step Pallet Structure has been skipped)

 

      
Procedure

Measurements can be done with or without a semi-automatic measuring device.

If the workstation is equipped as a semi-automatic workstation, pushing any of the measurement buttons, causes the system to query the related measurement value from the device and enters it in the respective field.

At any time, it is possible to enter values manually or to overwrite values originating from the device.

 

For a detailed description of the measurement procedure, see Teaching Products with the Teach-In Station.

 

      
Screen

 

      
Fields

Field

Description

Length

Length of the item in mm.

The length is measured as the longer horizontal side of the item when it is in its natural vertical orientation.

Width

Width of the item in mm.

The width is measured as the shorter horizontal side of the item when it is in its natural vertical orientation.

Length/Width

Ratio Length/Width

The ration must be >= 1 (Length >= Width) and < 4.

Height

Height of the item in mm.

The height is measured as the vertical side of the item when it is in its natural vertical orientation.

Layer

Height of a layer as determined from the supplier source pallet in step Pallet Structure.

The calculated Layer Height is shown for verification of the measured height.
If the user decides to use the calculated Layer Height instead of the measured item height, the button <Use Layer Height> can be used to copy the value into the Height field.

Weight

Weight of the item in g

Toppling Length

Toppling angle when item is toppled over the long edge in °.

Toppling Width

Toppling angle when item is toppled over the short edge in °.

AMCAP Compliance

The individual fields below the different dimensions indicate whether or not a dimension is AMCAP compliant.

The button is preselected in case that one or more dimensions are not AMCAP compliant.

The button can also be selected in case that all dimensions are AMCAP compliant if other physical properties of the packaging variant prevet it from being handled in AMCAP.

 

When this button is touched, a measuring history is shown.

The measuring history comprises all previous teaching measurements for different items of the same packaging variant.

 

If the actually measured dimensions are outside a given tolerance of the average of previously teached items of the same Packaging Variant, the actual variant must be considered a new variant.

Touching this button initiates creation of a new variant.
