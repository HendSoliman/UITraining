Layout

Purpose

This topic describes Layout configuration.

      
Configuration

A Layout describes how load units of a specific allocation type are booked onto a location. This occurs by defining a state-machine for the Location attributes Availability Layout Status and Reservation Layout Status.

 INFO 

Configuration of this state machine is complex. It is recommended to use the layouts predefined within standards.

 

The following layouts are provided by the demo project:

Layout

Description

DOUBLE

For bins that can store one big load unit or two small ones. Both small load units can be accessed without any relocation..

DOUBLE_DEEP

For a double deep bin that can store two load units. Load unit on the back position cannot be accessed without moving the load unit in the front position.

SINGLE

For a bin that can store exactly one load unit.
