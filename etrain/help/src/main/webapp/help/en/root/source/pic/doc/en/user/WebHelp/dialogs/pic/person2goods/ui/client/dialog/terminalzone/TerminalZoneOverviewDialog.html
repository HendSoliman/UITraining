Terminal Zones

Purpose

This screen is used to assign (mobile Picker-)Terminals to Pick Zones. Only standard grid operations are available.

These assignments ensure that a Pickers Terminal only gets Pick Walks of configured Pick Zone, regardless of configured Pick Zones of the Picker which are overruled by this configuration. This is relevant for example if a terminal is installed fix in a Pick Zone and is used by different Pickers.

 INFO 

For configuration purposes, see the related topic in the Warehouse Administration Guide.
