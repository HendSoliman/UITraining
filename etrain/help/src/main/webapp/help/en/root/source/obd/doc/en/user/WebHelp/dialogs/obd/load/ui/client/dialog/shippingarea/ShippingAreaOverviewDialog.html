Shipping Area

Purpose

The Shipping Area Dialog is used for administration of the Shipping Area for a graphical assignment of Docks to Marshalling Grids.
The assignments could be modified with Drag & Drop with

  • selection of the Marshalling Grid by a permanent left-button click
  • mouse movement to the desired Dock and releasing the button.

 INFO 

Displayed Data is automatically refreshed.
     
Screen

      
Dock Panel

The Dock Panel includes all available Docks. Available Docks have

  • Configuration Status "complete" and
  • Dock Function "outbound" or "inbound_outbound".

The Dock Status will be displayed by the signal light:

  • red: The Dock is "closing" or already "closed".
  • green: The Dock is "open".

If a truck has already arrived at the Dock a truck picture is displayed. The truck includes the following data:

Field

Description

First line

License plate of truck or other loading equipment.

Carrier

The assigned Carrier of the Route.

Route

The Route the Load is assigned to.

Load

The Load for loading at this Dock.

#DLU

The total number of Dispatch Load Units of the Load.

Depart

Planned time for departure of the Route.
     
Shipping Area Panel

The Shipping Area Panel includes the available Marshalling Grids. Available Marshalling Grids have

  • Configuration Status "ready" and
  • A valid zone assignment.

The current status of the locations (positions of the Marshalling Grid) are visualized by different colors:

  • white: Grid position is free and available for reservation of a Dispatch Load Unit
  • blue: Grid position is reserved and expects a Dispatch Load Unit
  • green: Expected Dispatch Load Unit has arrived at the Grid position
  • gray: The Marshalling Grid is closed.
