Announced Shipping Orders

Brief Description

This KPI reports the number of created shipping orders in the processing queue differentiated by the dimension "client" and "order type".

      
Screen

      
Input Parameters

Input Parameter

Description

Time Level

List information for the specified Time Level only and defines the time slice granularity which is to be displayed.

Start Time / End Time

List only data where the event time is within the specified range.

Client

List information for the specified client only.

Order Type

List information for the specified order type only.
     
Fields

Field

Description

Time Level

The displayed date is controlled by the selected Time Level which also defines the period of the time slice where the data is aggregated.

Client

The client for the announced order.

Order Type

The type of order created.

Sum of Orders

The count of the number of announced orders.

Sum of Order Lines

The total number of order lines for announced orders.

Average of Order Lines

The average number of order lines for announced orders.
     
Default Configuration

Parameter

Value

 5 min

 15 min, hour, day, week & month
