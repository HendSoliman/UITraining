Send away Pick Container

Description

This must be performed if a Pick Container finished due to processing the last Pick of related Pick Mission or if operator confirms that the Pick Container is full.

      
Triggers and Actors

The following users must perform this after Pick Container is finished:

  • Picker at Goods-to-Person Pickstation.
      
Prerequisites
  • All required Picks of related Pick Container are finished.
  • Operator performed a "Full" confirmation (see Full).
      
Work Flow

Step

Description

1.

Open the Picking Screen.

2.

Destination Push displays.

3.

Press "Enter" or click "OK".

4.

Send the Pick Container away from the required Put Location.

Required scan of Put Location

2a.

If scan of Put Location is required, Destination Push with Scan displays.

3a.

Scan required Put Location.
"Enter" key and "OK" button are activated.
Go to step 4.

4a.

Invalid Put Location is scanned.

5a.

The Put Location Invalid Scan (Push) displays.

6a.

Go to step 3a.
     
Result

All Picks of related the Pick Mission proceed to "finished" status. The Pick Container on the displayed Put Location is sent away and gets a transport request to next destination. The Pickstation's Put Location is free and ready for a Pick Container assignment.

     
Required Resources
  • Picking Screen and in more detail the following screens:
  • Destination Push
  • Destination Push with Scan
