Check Product

Purpose

Confirmation of the received quantity of the determined product with booking against the identified Receipt.

 INFO 

Te still open quantity of the Receipt is preselected per default.
     
Screen

      
Prerequisites
  • A Dock, Shipment, Load Unit and Load Unit Type have been selected and
  • A Receipt has been identified.
      
Fields

Field

Description

Client

The Id of the Client of the inventory to be received.

Dock

The Id of the selected Dock the operator is currently working at.

Supplier

The Id of the Supplier of the Receipt.

SKU

The Id of the Stock Keeping Unit of the Receipt.

Batch

The batch of the inventory to be received.

Quantity

The quantity of the inventory to be received.

 INFO 

Preselected is the still open quantity of the identified Receipt.

Quantity Unit

The base quantity unit of the Stock Keeping Unit.
     
Special Buttons

Button

Description

F3: Batch

Change the batch of the inventory to be received.

F4: Quantity

Change the quantity of the inventory to be received.

OK

Confirm the current receiving process of the inventory to be received and switch to "Scan Load Unit".

Cancel

Cancel and switch to "Scan Load Unit" .
