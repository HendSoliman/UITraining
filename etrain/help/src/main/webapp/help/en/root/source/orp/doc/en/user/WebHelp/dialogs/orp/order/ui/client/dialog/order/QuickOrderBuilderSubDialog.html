Quick Order Creation

Purpose

This screen opens through a button in the screen Orders. It provides "quick & easy" creation of an Order with multiple Order Lines. After Order level settings are selected - especially client and order type -, SKU or Load Units can be selected in the lower tabs. The Order Lines to be created are listed. Only Order Lines of one Retrieval mode can be created. When changing the Retrieval Mode during the Order Line creation process, all settings on Order Line level are lost.

The created Order is completed in a single step, with minimal input. All additional values are set to appropriate defaults.

 INFO 

Creating Order Lines with Load Unit Criteria is only allowed for orders of type "operator retrieval" and "stocktaking".
     
Dialog Screen

      
Fields

For detailed field information, see the Detail Dialog Order Line Inventory Criteria.

Field

Description

Client

Owner of the Order. Thus, owner of the Inventory which could be allocated with the Order.

Type

Controls principle Order planning and fulfillment processes. See Order Types in the Order Management Overview for details.

  • Shipping
  • Operator Retrieval
  • Stocktaking.

Priority

Influences the urgency. Detailed information about factors influencing the priority is only available within the Host system as the owner of the (Shipping) Orders. Internal Orders get a default value.
Use "Modify specific settings" button to manually overwrite it in the screen Orders.

Service Qualifier

Influences concrete fulfillment of the Order within specific Order planning or execution processes. May be project specific.

 INFO 

List of available service qualifiers can be limited using configuration topic "orp-order:OrderOrderType.allowedServiceQualifiers".
Preferred LU Type Considered while Retrieval Strategies determine the optimal Load Units to be delivered and as default proposal for the Pick Container types during pick operations.

Destination

Destination for all Dispatch Load Units of the Order. Can be a Facility, Area, Aisle, Zone or Location. Used by Order Management as a proposal. Can be changed manually or by business process specific decisions (project specific).

Latest Staging Time

Latest time to deliver to the Staging Area. Defaults to the current time to achieve an immediate Order processing.

Delivery Time

Time the Order should arrive at the Customer's unloading address. Defaults to the current time to achieve immediate Order processing.

Inventory - Order Lines

Quantity

Requested quantity in the chosen Quantity Unit
