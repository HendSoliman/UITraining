Strategy

Overview

The Logistics Manager provide a central subsystem for support of all strategic decisions when required in any business processes. There are two different main fields for strategic support which are provided by different components:

  • Putaway: Find the optimal storage location while Putaway processes dependent on specific requirements, basic conditions and configurable parameters.
  • Retrieval: Find the optimal load units and/or inventory units for a defined inventory request while retrieval or order reservation processes.
