Detail Dialog Client

Purpose

This screen opens when creating, viewing, or modifying Clients.

      
Screen

      
Fields

Field

Description

ID

Identifier of the Client. Unique within the system.

Identification

Name

Name of the Client.

Client Type

"Internal" or "external". An internal Client is used to administrate local SKUs like spare parts, which are not sent to a customer. An external Client is a Client used for a Customer.

Configuration Status

Defines the completeness of the configuration. If incomplete, the Client will not be used within the system.

Global Location Number

The Global Location Number (GLN) is used to identify the Client in a GS1 barcode and is part of the GS1 system of standards. The GLN is a 13-digit number that comprises a GS1 Company Prefix, Location Reference and Check Digit.

Owner

Owner

Client definitions sent from the host have the owner "host" and are not changeable. Client definitions created on screen have the owner "local" and are changeable.

Host

External system that defined the Client. Where all business processes for the Client are managed.

Default Host ID

The default Host ID. This Value is used as fallback if no Mapping Configuration for the host exists.

Status

Status

If the Client is locked for specific operations (i.e. the Client is switched to inactive).

To Be Deleted

Client is possibly to be removed, but some stock still exists, Advices or Orders for this Client. The Client is deleted when all stock, Advices and Orders are deleted.

Inventory

Default SKU Blueprint

Set to all SKUs of the Client as default. Blueprint configuration occurs in SKU Blueprints.
