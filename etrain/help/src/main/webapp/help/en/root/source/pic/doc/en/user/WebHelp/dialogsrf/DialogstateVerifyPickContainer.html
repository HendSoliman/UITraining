Verify Pick Container

Purpose

If scan of Pick Container is required to indicate the use of correct Pick Container the screen "Verify Pick Container" appears after confirmation of single pick operation. The verification is done by scan of Put Location (Alias).

      
Screen

      
Prerequisites

For entering the "Verify Pick Container" state the following prerequisites have to be fulfilled:

  • A valid pick operation is confirmed previously with picked quantity greater than 0.
  • Required scan of Put Location needs the appropriate configuration of Pick Zone (see Pick Zone).
      
Fields

Field

Description

Put Location

The Alias represents the name of the Put Location on the Pick Cart for easy identification. The Alias is only textual information and will not be validated from the system.

Put Location (Scan)

Input field for scan of Put Location barcode.
     
Special Buttons

Function/Button

Description

Scan "* Put Location"

With launching this dialog state the field "Put Location" is focused for entering the scanned Put Location. After scan of valid Put Location the function of key "Enter" is processed subsequently. In case of invalid Scan an error message appears.

Scan

Key "Enter"

The correct Put Location is scanned and the current Pick is confirmed.
