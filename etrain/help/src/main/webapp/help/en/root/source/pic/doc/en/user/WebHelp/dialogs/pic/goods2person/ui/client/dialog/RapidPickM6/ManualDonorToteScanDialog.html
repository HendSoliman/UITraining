Manual Source Scan

Purpose

This dialog is used to perform manual scan of donor (source) totes.

 INFO 

This is only for use in exceptional cases. For example in case of a "No Read" Exception of the next tote at the Source Location of the RapidPick station.
     
Screen

      
Fields

Field

Description

Source Location

Name of the Source Location of the RapidPick station.

Load Unit

Scan field for the donor tote for the next picking operation.
     
Special Buttons

Button

Description

If the scanned Load Unit is still in transport the operator is able to confirm the running transport:

With "OK" the system sends the related request to the underlying transport system (MFC) and the result is shown immediately.

Afterwards the RapidPick dialog is informed about the new scan.
