De-palletizing

Purpose

This operative screen supports all operations at a De-palletizing Workstation, which receives aggregated units which may be fully or partly segregated. It then provides a set of segregated units as output and optionally a remainder of the input.

 INFO 

For an introduction to De-palletizing, see, Material Flow Management.

 INFO 

Details if the displayed data, see De-palletizing Details.
     
Functions
  • De-palletizing into any container (trays or totes)
  • De-palletizing directly onto belt or conveyor.
