Preparation Step

Purpose

In the preparation screen the same information as in the staging screen are shown.

In this step the operator has to de-wrap the pallet and confirm this action with the next button.

      
Prerequisites
  • Current workstation is enabled
  •  Source pallet arrived at current source location.

 

      
Screen

      
Special Buttons

Button

Description

 

Switch to Teaching dialog to check if any sku palletized needs to be taught. 

 

Check that any product need to be taught (If PalletizingQuantityUnit for the SKUQuantityUnit is null, then need to be taught).

If yes, switch to Special Handling dialog. Otherwise, check whether active position is already exists, if yes, switch to Pending step, otherwise, switch to Active step.
