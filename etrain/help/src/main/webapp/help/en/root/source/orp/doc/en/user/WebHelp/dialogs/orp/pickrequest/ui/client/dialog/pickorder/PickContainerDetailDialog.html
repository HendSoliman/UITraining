Screen |  Fields

Detail Dialog Pick Container

Purpose

This screen opens when viewing a Pick Container.

      
Screen

      
Fields

Field

Description

Load Unit ID

Unique Load Unit ID which is related to the Pick Container. Also represents the Pick Container ID.

Pick Container

Base Load Unit

Base Load Unit ID that the Load Unit is stacked onto.

Load Unit Type

Load Unit Type of the related Load Unit.

Status

Current execution state of the Pick Container:

  • no PO (Pick Container is not assigned to a Pick Order yet) (project specific use only)
  • picking (Pick Container is currently used for Picking processes)
  • picked (Pick operations are finished and the Pick Container is scheduled for transport to the Destination Location)
  • complete (Pick Container arrived at the Destination Location of the related Pick Order (Line)).

Location

Current Location of the related Load Unit.

Dispatch Load Unit

External ID

Unique Dispatch Load Unit ID which represents the Order related Shipping equivalent of the Load Unit. Picking can be done directly into Dispatch Load Units. Can represent the external ID for Shipping service providers, e.g. parcel services.

Dispatch Load Unit Type

Dispatch Load Unit Type of the Dispatch Load Unit. Configured in the screen Dispatch Load Unit Types.

Status

Current execution state of the Dispatch Load Unit which represents the Pick Container:

  • available (Dispatch Load Unit is created and available, but unused)
  • used (Dispatch Load Unit is used and assigned to an Order)
  • picked (Picking into the Dispatch Load Unit is finished)
  • packed (Packing of the Dispatch Load Unit is finished)
  • shipped (Shipment of the Dispatch Load Unit is started).
