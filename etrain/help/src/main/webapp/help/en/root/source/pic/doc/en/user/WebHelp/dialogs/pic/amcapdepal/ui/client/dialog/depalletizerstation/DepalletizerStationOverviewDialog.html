De-Palletizer Stations

Purpose

Besides the general workstation configuration, for the De-Palletizer stations some further specific configuration is needed. This dialog allows to do such kind of configuration. The main part of this configuration is the assignment of the stations to the aisles. In standard AMCAP one De-Palletizer station is connected to two aisle without overlapping. But it is also possible to have overlapping, e.g. assignment of one aisle to more than one station.

The aisle assignment is essential for the determination of best fit De-Palletizer. But this assignment is not the only parameter for the de-palletizer determination. For detail on it see  De-Palletizing.
