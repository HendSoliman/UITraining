Load Unit Management

Overview

The following main business objects and tasks provide the basic part of the Load Unit Management:

  • Load Units
  • Load Unit Types
  • Unpacking/(De-)Stacking
  • Unpacking/(De-)Stacking with RF-Terminal
  • Relocation with RF-Terminal
  • Creation of empty Load Units (with Compartments).
