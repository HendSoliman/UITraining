Overview

Strategy Related Resources

Purpose

This topic describes how to configure strategy-related resources in the Logistics Manager.

      
Overview

The following strategy-related resources must be configured in the Logistics Manager of the Dematic iQ®.

Resource

Short Hint

Description

Housekeeping Configuration

load defaults from demo system

Configures mislocation issues that should be improved / eliminated by housekeeping for each area of the warehouse.

Putaway Strategy

load defaults from demo system

Can be configured in terms of assigning order sequences or weights to strategy aspects.

Preferred Destination

optional

Specifies a set of locations preferably used for the storage of inventory of the defined SKU.

Retrieval Strategy

load defaults from demo system

Can be configured in terms of assigning order sequences to strategy aspects.
