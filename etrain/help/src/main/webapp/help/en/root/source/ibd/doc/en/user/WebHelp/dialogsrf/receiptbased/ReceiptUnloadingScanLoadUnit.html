Scan Load Unit

Purpose

This step appears after a Shipment has been selected.
A Load Unit to be unloaded has to be scanned by the operator.

 INFO 

A GS1 (SSCC number) barcode is expected.
     
Screen

 INFO 

The scanned Load Unit in the displayed screen represents the 18 digits SSCC number from the GS1 barcode.
The prefix '00' is the Application Identifier for the SSCC number.
     
Prerequisites
  • A Dock and a Shipment have been selected.
      
Fields

Field

Description

Dock

The Id of the previously selected Dock.

Shipment

The id of the previously selected Shipment.

Load Unit

Input field for the SSCC number of the Load Unit.

 INFO 

A GS1 barcode is expected.
     
Special Buttons

Button

Description

Enter: OK

SSCC number is validated and the appropriate Receipt is selected.
Switch to step Check LU Type.

ESC: Change Shipment

Cancel and switch back to step Scan Shipment.
