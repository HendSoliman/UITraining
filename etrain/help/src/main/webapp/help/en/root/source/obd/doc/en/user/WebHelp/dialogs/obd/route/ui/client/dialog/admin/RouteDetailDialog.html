Route Detail

Purpose

This screen opens when creating, viewing or modifying a Route.

      
Screen

      
Fields

Field

Description

ID

Unique identification of the Route.

Load

The assigned Load of the Route.

Dock

The Dock the Route is assigned to.

Status

How far the Route was processed:

  • planned (Route has been created)
  • full (DLUs or Orders are assigned to the Route and given capacity limits have been reached)
  • active (Route is assigned to a Dock and loading is in progress)
  • loaded (Loading is finished)
  • finished (The truck has been departed).

Freight

Carrier

The assigned Carrier of the loading equipment wherein the Load of the Route was shipped.

License plate of truck

License plate of truck or other loading equipment.

Times

Earliest Staging Time

Earliest time for staging of units for this Route.

Latest Staging Time

Latest time when staging of units for this Route should be finished.

Docking

Planned docking time the Dock should be reserved for loading of the Route.

Expected Loading Time

The planned or expected loading time is used for slotting of the Docks. It is initialized automatically from the average loading time of the Itinerary and can be adjusted manually based on the number of DLUs.

Departure

Planned time for departure of the Route.

Description

Description of the Route.

Owner

Source of the Route:

  • host (initiated from an external Host by the host interface)
  • operator (manually created by a local user)
  • system (created automatically by the system within the related business process).
