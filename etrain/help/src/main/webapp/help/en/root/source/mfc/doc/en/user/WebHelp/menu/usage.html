Introduction

Purpose

In the MFC Menu the operating personnel gets all various functions for controlling and optimizing material flow.

In addition to that, there are screen for configuring the warehouse layout. These type of dialogs are used during commissioning.

 

      
Overview

The "Material Flow Controller (MFC)" consists of the following dialogs:

Section

Description
Archive Submenu  

Archive Transport Orders

This dialog gives access to historical data regarding transport orders
Archive Transport Units This dialog gives access to historical data regarding transport units.
System Submenu  
Message Trace Log of all telegrams being exchanged with subordinate systems.
Main dialogs  
Transport Orders Administration of transport orders
Transport Unit Administration of transport units
Status Displays the status of the material handling equipment elements.
Internal Transport Orders Creation of transport orders not initiated by the host system.
