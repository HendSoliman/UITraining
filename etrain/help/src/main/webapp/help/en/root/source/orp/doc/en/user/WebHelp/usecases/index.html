Use Case / Process

Description

Manual Creation of a Shipping Order

Describes the manually creation of a new local Shipping Order with possible selection criteria Stock Keeping Units or Load Units. The creation is done with use of the screen Orders.

Manual "Quick & Easy" Creation of an Order

The manual "quick & easy" creation of an Order is done with use of the screen Quick Order Creation. The created Order will be completed within a single step with a minimum of necessary input by the user and will be released immediately in case of desired. The Order could consist of one Order Line in case of used Inventory Criteria or of possible multiple Order Lines in case of used Load Unit Criteria.

Creation and Processing of an ErgoPal Shipping Order

Describes manual creation of a Shipping Order for manual palletizing at a special ErgoPal Pickstation. Creation occurs with the screen Orders.

Process: Order Pre-Cubing

Describes the default Order Pre-Cubing methods.

Process: Order Cancellation & Order Abort

Describes the order cancellation and order abort process and the resources involved in the process.
