Manual "Quick & Easy" Creation of an Advice

Description

Manual "quick & easy" Advice creation occurs using the screen Quick Advice Builder. The Advice is completed in a single step, with minimal user input, and is released immediately. It can consist of one or multiple Advice Lines.

      
Triggers and Actors

A local user starts this when:

  • an Advice must be quickly created with no special Advice or Advice Line properties.
      
Prerequisites
  • Desired and required SKU must be available.
      
Required Resources
  • Quick Advice Builder which can be launched from the Advices.
      
Work Flow

Step

Action

1.

Open the Advices and execute the button
("Quick Advice Creation") in the upper grid.

2.

Create a new Advice in the Quick Advice Builder screen and click "OK".
Required fields:

  • Client (owner of the Advice and inventory created during the Receiving process)

 INFO 

A Client must be selected first. The first entry in the list defaults.
  • Advice Type (defaults; see list of advice types here)

 INFO 

For field descriptions, see the Detail Dialog Advice.

3.

Create Advice Lines:

Select SKU and quantity with assignment to the Advice by clicking the appropriate button:

Required fields:

  • SKU (A grid displays entries which are filtered by the selected Client.
    The filter must be set to reduce the selection. Multiple selection is allowed. Press Tab to move to the quantity field.
  • Quantity (announced quantity for this Advice Line). Enter the quantity and press ENTER. The new Advice Line is added to the list.
  • Batch (optional and only editable if a set check box)
  • Best Before Date (optional only editable if a set check box)

Advice Lines to be created are listed on the right with their quantities and SKUs.

3a.

Remove undesired Advice Lines from the list by selecting the entry and use of the appropriate button.
4. Confirm the new Advice and all created Advice Lines by clicking "OK".
     
Result

An Advice with one or multiple Advice Lines was created and ready for the receiving process.
