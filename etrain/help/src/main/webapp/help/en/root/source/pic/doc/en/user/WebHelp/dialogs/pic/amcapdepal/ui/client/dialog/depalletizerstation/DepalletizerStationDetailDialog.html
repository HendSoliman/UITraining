Fields

Field

Description

Workstation

The assigned Workstation.

Depalletizer Station

Device ID

The device ID used by the Material Flow Controller (MFC). With this ID MFC informs about the status change of the stripper plate, the robot and the station.

Lock Status

The status of the complete De-Palletizer device.

  • Available (The De-Palletizer is available, but some components can be inactive, e.g. stripper plate and/or robot)
  • Locked (The De-Palletizer is locked and cannot be used, although all components like stripper plate and robot are available)

Stripper plate Operation Mode

The status of the stripper plate:

  • Automatic (Only automatic usage)
  • Manual (Only manual usage)
  • Off (Manually switched off)
  • Emergency (Short term lock; the emergency stop pressed)
  • Fault (Short term lock; runtime error)

Robot Operation Mode

The status of the robot:

  • Automatic (Only automatic usage)
  • Manual (Only manual usage)
  • Off (Manually switched off)
  • Emergency (Short term lock; the emergency stop pressed)
  • Fault (Long term lock; runtime error)

Allow Partial De-Palletizing

If set to true, the user can determine how many layers are to de-palletize at once.

In the case when the complete pallet is to de-palletize, the user de-wrapped the pallet at the De-Wrap station and send it to the De-Palletizer. On the way these none wrapped packages can fall down. This feature allows partially de-wrapping of the pallet.

Manual

A flag whether this is a manual De-Palletizer station.

Return Location

This configured location is only used for manual De-Palletizing stations (mandatory for manual stations).

In case of a no read at a manual station the package standing on the destination location is send back to this location. This action can be triggered from the manual de-palletizing dialog. Only in the case the dialog is at the wrong position and a no read for the package this is triggered automatically.

 INFO 

Such transports are created as UFO, because there is no logical Load Unit available for the package yet.
