Procedure

Stacking with RF-Terminal

Purpose

Using RF-Terminal Stacking, operators can stack or de-stack whole Load Units spontaneously. With the Target Load Unit scan, the target for the process is defined.

The target may be specified as:

  • a different known or unknown Load Unit or
  • the Source Load Unit itself where it will be booked as new Base Load Unit to the Location of its Base Load Unit.
      
Procedure

Step

Description

1.

Start RF-Terminal Stacking from the main menu.
Source selection

2.

Scan the Source Load Unit. (Confirmation occurs implicitly during scan or using "Enter" key.)
Target selection

3.

Scan the Target Load Unit. (Confirmation occurs implicitly during scan or using "Enter" key.)

3a.

The scanned Load Unit does not exist yet. A new Target Load Unit must be created.

3b.

Press the "LU-Type" button to select the load unit type.
Confirm

4.

Press "Enter" to confirm the booking.

The selected Source Load Unit is placed on the Location or on the scanned Target Load Unit.
