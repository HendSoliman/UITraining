Put Pick Container

Purpose

"Put Pick Container" appears in case of further Pick Container is required for processing the current Pick Walk. The Pick Container is assigned to the next free Put Location on operators Pick Cart. In this case the Put Locations are ascending numbered starting from 1 and not especially identifiable. In case scanning of Put Location is required a further input field for especially identifiable Location data is provided.

      
Screen

In case of additional Scan of Put Location is required the dialog is displayed as follows:

      
Prerequisites

For entering the "Put Pick Container" the following prerequisites have to be fulfilled:

  • An active Pick Walk is assigned to the Picker (the value of the Current Work Reference of the Worker is the Pick Walk ID)
  • Further Pick Container is required for processing the current Pick Walk.
  • A free location on Pick Cart is available (to be ensured only manually by operator).
  • A new (unknown) Pick Container is available.
  • Display of additional field for scan of Put Location needs the appropriate configuration of Pick Zone (see Pick Zones).
      
Fields

Field

Description

Load Unit Type

The Load Unit Type of the Pick Container.

Pick Container

The Pick Container collects all Picks of one Pick Mission and contains the Inventory Units that are already picked into.

Put Location

Only in case of additional scan of Put Location is required this further Input field is displayed.
The Alias represents the name of the Put Location on the Pick Cart for easy identification. The alias is only textual information and will not be validated from the system.
     
Special Buttons

Function/Button

Description

Scan
"* Pick Container"

With launching this dialog state the field "Pick Container" is focused for entering the scanned Pick Container. After scan of valid Pick Container the function of key "Enter" is processed subsequently. In case of invalid Scan an error message appears.

Scan
"* Put Location"

Only in case of required scan of Put Location the field "Put Location" is displayed and focused for scanning the Put Location Alias after entering a valid Pick Container. In case of invalid Scan an error message appears.

Change Load Unit Type

key "F10"

Dialog for change of Load Unit Type appears.

With cursor up and down in the reason list an alternative Confirmation Reason is selectable.

Cancel

key "ESC"

Cancel this dialog state and put not further Pick Container on Pick Cart. This action is only permitted in case of no further Pick Container is unconditionally required for proceeding the current Pick Walk.

Put Pick Container

key "Enter"

The new and scanned Pick Container is created with displayed Load Unit Type and assigned to displayed Put Location.
