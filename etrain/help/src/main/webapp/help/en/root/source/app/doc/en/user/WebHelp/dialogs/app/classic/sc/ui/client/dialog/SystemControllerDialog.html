System Controller

Purpose

The System Controller provides an overview over all the running virtual machines and shows their current state. Furthermore each virtual machine even each application can be stopped and started within the System Controller.

One System Controller can have one or more virtual machines (VM) with one or more applications. Each application, each virtual machine and the system controller can be monitored in the Runtime State Administration Dialog. Further-more they can stopped, restarted or shutdown within this dialog.

 

      
Functions

The System Controller dialog is used to perform the following tasks:

  • Managing the System Controller
  • Managing the Virtual Machines
  • Managing the Applications

 

      
Overview

The System Controller dialog shows a tree on the left hand side and a tabbed pane on the right hand side containing the following panels:

Panel

Description

Function

Tree Panel

Shows all virtual machines and the corresponding applications

Provides the ability to start, stop and shutdown the applications, virtual machines and the system controller

Runtime info Panel

Monitoring the system controller, virtual machines and applications

Runtime Environment Panel
