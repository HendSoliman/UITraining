Configuration Parameters - De-Wrap

Purpose

This topic describes the configuration parameters for the De-Wrap dialog.
Located in demo\target\configuration\pic-dewrap.xml.

      
DeWrapTopic

Key

Type

Dimension

Default

Description

QuestionData.

DeWrapQuestion.questionKey

String

TerminalId: With this the configuration can be different for each terminal.

 

 The I18N key under which the question can be found. Following property file is used: com.dematic.wms.pic.amcapdepal.ui.client.dialog.dewrap.properties

QuestionData.

DeWrapQuestion.action

DeWrapActionType.value

TerminalId: With this the configuration can be different for each terminal.

 

The action which is to perform when the question is answered with false. The possible values are:

  • Default (Automatic de-palletizing will be done)
  • Clarification (Source load unit will be send to the clarification station, which is configured for the current location)
  • Manual (The source load unit is send to the manual de-palletizing station).
ActionDependencySection.
ActionDependency.target		 DeWrapActionType.value TerminalId: With this the configuration can be different for each terminal.   The function for which the dependency is to define.

ActionDependency.PreCondition.
preCondition		 DeWrapActionType.value TerminalId: With this the configuration can be different for each terminal.   All here defined pre conditions must be fulfilled, e.g. must be true. Otherwise the function given in target is not available.
