Check LU-Type

Purpose

This step appears after a Load Unit has been scanned.
The Load Unit Type could be initially set, changed or adopted.

 INFO 

The type of the Load Unit is checked and changed only if necessary.
     
Screen

      
Prerequisites
  • A Dock, Shipment and a Load Unit have been selected.
      
Fields

Field

Description

Client

ID of the Client of the (optionally given) Advice Line Load Unit of the scanned Load Unit (could be empty).

Dock

ID of the selected Dock the operator is currently working at.

Shipment

ID of the selected Shipment.

Advice

ID of the (optionally given) Advice of the scanned Load Unit (could be empty).

Supplier

ID of the Supplier of the (optionally given) Advice of the scanned Load Unit (could be empty).

Load Unit Type

Load Unit Type of the selected Load Unit, if empty a type has to be selected.

Advised LU Type

If the selected Load Unit has been advised the related advised Load Unit Type is displayed.

Load Unit

ID of the previously scanned Load Unit.
     
Special Buttons

Button

Description

F2: Change LU Type

Change the Load Unit Type by selecting one of the list.

Scroll through the list of available Load Unit Types with the cursor keys (up and down).

Enter: OK

Confirm the selected Load Unit Type and
switch to step Identify Product if no Receipt could be uniquely identified and
switch to step Check Product otherwise.

ESC: Cancel

Cancel and switch back to step Scan Load Unit.
