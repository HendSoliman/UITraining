Internal Transports

 

      
Functions

The dialog Internal Transports is used to perform the following tasks:

  • Create internal transport orders

 

      
Overview

Whilst being "internal", an transport order being created by this screen can nevertheless be host relevant. In order to trigger a host notification select the appropriate host system from the combo box and tick the "Needs Confirmation" check box.

 

      
Screen

 

      
Entering Locations

To ease the entering of locations (either source or destination) a picker is provided. Proceed as follows:

Step

Action

1

Open the picker window either by pressing the picker symbol or the add button.
2 The picker dialog opens:



There are two possibilities to choose a location
3 a Navigate through the tree until a node is reached. Select the node and press return.
3 b If you know already a part of the location id (e.g. that you are searching for aisle 1 side 1 in the unit load area, UL011) then you can start typing the string.
After having entered at least five character the picker offers a completion list. Choose a location by clicking on it
     
Field Descriptions

The Internal Transport Orders window shows the following information:

Fields Internal Transport Orders

Description

Transport Unit

ID of transport unit

TU Type
  • Type of TU e.g. euro pallet

Move Reason

Reason for transport:

  • Putaway
  • Out
  • Picking
  • Relocate
  • Control
  • Sequence Test
  • Clarification

Priority

Controls the order of execution:

Values from 0-9 (0 = highest priority).

Group

Name of group if sequencing is required.

Sequence

Sequence into the group

Host. System
  • Communication name of the host system. Is used if transport completion is indicated by message.

Source

Source of the transport order.

Order Destination

Selected transport order destinations.

 

      
Special Buttons

The Internal Transport Orders window contains the following buttons:

Button

Description

Add

Add a destination, for the transport order.

Remove

Remove the selected destination.

Create Transfer Order

A transport order is created to the selected destination.
