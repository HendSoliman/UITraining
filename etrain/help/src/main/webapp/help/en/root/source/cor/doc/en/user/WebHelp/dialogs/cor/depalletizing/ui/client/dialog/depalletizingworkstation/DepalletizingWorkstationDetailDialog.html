Depalletizing Workstation Detail

Screen

      
Fields

Field

Description

Workstation

Unique ID of the Workstation.

Operation Mode

Basic de-palletizing process:

  • to container (Used for Workstations where products are de-palletized into a container or tote.)
  • to conveyor (Used for Workstations where products are put directly onto a belt or conveyor.)

Destination Label Type

Type of label to print for destination packaging units during de-palletizing. If not specified, no label prints.

  • sku packaging (Label containing SKU packaging information will print. (e.g. GTIN))
  • load unit (Label containing the load unit ID will print (e.g. SSCC)).

Label Destination Load Unit Type

Load Unit Type to use if Load Unit labels must be printed for the Destination Load Unit.

Source identified automatically

If set, the Workstation automatically uses the first Load Unit at the Source Location. This is scanned by the underlying system as the Source Load Unit. Otherwise, it must be scanned manually.

Destination identified automatically

If set, the Workstation automatically uses the first Load Unit at the Destination Location. This is scanned by the underlying system as the Destination Load Unit. Otherwise, it must be scanned manually.
