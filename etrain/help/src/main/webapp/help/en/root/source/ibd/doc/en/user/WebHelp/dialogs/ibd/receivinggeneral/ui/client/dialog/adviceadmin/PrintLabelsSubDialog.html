Print Labels

Purpose

With this function a defined number of SKU Labels with defined inventory quantity of a selected Advice Line could be printed while Receiving process.

 INFO 

The number of printed labels depends on the "Quantity per Label".

 INFO 

The print or reprint of the SKU Labels could only be done while receiving process is active.
Since the Advice Line status "Receiving complete" further reprints are not possible anymore.
     
Screen

      
Fields

Field

Description

SKU

The SKU Id is the Stock Keeping Unit Id of the announced Inventory specified by this Advice Line.

Remaining Quantity

The target quantity is the total quantity of Inventory that is announced by this Advice Line. The quantity is expressed in related to the 'Receiving Quantity Unit' of the Advice Line.

Labels

Number of Labels

The number of received Load Units and thus the number of labels to be printed.

Quantity per Label

The quantity on each of the Load Units for label data only.
