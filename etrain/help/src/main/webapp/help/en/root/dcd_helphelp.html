Using Dematic iQ Web Help

Purpose

This topic describes the features of the Dematic iQ Web Help.
Read also about Usage of Index and Search.

Buttons

Hides or shows the navigation area.

In order to memorize the setting, cookies must be enabled for the browser.

Opens the active topic in an HTML editor for editing
Requires that the HTMLEdit protocol handler is installed.
Also, the help system must be installed in a file system rather than being deployed on a web server.

Refreshes the display of the page in the content area.
This function must be used after editing in order to show the modified page in the Web Help environment.

Prints the Topic page actually shown in the content area. A dialog will be opened to select the printer.

Opens the home page of the help system.

Show the previous topic in the hierarchy of the table of contents.

Show the next topic in the hierarchy of the table of contents.

Opens the overview topic of the active topic.

Opens the parent topic of the active topic if available.

Go back to the last viewed topic.

Go forward to the next viewed topic.

Copies the hyperlink target of the active content page to the clipboard. This can be used for linking to the page from other documents.

Opens this help file

Closes the active help window
Logos

The logos in the upper left and right are linked to project specific targets, typically the home pages of the customer and Dematic.

Navigation
  • Use the Contents tab to navigate through the table of contents
  • Use the Index tab to find topics associated with specific key words.
  • Use the Search tab to search for specific words throughout the entire help pages.
Navigation in Content Pages
  • Use the shortcuts at the top of the page to directly navigate to a specific block within a topic.
  • Use the arrow links in the block navigation lines to navigate to page top, previous block, next block and last block, respectively.
