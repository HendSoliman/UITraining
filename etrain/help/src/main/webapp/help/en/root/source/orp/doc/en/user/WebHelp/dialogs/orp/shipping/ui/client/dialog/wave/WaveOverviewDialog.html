Waves

Purpose

A Wave groups Orders for collective disposition and execution for Picking and Shipping purposes. This screen shows all existing Waves in the system and the yet assigned Orders of each Wave.

 INFO 

New Wave creation occurs in the Orders screen by selecting yet assignable Orders.

Special Buttons

Button

Description

Delete Waves before release. A Wave can be closed, aborted or cancelled after the release.
Only possible for Waves that are not yet "released".

Release Waves for start of fulfillment by executing release of all related Orders (see screen Orders).

Only possible for Waves in "pending" status.

Start Waves. Activates the fulfillment of all related Orders.

Only possible for Waves in "release" status.

Cancel (i.e. dismantle) Waves disconnects all yet assigned Orders and leaves the Orders unchanged in other respects.

Only possible for Waves that are not yet "released".
