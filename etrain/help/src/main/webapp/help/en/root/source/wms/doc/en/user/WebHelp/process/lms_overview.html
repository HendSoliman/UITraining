Overview

Logistics Manager (LMS) as part of the Dematics Dematic iQ® is responsible for nearly all business processes within a warehouse. It manages all key resources, such as storage locations, handling areas, load units and inventory. Logistics manager is also responsible for covering inbound operations (receiving and putaway), outbound operations (order fulfillment planning and control, full load unit retrieval and picking) and internal operations (replenishment, housekeeping and stocktaking).

Logistics Manager typically receives advices (or purchase order based shipments) and shipping orders from an external host system. It does the process planning according to given specifications and restrictions and delegates all required fulfillment transactions to the underlying executing subsystem like Material Flow Controller, Pick Controller or Sortation Controller. Logistics Manager keeps the data concerning Load Units and Inventory, Storage Locations and Stock Keeping Units. The Inventory Units assign inventory attributes like Stock Keeping Unit, batch (lot) and quantity to a certain Load Unit and Location. If Dematic iQ® keeps inventory information of Load Units it is capable to operate more independent of the Host and enables more sophisticated optimization strategies, which make use of this data.
Further overview about the main logistics processes which are supported by the Logistics Manager of the Dematic iQ® is given in Introduction to LMS.

The operational reporting which supports the daily work by providing information on actual and current workload, resources and fulfillment status is described in Overview of the Standard Statistics Reports.

This is also supported by the Logistics Cockpit which provides information of current potential problems and additional Key Performance Indicators of the relevant concerns in the warehouse.

The strategically relevant management information based on long term data for trend information is provided by the Management Information System based on reports on specifically defined Key Performance Indicators.
