Status/Means of Transport

Purpose

The dialog Status/Means of Transport is used to view and administrate the status of Means of Transports.

 

      
Screen

 

      
Fields

The Means of Transport Tab dialog shows the following information:

Status/Means of Transport Tab

Description

Description

Description of MTS.

Error Flag

Error flag indicating whether an MTS error has occurred.

Lock status

Lock status.

Current Position

Current position of the vehicle (if applicable)

Last movement

Date of last movement.

Means of Transport

ID of the means of transport.

MTS type

Type of means of transport: conveyor, vehicle, multishuttle, ...

Operation Mode

Current status of MTS:

  • Automatic
  • Switched Off
  • Emergency Off
  • Manual operation
  • Service position
  • Base Position

Order State

Order status of the MTS:

  • No TO
  • With TO

 

      
Special Buttons

The Status/Means of Transport Tab window contains the following buttons:

Button

Description

Lock

Lock the MTS

Release

Release the MTS

Check error status

Refreshes the error status of the MTS

PLC off

Switches the PLC off

PLC on

Switches the PLC on

Move Vehicle

Moves a Vehicle to a selected Location
