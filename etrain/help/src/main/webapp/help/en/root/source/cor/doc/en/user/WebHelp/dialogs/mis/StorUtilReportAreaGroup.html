Storage Utilization Report - Area Aisle Detail

General information

This report gives an overview of the available storage capacity as well as on its utilization based on the Area and Aisle details.

Only Locations with an Aisle and Area but without a Level assignment are considered. 

      
Screen

      
Input Parameters

Input Parameter

Description

Area

List information for the specified Area only
     
Fields

Field

Description

Facility

The Facility where the Locations are located in.

Area Type

The type of the Area defines the overall use. Possible values are:

  • receiving (Area for operative receiving processes)
  • shipping (Area for operative shipping processes)
  • storage (normal Storage Area like High Bay Warehouse or Multishuttle)
  • buffer (Area for intermediate buffering of Load Units subsequent to specific business processes e.g. Picking)
  • clearing (Area for clearing purposes: Clarification Station are typically located within a Transport Area)
  • virtual (Virtual Areas are only logical Areas which are necessary for logical booking operations (e.g. for Inventory bookings) only)
  • transport (Area for transport purposes of Load Units only)

Area

The Area where the Locations are located in.

Number of Aisles

The number of Aisles in the specified Area.

Aisle

The Aisle where the Locations are located in.

Level

The Level where the Locations are located in. (In this report always empty)

Location Type

The Location Type represents the physical attributes of the Location.

Height Classification

For easier processing the heights of the Locations are divided in several project specific height classes.
Number of Locations The number of Locations in the specified aisle respectively Level.
Number of Blocked Locations The number of blocked Locations in the specified aisle respectively Level.
Number of Occupied Locations The number of occupied Locations in the specified aisle respectively Level.
Number of Partially Occupied Locations The number of partially occupied Locations in the specified aisle respectively Level.
Number of Empty Locations The number of empty Locations in the specified Aisle respectively Level.
