Change state to Delete

General Information

The state of the selected message will be set to "Delete".

      
Prerequisites

The current state of the selected message is "Error" (E).

      
Procedure

To perform this task, proceed as follows:

Step

Action

1

Choose one message header with state "Error"

2

Press "Change state to delete" button
