Log in to Dematic iQ®

Purpose

Only authorized personnel is allowed to access a Dematic iQ system. This chapter explains the login / logout procedures.

The management of user being allowed to access a Dematic iQ system is described in the context of the user management dialogs.

Description

Login

 

Procedure:

Step

Description

1.

Enter the user name in the Login name field.

2.

Enter the password in the Password field.

3.

Select the language in which the UI should show up in the Language field. This language can be changed online during the session later on.

If no language is chosen (empty field, default), the system uses the default language which has been assigned to the user.
4. Check the "Restore Layout" checkbox if you want to continue your session where you have left it.
In this case, the windows having been opened at logout time are restored, including column settings, filter values and sorting settings.
5. Click the [OK] button to login to the system.

 

After a successful login the user name is displayed in the status bar.

 

      
Logout

The user can leave the UI by pressing the logout button of the Toolbar. Alternatively, he can close the browser tab or window.

After a logout the login screen is displayed again.
