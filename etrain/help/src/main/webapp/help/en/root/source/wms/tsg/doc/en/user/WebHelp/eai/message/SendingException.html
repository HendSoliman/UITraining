Sending Exception

Symptom

The JCoffee Proxy is not reachable.

 

      
Brief description

A message can't be processed. It has an error message of type SendingException.

 

      
Categories
  • EAI

 

      
Possible errors

In case of a SendingException, the following reasons and possible:

  • The application server is not reachable.

 

      
Detail description

A message in state N (New) is available in the Transfer Tables dialog. After processing this message it went into state E (Error) and the error text contains an SendingException.
