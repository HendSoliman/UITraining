Push away finished Pick Container at RapidPick station

Description

This must be performed if a Pick Container finished due to processing the last Pick of a related Pick Mission or if a picker confirms that the Pick Container is full.

 INFO 

At automated stations (A1/A6), Pick Containers are send away automatically.
     
Triggers and Actors

This must be performed after a Pick Container is finished.

      
Prerequisites
  • All required Picks of related Pick Container are finished or
  • Picker performed a "Full" confirmation (see screen Exception).
      
Required Resources
  • RapidPick Screen
  • Picking step.
      
Work Flow

Step

Action

1.

Open the RapidPick Screen.

2.

The screen Picking displays.
3. Finished Pick Containers are indicated for push away.

 INFO 

Unavailable for automatic stations.
4. Touch respective Pick Container on the screen to confirm.
5. Push away the Pick Container.

 

      
Result

All Picks of the related Pick Mission proceed to "finished" status. The Pick Container on the displayed Put Location is sent away and gets a transport request to the next destination. The Pickstation's Put Location is free and ready for a new Pick Container assignment.
