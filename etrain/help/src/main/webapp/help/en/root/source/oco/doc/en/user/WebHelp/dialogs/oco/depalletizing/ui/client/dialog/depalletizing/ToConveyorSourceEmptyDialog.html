Source Empty

Purpose

An operator can book the Source Load Unit empty (only in operation mode "to conveyor"). The specified de-palletized quantity is booked accordingly.

      
Screen

      
Fields

Field

Description

Already Done Quantity

Predefined with quantity, "Already done" from step "Confirm". Defines the already de-palletized quantity.

Description

Confirm repack process with confirmation code "Source empty" and the given quantity.
Next step is Source Put.

Close screen and return to the step Confirm without any bookings.
