User Group

Purpose

This dialog is used for administration of groups.

It is possible to organize users in user groups, such as to define an order according to departments. Access rights (Permissions) are assigned to User groups. Users belonging to a certain user group will get their access rights.

The user groups themselves may be assigned to other user groups in turn, thus enabling a hierarchical user structure. 

      
Functions

The Group dialog is used to perform the following tasks:

  • Create User Groups
  • Modify User Groups
  • Delete User Groups
  • Assign Users to User Groups
  • Assign Permissions to User Groups
  • Assign User Groups to a User Group
