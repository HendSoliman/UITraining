Order Pre-Cubing Results

Purpose

Order Pre-Cubing is an Order planning step. The default Order Pre-Cubing method estimates the number of Dispatch Load Units per type. The result of Order Pre-Cubing can be used by an external Transport Management System (TMS) for planning transportation routes and truck capacities prior to Order Fulfillment.

 INFO 

For details on standard Order Pre-Cubing methods, see the Process Description "Order Pre-Cubing".

Fields

Field

Description

Client

Owner of the Order. Thus, owner of the Inventory to be allocated with this Order.

Order ID

Order the Dispatch Load Units are assigned to.

Status

Current execution state of the Order:

  • created (Order was created and not yet ready for pending, but ready for preparation phase (project specific use))
  • prepared (Order preparation phase is finished, but not ready for pending (project specific use))
  • pending (Order is awaiting release for fulfillment)
  • released (Order is released for fulfillment)
  • active (Order execution already started; Full Load Unit Retrievals and/or Pick Orders are in progress)
  • complete (Order execution finished and the Order is fully complete; all Order Lines are completed or cancelled)
  • to confirm (Order is ready for confirmation message to the host)
  • finished (All required Tasks for the Order are finished. If required, Order was confirmed to the host).

Type

Controls the principle Order planning and fulfillment processes. See Order Types in the the Order Management Overview for a listing.

Service Qualifier

Influences concrete fulfillment of the Order within specific order planning or execution processes. May be project specific.

 INFO 

Can limit the list of available service qualifiers using the configuration topic "orp-order:OrderOrderType.allowedServiceQualifiers".

Number of DLUs

Number of Dispatch Load Units in "planned" status, that are assigned to this Order .

Total Weight

Total weight of Dispatch Load Units that are assigned to this Order .

Total Volume

Total volume of Dispatch Load Units that are assigned to this Order .
