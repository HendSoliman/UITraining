Teach-In Step: Select Content Type

Purpose

In this and the previous step, the packaging variant is characterized with respect to its content.

All Content Types available for characterization are listed in dialog Dialog Content Types.

For characteristics see also the detailed list of Content Types.

If the number of different Content Types required for a project exceeds a certain ergonomic limit, two screens are provided for choice, otherwise only a single one.

If two screens are used for the choice, the first allows selection of the Content Group and the second screen is used for selection of the Content Type for the selected Content Group.
