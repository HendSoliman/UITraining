Packaging Types

Purpose

The packaging type defines the external structure of each product. The combination of packaging type and content type defines some default values which are common for all products of this combination.
In addition this configuration allows to use different colors in the pallet widget.
The packaging type is selected during the Teach-In process where each packaging type can be determined first with a packaging group and then with packaging detail.
