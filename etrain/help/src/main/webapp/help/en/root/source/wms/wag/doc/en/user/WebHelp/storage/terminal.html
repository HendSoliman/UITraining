Terminal

Purpose

This topic describes Terminal configuration.

      
Configuration

Terminal configuration is required if a UI client has a specific business role or when equipment (e.g. weighing scale or printer) must be assigned to the terminal.

The following Terminal attributes control business logic. They are not predefined by the system and must be configured:

Attribute

Mandatory

Hint

Terminal ID

yes

Identifies the UI client and must be configured in the UI client installation. The Dematic iQ® rich client UI uses the parameter -DterminalId=<terminal id> to identify the terminal. See user guide screen for details.

Terminal type

yes

See user guide screen .
