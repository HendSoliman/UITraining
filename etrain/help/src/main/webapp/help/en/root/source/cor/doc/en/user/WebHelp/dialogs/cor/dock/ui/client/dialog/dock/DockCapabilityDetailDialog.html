Dock Capability

Purpose

The administration of "Dock Capabilities" is done in the lower part of the Docks screen.
Dock Capabilities are individual features which are available for the Dock and which can optionally be used as criteria for assignment of trucks or Loads. Needs could be e.g. specific dock equipment, specific dock size, availability of conveyor for hanging garments or others.
