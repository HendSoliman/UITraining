Retrieval Analysis

Purpose

This screen supports the analysis of reasons for the cancellation of Order Lines. All Inventory Units for a cancelled Order Line are listed as well as the disposability regarding each single property.

 INFO 

In order to support a quick analysis a couple of predefined filters are available.

 INFO 

For further information about Retrieval Strategies see Retrieval Strategies - Basic Conditions.
     
Screen

      
Fields

Fields are described in the screen Order Line Detail.

Field

Description

Inventory Unit Match
Load Unit ID Load Unit, where the Inventory Unit resides in.
Base Load Unit ID Base Load Unit of the Load Unit, where the Inventory Unit resides in.
No Inventory Hold Enabled, if the Inventory Unit is not on hold.
Inventory available Enabled, if the Inventory Unit availability status is "free" or "reserved".
Load Unit OK Enabled, if no error occurred on a current open transport with this Load Unit.
Load Unit not reserved Enabled, if the Load Unit reservation status is in "free" or "inventory" status.
Base Load Unit OK Enabled, if no error occurred on a current open transport with this Base Load Unit.
Base Load Unit free Enabled, if the Base Load Unit lock status is "free" if stacked Load Units.
Base Load Unit not reserved Enabled, if the Base Load Unit reservation status is in "free", "inventory" or "children reserved" status.
Aisle not locked Enabled, if the Aisle the Load Unit is stored in has the Lock Status and Technical Lock Status "available".
Level not locked Enabled, if the Level the Load Unit is stored in has the Lock Status and Technical Lock Status "available".
Location not locked Enabled, if the Location the Load Unit is stored on has the Lock Status and Technical Lock Status "available".
Right Location Usage Type Enabled, if the location usage type is in "virtual area", "virtual aisle", "bin" or "transport point".
Location OK Enabled, if the error code of the location is "OK".
No Dedicated Pick Face assigned Enabled, if no Dedicated Pick Face was assigned to the respective SKU.
