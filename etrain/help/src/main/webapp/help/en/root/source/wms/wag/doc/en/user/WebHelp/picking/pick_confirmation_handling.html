Pick Confirmation Handling

Purpose

This topic describes configuration of Pick Confirmation Handling.

     
Configuration

Pick Confirmation Handling assigns a handling or booking process configuration to Pick Zone and Pick Error Code combination.

A default Pick Confirmation Handling configuration is created by system initialization for each not yet configured pick error code and pick zone combination. When a non-default behavior is required, default entries should be changed.
