Picking Screen

Purpose

The Picking screen provides all picking functionality for all kind of load units at a Goods-to-Person Pickstation. Features include: detailed instructions, information for pick operations, and information regarding the buffered and ordered Source Load Units, and the current allocations of the Pick Container.

Pick operations include the standard case for confirmation of a single pick:

  • without any variation
  • with handling of inventory shortage
  • with further split pick operation
  • with advised overdelivery
  • with reverse picking functionality.

 INFO 

Provided functionality is described in Picking Screen Details.

 INFO 

Operational Picking process is described in Goods-to-Person Picking.
     
Overview

Frame

Description

Source

Where to pick from: Current Source Load Unit with number of remaining Picks processed at this Pickstation, Icon for current instruction regarding Source Load Unit with short instruction hint.

Pick

What to do within current pick operation: SKU to pick (with SKU description), Batch, Best before date, Quantity with Quantity Unit.

Destination

Where to pick: Current Pick Container with Put Location, Customer and Service, Icon for current instruction regarding Pick Container with short instruction hint.

Load Unit View

 Graphical view of the Load Unit. This view of Inserts (compartments) or stacked Load Units on Base Load Unit only displays if the related Load Unit type is subdivided into smaller rectangles, i.e. configured with a layout for geometrical stacking (LU type with grid sizes greater than 0).
  • white: not yet stacked LU on raster
  • red: LU or Insert is selected for the current picking operation
  • green: free and not selected
  • yellow: occupied and not selected

Order Information

Additional text information from Order and Order Line.

For a detailed description, see Picking Screen Details.
