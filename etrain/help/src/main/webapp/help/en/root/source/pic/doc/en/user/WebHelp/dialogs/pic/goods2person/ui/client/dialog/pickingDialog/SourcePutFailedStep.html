Source Invalid Scan

Brief Description

This screen appears if an invalid Source Load Unit is scanned.

      
Screen

      
Prerequisites
  • Unknown Source Load Unit is scanned
  • Source Load Unit is not in transport to the current Pickstation
  • Source Load Unit without active Pick is scanned.
      
Fields

For field descriptions, see Picking Screen Details.

      
Functions / Buttons

Function/Button

Description

Enter "Source Load Unit"

Enter scanned Load Unit. Occurs after an invalid scan where operator is requested to rescan.

New Source Load Unit is scanned onto Pickstations Source Location.
