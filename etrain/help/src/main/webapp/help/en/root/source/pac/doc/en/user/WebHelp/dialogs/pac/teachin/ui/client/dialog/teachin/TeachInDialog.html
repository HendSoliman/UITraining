Teach-In Station

Purpose

This page contains an overview of the Teach-In Station dialog.

Teach-In Stations are used to gather information on the different packaging variants for products handled in an automated warehouse.
The parameters are especially required for Automated Mixed Case Palletizing, AMCAP.

Besides dimensions and weight, secure processing of packages in the system requires additional parameters which affect the handling in the various sub-processes such as de-palletizing, conveying, storage and palletizing.
In addition, the Teach-In Station is also used to capture an image of the packaging unit which will be used within the process as an aid for identification.

The set of parameters is grouped into those which need to be explicitly measured or determined by inspection and those which can be automatically calculated.
The automatically determined parameters may be manually overridden for a specific packaging variant if this is necessary.
