Sortlane

Purpose

This topic explains how to handle sortlane problems.

Starting point of the analysis of faults related to sortlanes is the  ZUSO monitor. If a sortlane sequencing is jammed up, it can be cleared by using this dialog.

 

Resource Type

Link
Clear complete sortlane Explains how the sortlane can be cleared after an severe error
Cancel seqeunce Explains how to manually cancel an individual sequence number.

 

      
Clear Complete Sortlane

The buttton "Zuso Clearing START" initiate a clearing of the sortlane. Transport units drive to a transport point after the sortlane and request a new destination from the MFC. The reaction to this request is in general project specific.

 INFO 

Be aware that in addition to all transport units on the sortlane, all transport units which have been activated in the multishuttle are cleared as well.

The buttton "Zuso Clearing FINISHED" forces the end of a clearing. Usually the clearing ends automatically by a StatusMaterialFlowTelegram from the PLC.

 

      
Cancel Sequence

A sequence on the sortlane can be canceled by the button "Sequence STORNO". The MFC sends a CancelTelegram without TUId to the PLC.
