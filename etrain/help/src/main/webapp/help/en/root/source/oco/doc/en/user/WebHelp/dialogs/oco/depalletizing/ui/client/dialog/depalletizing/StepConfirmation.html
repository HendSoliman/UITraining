Confirm

Purpose

Operation mode "to container": appears after a valid source pallet and destination container are confirmed. The operator must confirm the packing of a specified quantity from the source pallet to the destination container.

Operation mode "to conveyor": appears after a valid source pallet is confirmed. The operator must confirm the packing of a specified quantity from the source pallet to the conveyor.

      
Screen

Operation mode "to container":

Operation mode "to conveyor":

      
Prerequisites
  • Valid source pallet and destination container are identified (in operation mode "to conveyor" only the source pallet)
  • Destination container provides free capacity (only for operation mode "to container").
      
Fields

De-palletizing Details.

      
Next Step

After successful confirmation, the next step is calculated by default.

Operation mode "to container"

  • Source Put, if the Source Load Unit is no longer available.
  • Destination Put, if the Source Load Unit is available, but no free Destination Load Unit is available.
  • Stay in this step, if Source and Destination Load Units are available and update the quantities.

Operation mode "to conveyor"

  • Source Put, if the Source Load Unit is no longer available.
  • Stay in this step, if the Source Load Unit is available.
      
Special Buttons

Button

Description

Confirm current repack process

Related Button or "Enter" key:

Same for both operation modes

With confirmation of the last put from the current Source Load Unit, the operator indicates if the Source Load Unit is empty. If the answer is unexpected, the operator is asked again. The current put is confirmed with following confirmation code:

  • OK: number of puts is greater than 1 or operator answered as expected.
  • Source empty: operator confirmed Source Load Unit as empty, but there is remaining Inventory booked onto the Source Load Unit.
  • Source not empty: operator confirmed Source Load Unit as not empty, but there is no remaining Inventory booked onto the Source Load Unit.

The next step is based on the default calculation "Next Step".

Cancel current step

Related Button or "ESC" key:

In operation mode "to container" - only available if no automatic scanning of destination is configured.

In operation mode "to conveyor" - only available if no automatic scanning of source is configured.
For both modes the next step is Source Put.

Open the Change Quantity screen

Related Button or "F4" key:

Only available in operation mode "to container".
See Change Quantity for details.

Book Source Load Unit as empty

Related Button or "F6" key:

Operation mode "to container": operator is asked if the Source Load Unit is empty. After the operator's confirmation, additional confirmation is required. The Source Load Unit is confirmed with confirmation code, "Source empty" and quantity zero.
Without a second confirmation, nothing is done.

Operation mode "to conveyor": operator is asked to confirm the quantity that is already de-palletized. This quantity is predefined with the value of the field “already done”.
The Source Load Unit is confirmed with confirmation code, "Source empty" and the given quantity. (See "Source Empty" for details).

The next step is based on the default calculation "Next Step".

Book wrong SKU

Related Button or "F7" key:

Same for both operation modes

Confirm the Source Load Unit contains an incorrect SKU. The Source Load Unit is then confirmed with confirmation code, "Wrong SKU" and quantity 0.

Open the book breakage screen

Related Button or "F8" key:

Same for both operation modes

Enter broken/damaged quantity. Quantity entered will be deleted.
See Book Breakage for details.
