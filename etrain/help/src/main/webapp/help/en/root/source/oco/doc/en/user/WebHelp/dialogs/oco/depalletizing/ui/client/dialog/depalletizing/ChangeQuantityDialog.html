Change Quantity

Purpose

This displays if an operator selects “change quantity” in the Confirm step (operation mode "to container"). The operator specifies the changed capacity of the destination container.
The maximum quantity may not exceed the current quantity of the Source Load Unit.

      
Screen

      
Fields

De-palletizing Details.

      
Special Buttons

Button

Description

Change current capacity

Related Button or "Enter" key:

Change quantity. Update remaining puts and expected weight due to new destination container capacity. Changes are not saved and lost if a new Source Load Unit scan.

Cancel current state

Related Button or "ESC" key:

Revert to the Confirm step without changes.

Save changes

Related Button or "F3" key:

Change quantity. Update remaining puts and expected weight due to new destination container capacity. Changes are saved permanently and used if a new Source Load Unit Scan.
