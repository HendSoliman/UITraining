Screen |  Fields |  Buttons

Create Order Lines

Purpose

Create Inventory Order Lines by defining the requested Stock Keeping Units and quantities.

      
Screen

      
Fields

Quick Order Creation.

      
Special Buttons

Button

Description

Key: Enter

Create an Order with the given settings and all selected Order Lines, close this screen and return to the initial step.

Key: Escape

Return to the initial step for changing the general Order settings without saving already selected entries.

Key: F2

Create an Order with the given settings, all selected Order Lines, and stay in this step for creation of another order with the same general Order settings.

Add Order Line with selected SKU Quantity Unit to list of Order Lines.

Remove Order Lines from list.
