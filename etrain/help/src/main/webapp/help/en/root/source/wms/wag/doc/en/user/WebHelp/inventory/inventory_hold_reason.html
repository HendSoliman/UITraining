Inventory Hold Reason

Purpose

This topic describes the configuration of the Inventory Hold Reason.

      
Configuration

The Inventory Hold Reason specifies a reason with an attached configuration for inventory holds and single inventory unit holds. Inventory Hold Reasons that must be configured depend on host requirements and on the different configured Inventory Holds.

The following Inventory Hold Reason attributes control business logic. They are not predefined by the system and must be configured:
