Print Pick Labels

Description

For each Pick, a corresponding Pick Label can be printed. One label contains all Pick related information such as SKU, requested and picked quantity etc. A list of labels is also printable for entire Pick Walk as a type of Picking List.

      
Triggers and Actors

If Offline Picking, the following persons must perform this when certain Picks labels are needed.

  • Control Center Supervisor
  • Picker.
      
Prerequisites
  • At least one Pick Walk is available.
  • Printing configuration is complete.
  • A default label printer for the related Zone is configured (Pick Zones).
      
Work Flow

Step

Description

1.

Open the Pick Walk screen.

2.

Select one or more Pick Walk(s) from the Pick Walk grid.

3.

If labels for all Picks of selected Pick Walks are needed, click the "Print Pick Label(s) for selected Pick Walk(s)".

To print labels for certain Picks, use the "Print Pick Label(s) for selected Pick" button on the Retrieval Administration screen.
     
Result

Labels for selected Picks print on the default Label Printer.
