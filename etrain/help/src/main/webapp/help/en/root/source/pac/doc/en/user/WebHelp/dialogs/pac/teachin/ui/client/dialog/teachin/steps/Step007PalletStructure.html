Teach-In Dialog: Pallet Structure

Purpose

This step is used to verify information about the supplier pallet for the product to be teached.

The information to be checked is

  • Items / Layer
  • Layers / Pallet

These data are transmitted from the commercial system (ERP).
However, they must be verified since especially the number of items per layer must be correct because the process auf automatic de-palletizing is strongly dependent on this information.

Besides this, the user can also enter a measured value for the pallet height which, together with the number of layers per pallet is used to determine a reference value for the packaging height.

Exact knowledge of the packaging height is an important information for palletizing.

The value determined from the supplier pallet is of very good quality since it includes the effect of compression which is not available when measuring a single carton only.

Screen

 

      
Fields

Field

Description

Load Unit Type

A list of touchable buttons represents the available Load Unit types

Length

Length (in mm) of the source pallet.

The initial value is taken from the selected Load Unit Type.

In case that the source pallet shows overhang, the real length shall be measured and entered instead of the default value.

Width

Width (in mm) of the source pallet.

Items/Layer

Number of items per layer of the source pallet.

The initial value is taken from the Load Unit Capacity of the selected SKU Quantity Unit.

This entry must be compared with the real number of items on the pallet.

In case of a difference, the correct number must be entered.

Layers/Pal

Number of layers on the source pallet.

Base Height

Base height of the empty pallet (in mm).
The value is taken from the selected Load Unit Type.

Overhang

If measured length and/or width exceed the measures of the selected Load Unit Type, the difference is shown as overhang (in mm).

Total Height

The measured value for the pallet height (including base pallet) must be entered here  (in mm).

# Slipsheets

The actual number of slip sheets must be specified here.

A system parameter specifies the height of a single slip sheet.

The total height of slip sheets is subtracted from the pallet height when the layer height is calculated.

Layer Height

The calculated height of a single layer.
