Loads

Purpose

A Load comprises all Dispatch Load Units which are shipped together, usually in one truck or any other loading equipment.
The Load Administration Dialog is used to configure the Loads and provides the basic functionality for all lifecycle related operational steps as starting and finishing the loading.

In addition all Orders which are assigned to a Load are listed as well.

      
Screen

      
Sub-Screens

  • Assign Load to Dock
  • Details of the Orders of a Load are described in the screen Order Detail.
      
Special Buttons

Button

Description

Assign the selected Load directly to a Dock in case the Load is not assigned to a Route (see Assign Load to Dock).

Authorize the selected Load for Staging.

Authorize the selected Load for Loading. The Load must be already assigned to a Dock.

Complete the selected Load. The not yet loaded Dispatch Load Units will be assigned to a new Load which will be created directly.

Close the selected Load with printing the shipping manifest.

Confirm the departure of the selected Load with finishing the Route and release of the occupied Dock.
