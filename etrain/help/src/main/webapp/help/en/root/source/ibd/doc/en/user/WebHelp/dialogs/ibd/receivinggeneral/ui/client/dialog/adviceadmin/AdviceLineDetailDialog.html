Field

Description

Advice

Advice of this Advice Line.

ID

Identifies the Advice line uniquely within an Advice.

Overview

Status

How far the Advice Line was processed:

  • pending (Advice Line is completely prepared for processing and available for release)
  • receiving complete (receiving process of the Advice Line finished, i.e. inventory creation was completed)
  • complete (Advice Line is completely processed and all related Load Units arrived at final storage locations)
  • to confirm (Advice Line is ready for confirmation message to the host system)
  • finished (All required tasks for the Advice Line are finished. If required, advice was confirmed to the host).

SKU
SKU

Stock Keeping Unit of the announced inventory, specified by the Advice Line.
Supplier SKU

SKU of the supplier. May be for information purposes.
SKU Description

Description of the SKU.
Batch

Batch in which inventory was produced.
Best Before Date

Date before it is recommended to consume inventory. Used as FIFO criterion when goods are selected for an order.

Quantity

Quantity Target

Total inventory quantity announced by the Advice Line. Expressed in relation to the 'receiving quantity unit' of the Advice Line.
Quantity Current

Quantity of the so far created inventory for this Advice Line. Expressed in relation to the 'receiving quantity unit' of the Advice Line.
Receiving Quantity Unit

Each quantity in the Advice Lines and in the Advice Line Load units. Expressed in relation to the receiving quantity unit. LMS can only create inventory for an Advice Line, if the receiving quantity unit is configured in for the SKU in the SKU master data.

Inventory Detail

Mark
Special Inventory Mark

Can classify an inventory unit to have special requirements for use for customer orders. An inventory unit without a special inventory mark can be used freely to satisfy customer orders.

  • inventory on commission
  • order inventory
Special Inventory Reference

Links inventory marked as special inventory to a corresponding object. Ex: Inventory on commission can be identified by the commission ID.

Classification
Hold Reason

If advised inventory must be set on hold during creation and for what reason. If null, inventory is not to be set on hold. A use case for this attribute is the SAP quality state.
Quality Classification

Customers often order goods with a minimal quality classification (e.g. quality class A, 2nd choice). Quality classifications are ordered to be able to define, for example that quality class A is better than quality class B.
Product Code

Product code defined by the Host or manually by a user.
Date of Manufacture

Date/time when inventory was manufactured.

ERP
Host Quantity Target

Total inventory quantity announced by this Advice Line by the host. Expressed in relation to the 'receiving quantity unit' of the Advice Line. May be corrected, for example if a receiving quantity unit change.
Host Quantity Unit

Each quantity in Advice Lines and Advice Line load units is expressed relation to the receiving quantity unit. The LMS can only create inventory for an Advice Line, if the receiving quantity unit is configured for the SKU in the SKU master data.

Supply

Arrival Date

Set by receiving management when the first inventory is created for this Advice Line.

Date of Receipt

The date of receipt specifies the original date when the advised inventory was received in the warehouse. If this date is not null, this date is used instead of the arrival date to as date of receipt when creating inventory.

Tolerance
Overdelivery Tolerance (%)

Factor, in percent, up to which overdelivery is allowed.

Ex: Advised target quantity is 100. Overdelivery tolerance is 8. Delivery up to 108 is allowed.
Unlimited Overdelivery Allowed

If unlimited, overdelivery is allowed for this Advice Line.
Underdelivery Tolerance (%)

Factor, in percent, to which underdelivery is allowed.

Handling
Receiving Workstation

Workstation ID used to create the advised inventory.
Receiving Location

Location ID on which the advised load unit was created.

Advice & Advice Line

Delivery Text

Displays in receiving dialogs. Usually contains handling instructions for the advised inventory. If not set in the Advice Line, delivery text of the Advice displays.
