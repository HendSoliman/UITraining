Creation of a Stocktaking Order

Brief Description

To execute a Perpetual stocktaking with an explicit planned Counting Order, a Stocktaking Order must be created. This describes the creation of a new Stocktaking Order with SKU, Load Units or Locations. With subsequent (manual or automatic) release of the Stocktaking Order, the system creates Counting Orders separately per Stocktaking Zone. In automated Areas, a Counting Order will be retrieved on demand for execution of Counting operations. In manually operated Areas, manual execution is based on counting lists and final booking of Counting results on the administration screen.

      
Triggers and Actors

This starts when:

  • a Counting process for SKU(s) should be executed
  • a Counting process for Load Unit(s) should be executed
  • a Counting process for Inventory Unit(s) residing on one or multiple Locations, Aisles or Areas should be executed.
      
Prerequisites
  • Ensure that no open orders exist for the selected criteria
  • Ensure that no open movements exist for the selected criteria.
      
Required Resources
  • Orders.
      
Work Flow

Step

Action

1.

Open the Orders screen and click "Create a new Order".

2.

Create a new Stocktaking Order with the following basic attributes:

  • Type "stocktaking"
  • Client (according to given selection)
  • Priority (value > 0)
  • Configuration remain "incomplete".

3.

Create new Order Lines using the button

Order Lines can be created with selection criteria: Inventory, Load Unit or Location. One Order could have Order Lines of each of these criteria types.

4a.

Inventory Selection (SKU):

Use the filter to restrict the selection. Select the desired entries (multi-selection is possible).
If the yet selected entries should be extended to certain number of positions (started from the top), enter the total number of desired positions given in the "Selection Limit" and click the button

To create all Order Lines for the selected SKU, click "OK".

4b.

Load Unit Selection:

 INFO 

Select the desired Area for the Load Units first.

Use the filter to restrict the selection. Select the desired entries (multi-selection is possible).
If the yet selected entries should be extended to certain number of positions (started from the top), enter the total number of desired positions given in the "Selection Limit" and click the button

To create all Order Lines for the selected Load Units, click "OK".

4c.

Location Selection:

 INFO 

Select the desired Area for the Locations first.

Use the filter to restrict the selection. Select the desired entries (multi-selection is possible).
If the yet selected entries should be extended to certain number of positions (started from the top), enter the total number of desired positions given in the "Selection Limit" and click the button

To create all Order Lines for the selected Locations, click "OK".

 INFO 

To select an entire Aisle or Area filter and confirm a Location with usage type, "Aisle" or "Area".

5.

To finalize a Stocktaking Order, click "Modify" to change the configuration status of the order to "complete" in Orders screen.

The Order becomes available for manual or automatic release.
     
Result

A Stocktaking Order, for the desired criteria (Inventory, Load Units, Locations) is created and available for release. With release of this Order, all relevant Inventory Units for the given criteria are identified by the system and considered for subsequent generation of Counting Orders.
