Shorts Report

General information

This report lists all Pick Order Lines which are completed with shortages.

For each shortage the corresponding Error Code and Reason are shown.

      
Screen

Field

Description

SKU

The requested Stock Keeping Unit.

Pick Quantity

The requested quantity in Pick Quantity Units for this Pick Order Line.

Pick Quantity Unit

The ID of the SKU Quantity Unit which is used for display purposes within Pick operations.

Picking Start Time

The Pick Start Time denotes the point in time the Pick Order Line proceed to status "active".

Error Code

The error code identifies the last error which occurred with this Pick Order Line. Possible values are:

  • OK (Pick Order Line successfully completed)
  • missing quantity (Pick Order Line not fully picked due to Inventory shortage)
  • load unit missing (Pick Order Line not fully picked due to no Load Unit found)
  • wrong SKU (Pick Order Line not fully picked due to wrong SKU found)
  • wrong load unit (Pick Order Line not fully picked due to wrong Load Unit found)
  • wrong best before date (Pick Order Line not fully picked due to wrong Best Before Date of the Inventory found)
  • wrong packaging level (Pick Order Line not fully picked due to wrong packaging level)
  • breakage (Pick Order Line not fully picked due to found Inventory is broken)
  • blocked (Pick Order Line not fully picked due to Location to be picked from is blocked and could not be accessed)
  • underdelivery (Pick Order Line not fully picked due to explicitly Picking of less than target quantity)
  • overdelivery (Pick Order Line not fully picked due to explicitly Picking of more than target quantity)
  • split (Pick Order Line was split due to Pick Container was full or required use of different Quantity Unit)

Error Reason

The error reason provides a free text to explain the reason of the error.

Cancel Reason

The cancel reason specifies a more detailed reason of the cancellation of this Pick Order Line. Possible values are:

  • none (The Pick Order Line is not cancelled)
  • underdelivery tolerance (The Pick Order Line was cancelled due to violated underdelivery tolerances)
  • overdelivery tolerance (The Pick Order Line was cancelled due to violated overdelivery tolerances)
  • operator (The Pick Order Line was manually cancelled by an operator)
  • previous phase line cancelled (The Pick Order Line was cancelled due to the related Previous Phase Pick Order Line was cancelled)
