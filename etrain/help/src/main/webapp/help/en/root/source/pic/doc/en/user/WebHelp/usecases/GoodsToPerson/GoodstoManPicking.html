Goods-to-Person Picking

Brief Description

Picking at a Goods-to-Person Pickstation occurs on the Picking Screen. It provide all functionality for picking purposes at a Goods-to-Person Pickstation for all types of Load Units. Features include detailed instructions, pick operation information and different types of pick confirmation. Reverse picking execution and the splitting of a pick operation are also provided. It also supports external trigger for automatic process flow with signaling of the conveyed Load Units and external signals for pick confirmations (e.g. by additional Put-to-Light application).

 INFO 

Picking at the RapidPick stations (A1, A6 and M6) is described separately in Goods-to-Person Picking with RapidPick Dialog.

Use Case

Description

Pick Operation

Pick operations include the standard case for confirmation of the single pick

  • without variation
  • with handling of inventory shortage
  • with additional split pick operation
  • with advised overdelivery
  • with reverse picking functionality.

New Pick Container

Perform if additional Pick Containers are required at a Pickstation.

New Source Load Unit

Perform if all required Pick Containers are available and the start of pick operation is required.

Send away Pick Container

Perform if a Pick Container finished due to processing the last Pick of related Pick Mission or if an operator confirms the Pick Container is full.

Send away Source Load Unit

Perform if all active Picks of current Source Load Unit are finished or a Source Load Unit is scanned that is not needed until the next Pick Missions at this Pickstation.

Perform Zero-Cross Check

Perform if zero-cross or nearly zero-cross check of Source Load Units inventory unit is required because remaining quantity falls below configured SKU limit.
