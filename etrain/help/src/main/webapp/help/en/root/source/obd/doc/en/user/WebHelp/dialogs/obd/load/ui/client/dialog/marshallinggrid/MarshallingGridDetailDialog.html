Marshalling Grid Detail

Purpose

This screen opens when creating, viewing or modifying a Marshalling Grid.

      
Screen

      
Fields

Field

Description

ID

Unique identification of the Marshalling Grid.

Description

Description of the Marshalling Grid.

Status

Status

The status reflects the current use of the Marshalling Grid. Possible values are:

  • open (The Marshalling Grid accepts Load assignments)
  • closing (The Marshalling Grid accepts no further Load assignments but is still occupied. After all assigned Loads are finished the status will be  automatically set to 'closed')
  • closed (The Marshalling Grid is closed and accepts no assignment).

Configuration Status

The Configuration Status of Marshalling Grid defines the possible use by the values:

  • incomplete (Configuration is not finished and the Marshalling Grid is not for use)
  • complete (Marshalling Grid is available for use).

Occupation Status

The occupation status provides the values:

  • occupied (Currently occupied by a loading operation in progress)
  • free (Currently no loading operation in progress)
  • reserved (Reserved for a future loading operation).

Active Load

The Load which is currently staged on the "occupied" Marshalling Grid. In case of "Multiple Loads allowed" this Load will be loaded next.

Blocking Status

The blocking status reflects a current temporarily blocking situation. Possible values are:

  • not blocked (The Marshalling Grid is currently not blocked and accessible)
  • blocked (The Marshalling Grid is temporarily blocked (for a specific operation) and currently not accessible for standard operations).

Blocking Reason

Reason why the Marshalling Grid is temporarily blocked.

Zone

The identification of all locations belonging to the Marshalling Grid is done through the assignment of a Zone.

Dock

The Dock which is supplied by this Marshalling Grid for loading of the staged Dispatch Load Units.

 INFO 

This assignment is not mandatory and could be done dynamically after a Load has been marshaled.

Automatic Loading

A Marshalling Grid can be configured for automatic loading if no explicit loading operation through scanning of the loaded units is performed.
In case of enabled and a Dock is yet assigned the load process will be booked immediately after a unit has been confirmed to be arrived in the Marshalling Grid. In case of a Dock is not yet assigned this is done after assignment of a Dock without any additional user interaction.

 INFO 

"Automatic Loading" can be activated only if "Multiple Loads allowed" is disabled in order to prevent erroneous loading to the wrong truck.

Multiple Loads allowed

Specifies whether or not it is possible to assign multiple Loads to a Marshalling Grid at the same time. In case of disabled only one Load could be assigned to a Marshalling Grid at a time.

 INFO 

Independent of this configuration it is never allowed to stage Load Units of different Loads on the same Marshalling Grid simultaneously.
