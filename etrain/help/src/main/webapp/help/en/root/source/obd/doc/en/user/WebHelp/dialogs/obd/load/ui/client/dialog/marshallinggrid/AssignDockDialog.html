Assign Marshalling Grid to Dock

Purpose

The detail dialog is opened by applying the "Assign Dock" button in the Marshalling Grid Administration dialog and provides the functionality to assign or reassign the Marshalling Grid to a Dock.

      
Screen

      
Fields

Field

Description

ID

The selected Marshalling Grid.

Dock

The new selected Dock that should be assigned to the Marshalling Grid.
