This topic describes the configuration parameters of Picking.

PIG - Picking

Key

Type

Dimension

Default

Description

Picking.PickStructureRemainingTime

int

 

168 (7 days)

Defines the time after which PIC structures are deleted by the cleanup job (specified in hours).

Picking.InjectPickInActiveMission

boolean

 

false

Defines whether new Picks created while replenishment or replacement reservation are tried to get injected into active Pick Missions.

 INFO 

To ensure that Pick Missions which kept open after the failed replacement reservation are completed activate CloseMissionsWithDanglingReplacementReservationDemon

Picking.PickNumberRange.pickNumberRangeId

String

 

DefaultPickNumberRange

Id of the number range of pick IDs.

Picking.PickNumberRange.pickMissionNumberRangeId

String

 

DefaultPickMissionNumberRange

Id of the number range of pick mission IDs.

Picking.BinsAreAssignedToZones

 boolean   false Defines whether Bin Locations in storage areas are assigned to Zones of type PICK in addition to the assignment to the type PICK CONTROLLER. Only in this case this will be checked and an alarm is raised if a putaway operation is finished with open Pick Order Lines but without having a PICK zone at the destination location.

Picking.PrintingDateFormat

 String   dd.MM.yyyy Defines the format in which the date will be printed.

Picking.PickMissionRequestFailedWaitingTime

 int   180 Defines how long to wait until a request is considered a failure. (specified in seconds)
