Use Case Overview

Purpose

Describes all use cases which can be performed in a Person-to-Goods Pick Controller. Use cases are valid for specific functions and configurations of Route Based Picking with respect to the special tasks of Person-to-Goods Picking.

      
Overview

Use Case

Description
Offline Picking: Create Pick Walks

Starts Offline Picking with activation of one or multiple Pick Walks and prints configured default Pick List.

Offline Picking: Finalize Pick Walk

Finalizes Offline Picking with confirmation of single, differing Picking results of one Pick Walk.

Print Pick Label(s)

For Offline Person-to-Goods Picking a label can print for each Pick.

Print Pick List(s)

For Offline Person-to-Goods Picking the Pick List provides single instructions for pick operations during the Pick Walk. Each Pick of the Pick Walk is represented as a single list item.

Person-to-Goods Picking with RF-Terminals

Picking in Person-to-Goods Pick Zones is done using the Dialog RF-Picking which provides all functionality for picking purposes with Person-to-Goods picking strategies.
