Navigation

Purpose

This chapter describes the navigation between dialogs.

 

      
Overview

Topic

Description

Implicit Navigation

The implicit dialog navigation is offered wherever it is possible to navigate from a data set to its references

Explicit Navigation

Explicit dialog navigation is offered additionally where it makes sense, without having references
     
Implicit Navigation

Within data grid dialogs it is possible to navigate from a given data set to its references. Navigation commands are offered implicitly for all references of a data set as long as their key attributes are available.

The following navigation commands exist:

Navigation Command

Description

Navigate to the administration dialog of the reference. The navigated dialog will be opened and filtered by the reference.

NOTE: Navigating to an previously opened dialog modifies its current filter settings. To save previously done filter settings you may "pin" the dialog.

Shows the reference in its detail dialog

Navigate to Audit Trail dialog. Opens Audit Trail dialog and filters Audit Trail data for the given reference. 

 

Available navigation is offered by a popup menu when moving the mouse over cells displaying references.
