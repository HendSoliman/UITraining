OrderFulfillment.
NotifyImmediatelyOnArrival

boolean

OrderType (prefix:
ORDERTYPE:)

false

This property defines if the Inventory Management is to be informed on each single arrival of a Load Unit at the final destination of the related Order or if the notification is to be done after the whole Order has been finished.

 INFO 

Note: As a result of the notification the Load Unit looses its relation to the Order and may even be deleted. Therefore this property should be set to true only if no host confirmation of the Order is required or if the host confirmation does not include Load and/or Inventory Unit specific information.

boolean

  

false

If set to true, Orders that causes a concurrent access exception are skipped temporarily in the second reservation phase of the reservation type "two phase".
