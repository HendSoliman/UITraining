Load Unit

Purpose

This state allows users to enter (scan) a Load Unit for Receiving. An existing or a new Load Unit can be entered. For a new Load Unit, a Load Unit Type must be selected.

 INFO 

This step is not available for Returns.
     
Screen

      
Fields

Receiving

      
Special Buttons

Button

Description

If an existing Load Unit (for the entered ID) this loads and displays. The screen proceeds to the next step "Complete Receiving".

If no Load Unit for the entered (scanned) ID can be found, a selection screen for available Load Unit Types appears.

Upon type selection, the next state diplays.


If using an existing Load Unit, the following constraints must be fulfilled:

  • Load Unit must be at current Workstation
  • Load Unit status must be "no TO"
  • Load Unit must be empty (or at least one insert on it).

Previous screen state displays.

Print Labels screen opens.

 INFO 

Enabled only if the Product Code of the SKU is available.

For additional deposit on yet stored Load Units. Load Unit selection is restricted with the following constraints:

  • Load Unit is stored.
  • Reservation Status of the Base Load Unit is free.
  • Lock Status of the Location and the Aisle is available.
  • Load Unit is empty (or at least one Insert on the Load Unit).
With multi-selection, any entry can be selected. All selected Load Units directly get an Operator Retrieval Order to the current Workstation by clicking "OK".

Cancel process and go to initial state.
