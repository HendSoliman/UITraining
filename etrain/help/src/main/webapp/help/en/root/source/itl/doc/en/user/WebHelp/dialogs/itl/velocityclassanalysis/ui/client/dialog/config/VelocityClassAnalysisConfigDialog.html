Velocity Class Analysis Configuration

Purpose

Velocity Class Analysis takes place for the configured velocity classes (inventory turnover ratio) of the SKUs. For each velocity class, the minimum number of accesses required is defined to get the appropriate velocity class assigned. Single access is defined by one, singular shipment booking of each Inventory Unit.
Velocity Class Analysis can consider up to 3 time periods each with its own weightage for calculation of velocity classes, i.e. the Inventory turnover ratio during one time period is considered with assigned weightage for calculation of the overall Inventory turnover ratio.
The result represents the basis for calculation of the velocity classes.
Here the definition is done for

  • the limits for the number of accesses of the velocity classes
  • the considered time periods and assigned weightage for calculation of the velocity classes.
