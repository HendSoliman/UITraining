Failed Logon History Dialog

Purpose

This screen displays all failed login attempts. Every authentication failure is registered and logged by the system.

 

      
Functions

Operators can perform the following tasks:

  • View failed logons

Screen

 

      
Fields

Field

Description

Logon name

 Name used by the user to log on.
Logon time Date/time when the logon attempt failed.
Attempt Number of failed logon attempts performed by the user.
Terminal ID Number of the terminal at which the logon was attempted.
Active If the failed logon attempt is still active. The user is deactivated if the number of logon attempts exceeds the permissible maximum number of logon attempts. If a successful logon, all failed logon attempts are deactivated.
User ID Unique ID of the user who tried to log on.
Logon ID Number used by the user to log on.
