Worker Detail

Purpose

This screen opens when creating, viewing or modifying a Worker.

      
Screen

      
Fields

Field

Description

User

Unique ID of the assigned User.

Status

Active / Inactive (if inactive, a Worker cannot be used as a valid resource by the system).

Worker Group

Each Worker is member of exactly one Worker Group which defines the allowed workload.

Current Work
Current Work Reference

The ID of the referenced business object relates to the specific operation. E.g. in case of picking this is the ID of the Pick Walk.
Type of Reference

The type of the reference currently assigned.

  • undetermined ready for work and assignment of work.
  • pick walk one more more pick missions have been assigned.
  • counting order a stock taking order has been assigned.

Last Confirmed Location

Location of the last confirmation which relates to the specific operation. E.g. in case of picking this is usually the last confirmed Pick Face.

Last Confirmation Time

Time of the last confirmation.
