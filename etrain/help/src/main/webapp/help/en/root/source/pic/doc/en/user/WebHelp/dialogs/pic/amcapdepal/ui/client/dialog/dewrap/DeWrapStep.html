Build Pallet Step

Purpose

This screen is used for de-wrapping operation. The operator can enable/disable the check boxes to choose the reason. Then the confirmation reason and the corresponding function button will enabled.

 

      
Prerequisites
  • Current workstation is auto-depalletizing workstation.
  •  Source pallet has arrived at the check location of current auto-depalletizing workstation.

 

      
Screen

De-wrap rainbow pallet.

 

 

Partial depalletizing for automatic depalletizing. Need to set "Allow Partial-Depalletizing" attribute of DepalletizerStation as true.

 

 

DeWrap source pallet which is used for preventive calibration. Hit message will dispaly

 

 

DeWrap single SKU

 

 

De-wrap single sku with confirmation reason

 

 

De-wrap single sku with partial-depalletizing allowed

 

 

De-wrap single sku with order cancelled, then need to send it to clarification

 

      
Fields

Field

Description

Pallet ID

Valid Load Unit ID for identification the Source Pallet.

Total Layers

The layer number of all the products stored in the source pallet.

Product Code

Code that identifies this SKU in this packaging or Quantity Unit. For example, the European Article Number.
Depalletizing Number of Layers The layer number of the products need to be depalletized stored in the source pallet.
Unit Sku Quantity Unit
Quantity The package number based on current packaging-type.
Number of Layers to De-wrap The number of layers need to be depalletized.
Packaging Variant Click to choose the correct packaging variant for inventory unit

Confirmation Reasons

Field

Description

Miss Quantity

The Quantity is wrong

Inventory breakage

The inventory 's package is broken

Missing Load Unit

The load unit is missing
Wrong SKU The sku is wrong
Wrong Best Before Date The Dewrap date is not before the "Best Before Date"
Wrong Load Unit The load unit is wrong
Wrong packaging level The package variant is wrong

     
Procedure

Step

Description

1.

After check the source pallet, disable/enable check box for corresponding situation. If no exception case, click Start Auto-Depal button to do auto depalletizing.

2.

If confirmation reason is necessary, select the right confirmation reason.
3 Double confirm the Packaging Variant for each layer.
4 If "Allow Partial-Depalletizing" is set to true for current depalletizer station, change the value of field "Number of De-wrapped Layers" to do partial depalletizing.
5 Click the function button to do proper handling.
