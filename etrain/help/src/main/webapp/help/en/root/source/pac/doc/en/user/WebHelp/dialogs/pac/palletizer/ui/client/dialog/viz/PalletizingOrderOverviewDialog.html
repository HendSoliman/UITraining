Palletizing Order Overview Dialog

Purpose

This Screen is used to display pallet layouts for RAPIDPAL.

 

      
Functions
  • Display of all Orders associated with RAPIDPAL service qualifier
  • Display of all Pick Containers that are used for a selected Order
  • Display of the Pick Order Lines and Pallet Planning Results for that container. Additionally show a 3D-representation of the layout of that pick container.
  • The slider is used to manually track the build of the pallet as it follows the build sequence.
      
Overview
  • Order
  • Pick Container 
  • Pick Order Lines
  • Palletizing Result
  • The colors used to display the packages are defined in the Packaging Type (PACT) dialog

3D Widget Interaction

The 3D Widget utilizes a couple of interaction methods

Mouse interaction:

  • left click & drag rotates the entire pallet on its x - axis.
  • mouse wheel zooms the pallet in and out.
  • right click shows a context menu containing data on a selected package on the pallet.

 

Keyboard interaction:

  • W, A, S, D - keys move the camera in 4 directions
  • Arrow - keys rotate the entire pallet around its x and y - axis
  • Page up / down - keys move the entire pallet along the y - axis
  • Plus (+) and Minus (-) keys "explode" the pallet layers
