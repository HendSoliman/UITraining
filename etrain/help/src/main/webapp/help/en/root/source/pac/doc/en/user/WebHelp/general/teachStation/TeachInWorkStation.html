Teach-In Workstation

Purpose

This chapter describes the Teach-In workstation which is used for physical measurements of several Teach Parameters.

 

      
Workstation

 

 

      
Measuring Devices

The Teach-In Station is equipped with several devices for measuring of packaging parameters.

Device

Description

Length/Width Meter

This device is used for measuring length and width of the teach item.

The teach-item must be placed in upright position on the base plate.

For measuring, the slide must be pushed to the left until the packaging is slightly pressed against the fixed plate at the left.

For measuring the length, the teach-item is placed with the long side from left to right.

For measuring the width, it must be turned by 90Â° with the short side from left to right,

Display Length/Width

The display shows the actual measure of the Length/Width meter.

Push buttons are available for calibration.

 

Height Meter

The device is used for measuring the Height of the teach item.

Display Height

The display shows the actual measure of the Length/Width meter.

Push buttons are available for calibration.

Scale

The scale is placed underneath the measuring support.

It is used for measuring the weight of the teach item.

When measuring, it must be strictly avoided to place any item except the teach item on the measuring support.

Display Scale

Display of the scale showing the actual weight.

Operating controls are available for setting the tare and for calibration of the scale.

Toppling Plate

When measuring toppling angles, the teach item is placed on the toppling plate in an upright position.

For measuring "Toppling Length" the long side shows from left to right.

For measuring "Toppling Width", the short side shows from left to right.

Toppling Handle

The Toppling Handle is turned to change the angle of the Toppling Pallet.

The Toppling Angle is determined when the teach-item starts to topple.

At this moment, the Toppling handle is no longer turned and the teach-item is taken off the Toppling Plate.

Display Toppling Angle

The display shows the actual angle of the Toppling Plate.

Push buttons are available for calibration.

Digital Measuring Slide

This device is used for measuring the following parameters (for detail, see Teach Parameters Overview)

  • Edge radius
  • Nesting Height
  • Additional Height (Vertical Shift)
  • Pack Overhang Reduction
  • Support Edge Extension
  • Edge Reduction Top Length
  • Edge Reduction Top Width
  • Edge Reduction Top Height
  • Edge Reduction Bottom Length
  • Edge Reduction Bottom Width
  • Edge Reduction Bottom Height

The device has an own electronic display showing the actual extension of the measuring slide.

 

      
User Interface

Device

Description

Display

A touch screen is available for user interactions with the Teach-In Station Dialog.

The screen can also be used for maintenance of the Soft PLC in the MicroBox controlling the devices. 

Keyboard

The keyboard with trackball is used for data entry.

During normal teaching, all entries can be done using touch operations.

Camera

The camera is used for taking a picture of the teach-item.

Hand Scanner

The hand scanner is used for identification of the teach-item by scanning the product barcode.

 

      
Other Equipment

Device

Description

Cable for Power Supply

The cable is used for charging the battery.

It can also be used as power supply for stationary operation

Battery

The Battery is used as power supply for the entire Teach-In Workstation during autonomous operation

WLAN antenna

The antenna is used for connection to the IT network and the WCS server.

Cabinet with Siemens MicroBox

The cabinet hosts the electronic equipment of the workstation.

The main part is a Siemens MicroBox

  • with a windows system running a browser with the Teach-In Station dialog having connection to the WCS server
  • and a soft PLC controlling the measuring devices and communication the results to the WCS server.

A main power switch is available.

Before power is switched off, the system must be closed down regularly.

Calibration Mark Length/Width

The ground plate shows a mark with a specific length for calibration of the Length/Width meter

Calibration Mark Height

The back wall shows a mark with a specific height for calibration of the Height Meter
