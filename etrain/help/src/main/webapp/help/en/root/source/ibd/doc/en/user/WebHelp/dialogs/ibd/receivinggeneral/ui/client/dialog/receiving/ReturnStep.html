Return

Purpose

This state displays after a valid Advice and Advice Line were loaded and the Advice Type is "Return". It provides functions to receive the desired quantity into an empty Load Unit and into an existing Load Unit with the same SKU and Batch.

 INFO 

When receiving into an existing Load Unit with the same SKU and batch, the complete quantity is always received.

Fields

Receiving

      
Special Buttons

Button

Description

When no stored Load Unit was requested for the current Advice Line, the operator is asked what to do with the entered (scanned) Load Unit.

Click Cancel to stay in current state. Click OK to proceed with normal Receiving process (see confirmation of step "Load Unit"). If a Load Unit was already requested for the current Advice Line (using F7 function), no other Load Units can be used for Receiving. The screen requires a scan of the requested Load Unit.
When a valid Load Unit was entered (scanned), the screen proceeds to the step "Complete Receiving".

Previous screen state displays.

Print Labels screen opens.

 INFO 

Enabled only if the Product Code of the SKU is available.

For Return Advices only if deposit the same Inventory (SKU and batch) on Load Units stored yet is desired. Load Unit selection is restricted with the following constraints:

  • Load Unit is stored.
  • Reservation Status of the Base Load Unit is free.
  • Lock Status of the location and the Aisle is available.
  •  Load Unit contains the same SKU and batch (of current Advice Line).

Click "OK" for a new Advice Line Load Unit in status "pending" to be created (for the complete remaining quantity).

 INFO 

With this step, Receiving will only be allowed into this new assigned Load Unit.

Advice Line data (displayed in upper frame "Advice Data") is released for changing. Changeable fields are:

  • Batch
  • Best Before Date
  • Destination.

Cancel process and go to initial state.
