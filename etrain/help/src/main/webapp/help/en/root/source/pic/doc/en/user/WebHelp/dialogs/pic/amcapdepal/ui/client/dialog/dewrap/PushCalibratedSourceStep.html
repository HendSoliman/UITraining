Push Calibrated Source Step

Purpose

This screen is used to push away the source pallet from depalletizer, or send the pallet without pick missions to clarification.

 

      
Prerequisites
  • A source pallet arrives at the check location of current workstation, and There is no active pick mission for this source pallet.
  • There is no pick mission for this pallet.

 

      
Screen

Scenario 1: A source pallet arrives at the check location of current workstation, and There is no active pick mission for this source pallet.

 

Scenario 2:There is no pick mission for this pallet, need to send it to the clarification

 

      
Special Buttons

Button

Description

 

For scenario 1: Release current source pallet. The source pallet will be stored into inventory storage.

For scenario 2: send the pallet withou pick missions to clarification.
