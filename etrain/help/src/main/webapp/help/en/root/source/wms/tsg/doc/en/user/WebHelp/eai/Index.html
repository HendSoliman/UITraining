Introduction

The Enterprise Application Integration (EAI) subsystem of the Software Suite provides the connection between arbitrary host systems and any local subsystems.

During the transformation of messages which are exchanged between host and local subsystem some errors can occur.

This section describes such errors and gives one or more solutions to solve this.

      
Overview

This section is structured as follow:

Chapter

Description

Application starting errors

Describes errors related to starting of the application

Translator error

Describes errors which occur during establishing the EAI background processes.

Message error

All possible errors which can occur during transforming the incoming or outgoing message are described in this chapter.

CleanUp error

All CleanUp related errors are described in this chapter.
