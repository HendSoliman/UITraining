Routing

Purpose

The Routing window allows you to display and manage the routing configuration.

 

      
Functions

Operators can perform the following tasks:

  • Create, modify and delete Connections and Global destinations
  • Assign TU Types

 

      
Overview

This window contains following views:

  • Connections
  • Global Destinations
