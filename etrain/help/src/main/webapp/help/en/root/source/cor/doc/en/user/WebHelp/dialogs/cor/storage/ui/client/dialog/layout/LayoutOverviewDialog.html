Layouts

Purpose

A Layout describes how load units of a specific allocation type are booked onto a Location. Thus, the layout specifies the position of the Load Unit on the Location and the occupation of the Location after booking the Load Unit onto the Location. The allocation type is assigned to a Load Unit Type using the configuration, "Location Type Load Unit Type".

The current occupation of a Location is aggregated in the Location attribute Availability Layout Status. The current occupation of a Location, including the reservations on the Location, is aggregated in the Location attribute Reservation Layout Status. The combination of these status attributes with the Layout provides information about the occupation and blocking situation on the Location.

The Layout defines mainly a state-machine for the Location attributes, Availability Layout Status and Reservation Layout Status. This state-machine is represented by the Layout Status Transitions. With this information, all physical status transitions of the selected layout are defined.
