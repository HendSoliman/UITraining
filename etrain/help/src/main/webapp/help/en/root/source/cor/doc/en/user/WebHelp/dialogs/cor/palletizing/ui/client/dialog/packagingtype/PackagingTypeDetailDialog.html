Description

Packaging Type ID

Unique for each packaging group and packaging detail combination:

  • Carton Solid (solid plane closed carton)
  • Carton Corrugated (corrugated closed carton)
  • Tray (products on a tray)
  • Tray Upside (products which have a tray at the upper side)
  • Tray Poly (tray consists of polystrene)
  • Shrink Wrapped (products which are wrapped with a foil)
  • Shrink Wrapped Plate (products wrapped with foil and tray at bottom)
  • Paper wrapped (products wrapped with paper)
  • Crate (products inside no specific crates)
  • Crate Collapsible (products in collapsible crates)
  • Tote (products inside no specific totes)
  • Tote Tapered (products in tapered totes)
  • Tote Collapsible (products in collapsible totes)
  • Tote Nestable (products in nestable totes)
  • Canister (products inside no specific canisters)
  • Canister Nestable (products in nestable canisters)
  • Bag Paper (products in bag consists of paper)
  • Bag Plastic (products in bag consists of plastic)

Description

A description for the packaging type which describes the packaging group and packaging detail combination textual.

Packaging Group

The packaging group is used for first identification of the packaging:

  • Carton
  • Tray
  • Tote
  • Crate
  • Canister
  • Shrinked Wrapped
  • Bag.

Color

For the packaging type a specific coloring can be defined here which is shown in the pallet widget.

Packaging Detail

The detailed identification of the packaging is done with this parameter:

  • None (no specific grouping)
  • Solid (solid packaging)
  • Corrugated (corrugated packages)
  • Upside (products with inverted packaging, e.g. upside down)
  • Square (square packaging)
  • Collapse (collapsible packaging)
  • Nestable (Nestable packaging)
  • Tapered (packaging which are tapered)
  • Plate (packages with a plate at bottom)
  • Poly (packaging consists of polystyrene)
  • Paper (packaging consists of paper)
  • Plastic (packaging consists of plastic).
