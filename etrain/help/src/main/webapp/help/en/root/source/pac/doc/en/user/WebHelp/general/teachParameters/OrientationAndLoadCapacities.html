Load Capacities

 

Parameter Name:
  • loadCapacityLengthWidth
  • loadCapacityLengthHeight
  • loadCapacityWidthHeight

Display Name:

 

 

Description:

The Load Capacity specifies the maximum load which can be applied to a Packaging Variant in a given orientation.

Unit:

kg/m²

Example:

750

Limitations:

300 ≤ loadCapacity* ≤ 2000

Data Source:

Formula (see below)

Used for:
  • Pallet planning with PackBuilder 

Usage for pallet planning:

For every pack object on a pallet, PackBuilder calculates the load resulting from all other pack objects lying on top.

Only those pallet structures are retained as candidates for which the calculated load is lower or equal to the load capacity for all pack objects

Formula:

The following formula assumes Length/Width orientation.
For other orientations, dimensions need to be permuted accordingly.

See also limitations, obove.

 

loadCapacity        = loadWeight * strengthClassFactor * packagingFactor / (length * width)

 

loadWeight          : Weight of all packages of the same kind on top of the bottom package.

                      The maximum height of the assumed pallet is a parameter depending on the content type.

loadWeight          = (layerCount - 1) * weight

layerCount          = ceiling(Hmax[contentType] / compressedHeight)

strengthClassFactor : A factor depending on the strengthClass specified during teaching (values between 0.25 and 1.5).

packagingFactor     : A factor depending on the packagingType (values between 0.7 and 2)

 

Formula (Excel):

loadCapacityLengthWidth =

MAX(
    MIN(
        ROUND(
              (1000 * ((ROUNDUP(STACKHEIGHTMAX(CONTENT_TYPE)/(HEIGHT - NESTING_HEIGHT);0)-1)*WEIGHT) / (LENGTH * WIDTH))
               * PACKAGING_FACTOR(PACKAGING_TYPE)
               * STRENGTH_FACTOR(STRENGTH_CLASS)
               );
              0);
        MaxLoadCapacity);
    MinLoadCapacity)

 

Rule:

Attention: the rule below is not in accord with the formulae above and must be adapted.

rule "LOAD_CAPACITY_LENGTH_WIDTH"

 when

   pqu : PalletizingQuantityUnitDto(weight        != 0,    length        != 0,    compressedHeight != 0, width != 0,

                                    strengthClass != null, packagingType != null, contentType      != null)

         ptValues : PackagingTypeValue(packagingTypeId == pqu.packagingType.packagingTypeId)

         ctValues : ContentTypeValue  (contentTypeId   == pqu.contentType.contentTypeId)

         scValues : StrengthClassValue(strengthClassId == pqu.strengthClass)

 then

   //System.out.println("LOAD_CAPACITY_LENGTH_WIDTH");

   pqu.setLoadCapacityLengthWidth(

     Math.max(

       Math.min(

         floorToTens(

           ((double) pqu.getWeight() /(double) 1000 / ((double) pqu.getLength() * (double) pqu.getWidth() /

            (double) 1000000)) * ((double) ctValues.getMaxLoadingHeight() * (double) ptValues.getPackagingFactor() /

            (double) pqu.getCompressedHeight() - (double) 1)

          * (double) scValues.getFactor()),

         1500),

   getMinLoadCapacity()));

 end
