Putaway Failure Analysis

Purpose

Several reasons could apply for an unsuccessful Putaway operation. This screen helps users analyze why the process fails for some Load Units.

Columns with "No" show a reason for  the unsuccessful Putaway process.

 INFO 

If the Location accessible via Area Access is free of errors, the entry is marked "green".

 INFO 

Only Locations with a "free" or "partial" availability status display.
     
Screen

      
Fields

Field

Description

Destination Location

Destination Locations for the Load Units depend on a valid Area Access entry for the Load Unit Type of the Load Unit in status, "NO_TO" or "scheduled".

Location Configured

If active, the Location has the configuration status "complete".

Defines if Location configuration was accomplished and the Location can be used by the system.

Location Not Locked

If active, the Location has the lock status "available" or "locked for output".

Long lock status specifies if the Location was locked for specific operations; meant as a long term lock set by an user.

  • available (all planning operations are allowed)
  • locked for input (inbound planning operations will not be assigned to the Location automatically)
  • locked for output (outbound planning operations will not be assigned to the Location automatically)
  • locked for input and output (inbound and outbound planning operations will not be assigned to the Location automatically)

Location OK

If active, the Location has the error code "OK".

Error code identifies the last error which occurred at this Location.

  • OK (no error occurred)
  • bin full (Location is physically full, but logically within the system free)
  • bin empty (Location is physically empty, but logically within the system occupied)
  • multiple additional codes are set by the underlying transport system (MFC) if any error during transport operation

Height Class Fits

If active, the Load Unit's Height Classification is lower than or equal to the Location's Height Classification.

Putaway Allowed

If active, Putaway is allowed for this Location.

No Dedicated Pick Face

If active, no Dedicated Pick Face is assigned to this Location.
A Dedicated Pick Face is a Location used for Picking if it is assigned to the SKU used for this Picking process.

Aisle Configured

If active, the Aisle has the configuration status, "complete".
Defines if Aisle configuration was accomplished and the Aisle can be used by the system.

Aisle Not Locked

If active, the Aisle has the lock status "available" or "locked for output".

Long lock status specifies if the Aisle was locked for specific operations; meant as a long term lock set by an user.

 INFO 

With a lock of this type, all current Inventory reservations within the Aisle are cancelled with an immediate new reservation process.

Values are identical to those of the Technical Lock Status.

Aisle Fill Level

Used capacity of the Aisle, in percent. If active, Aisle Fill Level is lower than 100 percent.

Level Configured

If active, the Level has the configuration status "complete".
Defines if Level configuration was accomplished and the Level can be used by the system.

Level Not Locked

If active, the Level has the lock status "available" or "locked for output".

Long lock status specifies if the Level was locked for specific operations; meant as a long term lock set by an user.

 INFO 

With a lock of this type all current Inventory reservations within this level are cancelled with an immediate new reservation process.

Values are identical to those of the Technical Lock Status

Level Fill Level

Used capacity of the Level, in percent. If active, Level Fill Level is lower than 100 percent.

Weight Limit Not Reached

Maximum overall weight limit of all Load Units stored on this Location. Measuring unit is project-specific. If active, Weight Limit for this Location is not yet reached.
