Personalization

Purpose

A user may configure the UI Frame individually and save settings personalized. On login the personalized settings will be restored.

      
Overview

The Personalization of the UI Frame includes

Topic

Description

Desktop Layout

Save dimensions and layout of the UI Frame including open dialogs

Favorites

 Dialogs marked as Favorites

Data Grid Layout

 Column configuration and Filter Settings of Data Grids
Temporary Layout Includes current Desktop Layout and all Data Grid layouts
     
Favorites

Favorites list below the UI Frame Tree view. Users can add dialogs of the tree view to the favorites list. Favorites display/hide together with the tree view.

A dialog can be added to the favorites list either by the context menu from the Menu tree or by clicking the star icon in the righter dialog window caption.

The favorites list may be configured within the favorites tree view.

Procedure:

Step

Description

1.

Right-click an element in the UI Frame tree view, e.g. "Permissions"

2.

 Click the [Add to favorites] button.The favorite displays in the favorites list, or
1b. Drag the element from the menu tree and drop it to the Favorites Accordion button, or
1c. Click the star icon in the righter dialog window caption
3. Click "Favorites" button in Accordion Menu. Select items to remove or move up/down

 

      
Desktop Layout

The actual desktop layout can be saved and restored. Sizes of the UI Frame and its Views are stored as well as open screens. On restart of the UI Frame, a saved layout is restored; meaning the Frame appears with sizes it was stored and all dialog are automatically reopened. A saved desktop layout can be undone using the Reset option; meaning default values are applied.

Additionally the user may chose to show/hide these Items:

  • Message Area
  • Menu Tree
  • Status Bar
  • Title Bar 

Hiding/showing all of these items at once can be achieved using the Full Screen option. 

To change the Desktop Layout the UI Frame Toolbar menu offers items:

Button

Description

Save

Saves the current desktop layout, its dimensions and open screens.

Restore

Restore a saved desktop layout. Reapplies stored dimensions. Opened screens at save time are reopened.

Reset

Reset desktop layout. Saved layout data is deleted and default values are applied.

 

Procedure:

Step

Description

1.

Select [Desktop Layout] in the Tools Menu.

2.

Select the desired option: Save, Restore, Reset.

 

3.

On 'Reset', the following message appears:

 

Click the [YES] button to confirm.

Click the [NO] button to cancel.

 

      
Data Grid Layout

The layout of each single Data Grid can be personalized. The Data Grid layout includes the column configuration as well as filter and sorting settings. In detail a Data Grid layout has the following aspects.

Dialog

Description

Change Layout

Dialog for changing the layout of the grid
Layout Sets Dialog for saving layout sets
Filter Sets Dialog for saving filter sets

 

      
Temporary Frame Layout

All layout settings including Desktop and Data Grid Layouts will be saved on logout and may be restored on login.
