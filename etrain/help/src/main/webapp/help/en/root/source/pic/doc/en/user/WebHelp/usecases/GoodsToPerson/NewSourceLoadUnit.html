New Source Load Unit

Description

If all required Pick Containers are available at the Put Locations and the next Source Load Unit is available, the Pick operation can be started with a Source Load Unit scan.

      
Triggers and Actors

This must be performed after all required Pick Containers are available and former Source Load Unit sent away. Or the screen just started and Source Load Unit arrives at Source Location of Pickstation.

Following persons need to perform this:

  • Picker at Goods-to-Person Pickstation.
      
Prerequisites
  • All required Pick Containers are put on the correct Put Locations.
  • Source Load Unit available on Pickstations Source Location.
  • Screen just started. Or former Source Load Unit was sent away and a new Source Load Unit is required.
  • If an invalid scan: Unknown Source Load Unit or Source Load Unit without active Picks is scanned.
      
Work Flow

Step

Description

1.

Open the Picking Screen.

2.

Source Put displays.

3.

Scan the available Source Load Unit.
"Enter" key and "OK" button activate with the scan.

Invalid scan

3a.

Invalid Source Load Unit is scanned.

4a.

Source Invalid Scan displays.

5a.

Go to step 3.
     
Result

The Picking Dialog proceeds to the "Picking" state.

      
Required Resources
  • Picking Screen and in more detail the following screens:
  • Source Put
  • Source Invalid Scan.
