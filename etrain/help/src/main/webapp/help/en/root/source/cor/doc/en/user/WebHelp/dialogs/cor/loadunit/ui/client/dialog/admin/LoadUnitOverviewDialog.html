Load Units

Screen

The lower grid tab "Load Unit Hierarchy" displays the hierarchy of stacked Load Units for the Load Unit selected in the upper grid.
The lower grid tab "Load Unit Request" displays the optionally existent Request for the Load Unit selected in the upper grid.

 INFO 

A Load Unit Request is set by specific business processes which currently requests the Load Unit for fulfillment. Only one request at at time could exists (see the Detail Dialog Load Unit for the list of different types).
