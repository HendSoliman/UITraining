Change Quantity

Purpose

This dialog allows operators to change the quantity of a selected Inventory Unit. A comment helps to track this change.

An Inventory Movement Type and/or an Inventory Movement Reason help to define the specifics of the related booking.

 INFO 

The Unit of the displayed dimensions are given by the appropriate Base Quantity Units defined by the Dialog Unit of Measure.
     
Screen

      
Fields

Field

Description

Load Unit

Object, on which inventory can be stored and transported.

Base Load Unit

Object on which the Load Unit is piled up. It is the Load Unit used for transports.

SKU

SKU of the selected Inventory Unit within the Client.

Batch

Batch, in which the inventory was produced.

Client

Owner of the Inventory Unit.
Quantity Total quantity of this Inventory Unit expressed in the Base Quantity Unit of the SKU.

Free Quantity

Quantity available for disposition, provided the inventory is not on hold. Quantity is expressed in the Base Quantity Unit of the SKU.

Base Quantity Unit

Unit of the quantity of the SKU.
Movement Type

An already existing Inventory Movement Type can be selected.
Values are configured in Dialog Inventory Movement Type Administration.

Movement Reason

An already existing Inventory Movement Reason can be selected.
Values are configured in Dialog Inventory Movement Reason Administration. When there is no Inventory Movement Type selected, all available Inventory Movement Reasons display. When an Inventory Movement Type is selected, only the Inventory Movement Reason assigned to this type display.

New Quantity

New desired quantity.

Comment

Comment for the quantity change.
