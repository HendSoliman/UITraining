Bearbeitung von Hilfe-Kapiteln

Zweck

Dies scheint das erste mal zu sein, dass Sie den <Bearbeiten> Knopf verwendet haben.

Hier finden Sie Informationen zur Einrichtung der Infrastruktur für die Bearbeitung von Hilfe-Kapiteln.

Voraussetzungen
  • Auf diesem Computer muss ein HTML Editor installiert sein.
    Empfohlene HTML Editoren sind
  • Microsoft FrontPage
  • Microsoft SharePoint Designer
     
  • Für umfangreiche Bearbeitungen sollte DocHTML installiert sein.
    Geringfügige Änderungen können auch ohne dieses Tool vorgenommen werden.
    DocHTML erleichtert die Bearbeitung von HTML Seiten entsprechend den Standards für Dematic iQ.
    DocHTML kann von hier installiert werden.

  • HTMLEdit muss installiert sein.
    Dieses kleine Hilfsmittel erlaubt es aus einem Browser heraus einen HTML Editor zu starten.
    HTMLEdit kann von hier installiert werden.

 

Die Infrastruktur ist installiert. Diesen Hinweis nicht mehr anzeigen.

 

Hintergrund

Aus Sicherheitsgründen ist es normalerweise nicht möglich ein Programm wie einen HTML Editor aus einem Browser heraus zu öffnen.
Durch die Installation eines zusätzlichen Programms auf Ihrem Rechner ist es jedoch möglich, dies zu erreichen.

Dematic iQ verwendet dazu folgendes Verfahren:

Der Knopf Bearbeiten ist mit folgender URL verknüpft: <htmledit:<Dokument URL>
htmledit ist ein Protokoll ähnlich wie http jedoch spezifisch für Dematic iQ.
Nach aktuellem Kenntnisstand wird dieses Protokoll von keinem anderen Programm verwendet.

Um einen HTML Editor auf dem lokalen Rechner zu starten, muss ein Protokoll-Handler für das htmledit Protokoll registriert werden, der das zu startende Programm festlegt.
Solche Protokoll-Handler können manuell über den Windows Explorer angelegt werden (Extras -> Ordneroptionen -> Dateitypen) mit dem Typ URL: <protokoll> ohne Dateierweiterung.
Da der Web Browser die gesamte URL (einschließlich "htmledit:") an das im Protokoll-Handler festgelegte Programm übergibt, kann der Editor nicht direkt mit dieser URL gestartet werden. Stattdessen muss zunächst die URL des Dokuments über ein kleines Zusatzprogramm extrahiert werden. Dazu werden String-Funktionen des Windows Scripting Host verwendet.

Die gesamte Infrastruktur kann mit Hilfe des HTMLEdit Installers installiert werden.

Diese Lösung hat keine Sicherheitsprobleme, da der Anwender selbst festlegt, wie das spezielle Protokoll verwendet wird.
Es wird kein ActiveX verwendet.
