Manual Depalletizing

Purpose

All products or pallets which cannot be automatically de-palletized must be de-palletized manually.

For example. this applies to

     Mixed pallets.

     Products with a shape which is not suitable  for automatic handling.

     Situation where the de-pal quantity is not a multiple of a layer.

In addition, the manual De-Pal stations can, in principle, also de-palletize single-SKU pallets in case that a robot is out of service or in order to cover load peaks.

 INFO 

Depalletizer station status

Status Description
auto for automatic de-palletizer. All other status values are only for manual de-palletizer
blocked current manual depal workstation is not available or the terminal is inactive or the source location is locked
idle the workstation is active, but no source pallet used for manual depalletizing is on the way to current source location
staging next pallet is on the way to the source location
preparation the pallet is at the source location and waiting for de-wrapping
pending all preparations are done and the pallet is ready for de-palletizing
active the pallet is to de-palletized as next
special handling for one or more products on the pallet teaching is necessary

 

 

Workstation button color scheme

disable for status blocked
grey / enabled for status idle
red for status staging, preparation and special handling
yellow for status pending
green for status active
