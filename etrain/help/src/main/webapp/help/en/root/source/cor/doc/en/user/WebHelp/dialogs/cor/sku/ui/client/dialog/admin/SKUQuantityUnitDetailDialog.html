Field

Description

Unit

Identifier of the Quantity Unit. Usually defined by a host and refers to Quantity Unit definition inside the host.

Quantity Unit

Quantity Units

Factor To Base

Quantity conversion factor of Inventory of this SKU in this packaging level to the Base Quantity Unit of the SKU.

Base Quantity Unit

If this Quantity Unit specifies the Base Quantity for this SKU. Read only field. Can be entered in the Detail Dialog SKU.

Unit of Measure

If this Quantity Unit is related to a Unit of Measure.

Default Pick

The Picking subsystem displays this Quantity Unit as the default during picking operations.

 INFO 

This only applies if exactly one Quantity Unit is configured as "Default Pick". Otherwise always the Base Quantity Unit is used for picking operations.

Pick

The Picking subsystem can use this to decide which Quantity Unit to display during Picking operations depending on various conditions.

 INFO 

The standard only use the Default Pick Quantity Unit. Rules for the usage of different Pick Quantity Units must be provided on a project specific base.

Quantity Units of this type are used for cubing.

Supplier

If this Quantity Unit is configured as the supplier Quantity Unit of the SKU.

Order

If this Quantity Unit is configured as the (Customer) Order Quantity Unit of the SKU.

Sales

If this Quantity Unit is configured as the sales Quantity Unit of the Stock Keeping Unit. Configuration may be overruled by a Host Plant configuration of a SKU, if a corresponding configuration exists.

Handling

Preferred Load Unit Type

Load Unit Type preferably used to store Inventory of this SKU in this packaging level.

Default Pack Breakdown

Type of packages used by default inside of packages of this type and this SKU.

Preferred Fulfillment Zone

Zone where requests for Inventory of this SKU and of the packaging type specified by this SKU Quantity Unit are preferably fulfilled.

Preferred Fulfillment Handling Type

How requests for Inventory of this SKU and of the packaging type specified by this SKU quantity unit are preferably fulfilled:

  • full load unit (Load Unit is shipped in the form as it is stored in the warehouse and no Picking is involved)
  • full case picking (Full cases (i.e. goods in cartons) instead of single items are picked onto the Dispatch Load Unit)
  • split case picking (Picking of individual items is required to cover the quantity required for an Order)

Labeled

If the packaging level defined by the SKU Quantity Unit is labeled with a barcode that can identify the SKU of the package.

Product

Product Code

Code that identifies this SKU in this packaging or Quantity Unit. For example, the European Article Number.

Product Code Type

Type of product code used in the attribute Product Code. For example, the European Article Number Type.
