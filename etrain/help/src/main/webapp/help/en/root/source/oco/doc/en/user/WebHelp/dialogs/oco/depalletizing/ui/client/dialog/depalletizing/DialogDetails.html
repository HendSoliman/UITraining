De-palletizing Details

Source Pallet

Field

Description

ID

Source Load Unit ID that was scanned automatically or manually by an operator.

Load Unit Type

Type ID of the Source Load Unit.

SKU

SKU ID on the Source Load Unit.

Client

Name of the Client of the SKU on the Source Load Unit.

Description

Description of the SKU on the Source Load Unit.

Batch

Batch of the Inventory Unit on the Source Load Unit.

Best Before Date

Best before date of the Inventory Unit on the Source Load Unit.

Quantity

Current remaining quantity of the Inventory Unit on the Source Load Unit. Expressed in the displayed packaging level.

Packaging Level

The packaging level of the Inventory Unit on the Source Load Unit related to the displayed quantity.
     
To De-palletize

Operation Mode: to container

Field

Description

Quantity Current Put

Quantity to be packed into the current destination container. Expressed in the displayed packaging level.

Packaging Level

Same packaging level as displayed in the corresponding field of Source Pallet.

Additional Puts

Number of additionally required destination containers.

 INFO 

Calculated regarding the current quantity and packaging level.

Expected Weight

Weight in kg expected on the destination container after the packing operation.

Operation Mode: to conveyor

Field

Description

Remaining Quantity

Total quantity less the done quantity. If less than zero, “too many units de-palletized!” displays.

Already Done

Already de-palletized quantity, based on received messages from an installed count controller.

 INFO 

Prerequisite: automatic counter installed and connected to the screen (from the conveyor PLC).

Total

Total quantity to be de-palletized from the source pallet. Expressed in the displayed packaging level.

Packaging Level

Same packaging level as displayed in the corresponding field of Source Pallet.

Label Panel

“Attach Labels!” displays if a label printed which must be attached by the operator before putting the product onto the conveyor or belt.
     
Destination

Only available in the operation mode, "to container".

Field

Description

Destination Container

Destination Container ID that was scanned automatically or manually by an operator.

Load Unit Type

Load Unit Type ID of the Destination Container.

Weight

Measured weight of the destination container (in kg). Based on received messages from an installed scale.

 INFO 

Prerequisite: automatic scale installed and connected to the screen.
     
Source LU Queue

The Source Load Unit Queue shows all Load Units currently ordered to this De-palletizing Workstation.
