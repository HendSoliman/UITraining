Check LU-Type

Purpose

"Check LU-Type" step appears after a Load Unit was scanned. The Load Unit type can be initially set or changed.

 INFO 

The Load Unit type is checked and changed only if necessary.
     
Screen

      
Prerequisites
  • Advice was selected
  • Load Unit was scanned.
      
Fields

Field

Description

Client

ID of the previously selected Client.

Dock

ID of the previously selected Dock.

Advice

ID of the previously selected Advice.

Supplier

 ID of the previously selected Supplier. (optional)

LU-Type

 Load Unit type of the selected Load Unit. If a new Load Unit was scanned, is empty and a type must be selected.

Advices LU-Type

 If the selected Load Unit was advised, the related advised Load Unit type displays.
     
Special Buttons

Button

Description

F2: LU Type

Change the Load Unit Type by selecting one of the following:

Scroll through list of available Load Unit Types with cursor keys (up and down). Click "Close". Selected Load Unit Type will be determined in the appropriate field.

OK

Confirm the selected Load Unit Type and switch to "Identify Product" if more than one appropriate Advice Line is available or to "Check Product" if exactly one Advice Line could be determined uniquely.

Cancel

Cancel and switch back to "Scan Load Unit".
