Confirm Pick Walk

Description

With finalization of a single Pick Walk, each Pick is confirmed with its picked quantity, used Pick Container and measured weight, as well as Confirmation Code and Reason if an exceptional result.
It is also required to split a Pick into another Pick Container due to completed container weight or volume.

      
Triggers and Actors

If one or multiple Pick Walks are completed in the process of Offline Picking, Pick Walk must be confirmed separately against the system by a Picker.

      
Prerequisites
  • At least one Pick Walk in Pick Zone of type "Person-to-Goods" in status "active" is available which could be done with the use case Create Pick Walks.
      
Required Resources
  • Pick Walk
  • Pick Walk Confirmation.
      
Work Flow

Standard case

Step

Description

1.

Open the Pick Walk screen.

2.

Select one valid Pick Walk in "active" status from the Pick Walk grid.

3.

Push Confirm Pick Walk

4.

The Pick Walk Confirmation screen opens.

5.

For each Pick Mission of the Pick Walk

5a

 Set a Pick Container.

 INFO 

The screen asks for each different Pick Mission of the selected Picks for a new ID of the Pick Container. There is a validation to not use the same ID multiple times during one Confirmation.
5b Check all shown Picks and confirm with OK.

6.

 In the Overview screen you get an overview of all Pick Missions of this walk for another check.
Additionally the Picker Id needs to be entered.

7

Confirm and save all changes.

Short case

Step

Description
1-5a See above
5b Check all shown Picks. For the ones were not enough inventory was found:
Mark the Row and push the Short button
5c In the Short Step the quantity and the Reason for the specific pick need to be entered. After confirmation the screen returns to the Pick Mission Step.
6-7 see above

 

Split Pick

Step

Description
1-5a see above

5b

Check all shown Picks. For the one were a split happened:
Mark the Row and push the Full button

5c

Enter values for the required data for the new Pick Mission:

  • The new Pick Container
  • The picked Quantity (Current)
5d After confirmation the second step of Full handling opens.
Single Picks can be moved between the Pick Containers now in case the order of the picking was not done in the expected order.
5e Confirm with OK. The screen returns to the Pick Mission Step.
6-7 see above

 

      
Result

All Picks of the Pick Walk are confirmed appropriate to all entered and changed values and the related Inventory Units are booked accordingly.
The Picks and the Pick Walk proceed to "finished" status and the Pick Container is booked to the given destination location.
