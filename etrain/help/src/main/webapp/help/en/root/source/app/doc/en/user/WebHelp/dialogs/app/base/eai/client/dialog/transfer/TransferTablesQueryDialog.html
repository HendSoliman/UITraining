Transfer Table Overview

Purpose

In order to provide an overview over all host systems and all message types an overview dialog exists in Transfer component. This dialog shows the quantity of messages grouped by their state.

      
Main Functions

The user can select a host system in master grid for have a look at all message types contained in it. After selecting a message type in detail grid, the user can press the "View/Edit Message" button, which opens another dialog containing messages for the selected host.
