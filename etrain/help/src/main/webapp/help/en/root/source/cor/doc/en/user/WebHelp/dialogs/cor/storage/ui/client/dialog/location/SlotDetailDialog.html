Slot - Detail

Slotting Definitions

  • A Slot is an area within a bay where Load Units can be stored one behind each other (Z-direction) but not side by side (X-direction)
  • In X-direction, a fixed raster is used which can be defined depending on project conditions.
  • In Z-direction, space is divided into one or multiple partitions with equal spacing (DZ):
    • single-deep (sd): 1 partition
    • double-deep (dd): 2 partitions
    • four-deep (4d): 4 partitions.
  • Z-partitioning is the same for a whole MS area. This applies to the number of partitions as well as to spacing in Z-direction (DZ).
  • Slots are always filled from back to front
  • A Load Unit may occupy multiple Z-partitions. The number of required partitions is determined from
    • the depth of the load unit (DLU)
    • additional space required for finger engagement
    • the depth of a Z-partition (DZ)
  • The first Load Unit allocated to the deep position determines the width of a Slot. The Slot width (WSlot) is determined from
    • the width of the Load Unit
    • additional space required for the telescope on both sides.
  • The Slot for this first Load Unit is created by a split from the existing empty Slot.
  • Each Slot that becomes empty will be merged with the empty neighbor or simply enlarged to span the whole empty space.
  • Restricted Mixed Size Slotting: Load units stored in an already occupied slot must not have a larger width than the Slot width minus the space required for the telescopes. The Load Unit at the deepest position determines the slot width and all other Load Units stored in front must adhere to this width. Furthermore, the variation of X-extensions are limited to an extent which allows to extend the telescopes to the entire shelf depth, if necessary.
    • The maximum width of the storable Load Unit also depends on the configuration of the Control Data property cor-putaway:PutawayAreaConfigurations.OccupiedSlotWidthTolerance.
  • Extended Mixed Size Slotting is a variant where it is allowed to store large items in front of smaller items if the adjacent slot is not occupied. This extends the Restricted Mixed Size Slotting and is enabled by default.
      
Positioning Parameters

The figure demonstrates the different parameters required for management of Load Unit positions within a bay for MS-Flex.

  • An offset of Wgap is used to assure the required gap for the shuttle telescope. At the first left position of a bay no Wgap is used, the bay always starts at 0. The resulting offset (because of the initial Wgap=0) is fully handled by the PLC.
  • Load Units always will be aligned at the left side of a Slot.
  • The Slot width (WSlot) is determined from the width of the Load Unit in the back position (WLU) by Wslot = WLU + Wgap. The Slot width in Raster Units (RU) is calculated according to RUWSlot = Ceil(WSlot / Raster Unit)
  • The number of depth positions (NZ) required for a Load Unit with a given depth (DLU) needs to consider the gap required for shuttle finger engagement (DZ) . It is calculated from NZ = Ceil{(DLU + Dgap) / DZ}
  • The X-positions for every Load Unit are communicated to the MS-Flex PLC through parameter P in Raster Units. The example above assumes that the Multishuttle does not add an offset to the given P-Parameter.

Fields

Field

Description

Slot ID

Unique ID of the Slot within the Location.

Status

Extent to which the Slot is available for the storage of additional Load Units. Accounts for reservations for Putaway and occupied positions.

  • free (No Load Unit is currently stored within the Slot)
  • partial (The Slot is occupied with at least 1 Load Unit but is not yet full)
  • full (The Slot is completely filled with Load Units)
  • blocked (The Slot is blocked and cannot be used for storage allocation. This status is usually used for Slots covering gaps with physical limitations (e.g. cooling or sprinkler))
  • unused (The slot is not in use. It has been generated in advance for performance reasons).

Left Neighbor

Identifies the Slot adjacent to the left when looking in z-direction.

 INFO 

Required for Extended Mixed Size Slotting only.

Right Neighbor

Identifies the Slot adjacent to the right when looking in z-direction.

 INFO 

Required for Extended Mixed Size Slotting only.

Temporarily Blocked For Putaway

If this Slot is currently blocked for put away operations because in a multi-deep Location a Load Unit at a rear position is reserved for retrieval. Set in order to avoid shuffle moves.

X Start

X-coordinate of the left boundary of the slot, specified as integer in raster units of the Area where the left boundary of the Location (bay) is at 0.

X End

X-coordinate of the right boundary of the slot, specified as integer in raster units of the Area.

Width

Width of the slot, specified as integer in raster units of the Area.

Virtual Width

The width to which the Slot could be extended if the neighboring Slot would be used. Only a right neighbor is considered.

 INFO 

Required for Extended Mixed Size Slotting only.

Shuttle Width

Shuttle width for the next storage/retrieval operation using this slot (in mm). When using this width for extension of the flex axis, there will be no restriction for extension of the telescopes.

 INFO 

Required for Extended Mixed Size Slotting only.

Reservation Capacity

Number of positions that are not yet occupied or reserved by Load Units.

Available Capacity

Number of positions that are currently not occupied by Load Units.

Discriminating Key

Discriminating key from the Load Unit in the front position.
