Located in demo\target\configuration\cor-putaway.xml.

PutawayAreaConfigurations.

AreaInternalRelocationRestriction

AreaInternal
Relocation
Restriction

area

NONE

This parameter defines general restrictions on possible Destination Aisles for Aisle internal relocations. Possible values are:

  • none (There are no general restrictions on possible Destination Aisles for Aisle internal relocations.)
  • to larger aisle ids (Area internal relocations are only possible within the same Aisle or to an Aisle with an id that is alpha-numerically larger than the current Aisle. This option might be useful in case there is no conveyor loop for the Area.)
  • to smaller aisle ids (Area internal relocations are only possible within the same Aisle or to an Aisle with an id that is alpha-numerically smaller than the current Aisle. This option might be useful in case there is no conveyor loop for the Area.)

PutawayAreaConfigurations.
MaximumFillLevel		 float area 0.98 This parameter specifies per Area the maximum fill Level of aisles/levels. In the default Strategies for classic Areas all aisles, but the aisle of the Load Unit´s current Location, having a fill level higher than or equal to the Parameter will be ignored.
In the default Strategies for Multi Shuttles all levels, but the level of the Load Unit´s current Location, having a fill level higher than or equal to the Parameter will be ignored.
PutawayAreaConfigurations.
OccupiedSlotWidthTolerance		 int area 2 If the difference between the width of a partially occupied slot and the required width exceeds this parameter, the slot has not to be preferred by the put away strategy. To be configured in raster units.
Only relevant for Multishuttle Flex warehouses.

StrategySwitch.
ThresholdPreferEmptyBins

 int

area

  

The put-away strategy is automatically configured to prefer empty bins in multi deep areas, when the number of empty bins in an aisle is greater than this parameter for one aisle in the area. The strategy switch is done with the demon "SwitchDoubleDeepPutawayStrategyDemon".

StrategySwitch.
ThresholdPreferPartialBins

int

area

  

The put-away strategy is automatically configured to prefer partial bins in multi deep areas, when the number of empty bins in an aisle is smaller than this parameter for one aisle in the area. The strategy switch is done with the demon "SwitchDoubleDeepPutawayStrategyDemon".

StrategySwitch.

IsPreferPartialBins

boolean

area

false

When this parameter is true, partially occupied bins are preferred by bin selection. When this parameter is false, empty bins are preferred. This parameter is switched by the demon SwitchDoubleDeepPutawayStrategyDemon
