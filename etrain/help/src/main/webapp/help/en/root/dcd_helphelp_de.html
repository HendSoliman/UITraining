Verwendung der Dematic iQ Web Hilfe

Zweck

Dieses Kapitel beschreibt die Benutzung der Dematic iQ Web Hilfe.
Siehe auch: Index und Suche in der Dematic iQ Web-Hilfe

Schaltflächen

Verbirgt oder zeigt den Navigationsbereich.

Zur Speicherung der Einstellungen müssen Cookies für den Browser erlaubt sein.

Öffnet das aktuelle Kapitel in einem HTML Editor zur Bearbeitung
Erfordert die Installation eines HTMLEdit Protokol-Handlers.
Zusätzlich muss sich das verwendete Hilfe-System in einem Verzeichnis befinden und darf nicht über einen Web Server gestartet worden sein.

Aktualisiert die Anzeige des Dokuments im Inhalts-Bereich.
Diese Funktion muss nach einer Bearbeitung des Dokuments verwendet werden um die Änderungen anzuzeigen.

Druckt das aktuell angezeigte Kapitel. Es wird ein Dialog geöffnet, über den der Drucker ausgewählt werden kann.

Öffnet die Startseite des Hilfesystems.

Zeigt das vorige Kapitel in der Gliederung des Inhaltsverzeichnisses.

Zeigt das nächste Kapitel in der Gliederung des Inhaltsverzeichnisses.

Öffnet das Übersichtskapitel, das der aktuellen Seite zugeordnet ist.

Öffnet das übergeordnete Kapitel, falls ein solches für die aktuelle Seite festgelegt ist.

Geht zurück zum vorher angezeigten Kapitel.

Geht vor zum nächsten bereits angezeigten Kapitel.

Kopiert das Hyperlink-Ziel des aktuellen Kapitels in die Zwischenablage. Damit können dann in anderen Dokumenten Links zu dieser Seite erzeugt werden.

Öffnet diese Hilfe

Schließt das aktive Hilfe-Fenster
Logos

Die Logos in den oberen rechten und linken Ecken führen zu projektspezifisch anpassbaren Seiten, z.B. den Homepages des Kunden und Dematic.

Navigation
  • Verwenden Sie die Registerkarte Inhalt zur Navigation durch das Inhaltsverzeichnis
  • Verwenden Sie die Registerkarte Index zur Suche nach bestimmten Schlagworten.
  • Verwenden Sie die Registerkarte Suchen zur Suche nach einem angegebenen Text innerhalb des gesamten Hilfesystems.
Navigation in Inhaltsseiten
  • Verwenden Sie die Links im Seitenkopf zur direkten Navigation zu den einzelnen Blöcken einer Seite.
  • Verwenden Sie die Pfeil-Symbole in den Block-Navigationszeilen zur Navigation zum Seitenanfang, zum vorigen oder nächsten Block oder zum Seitenende.
