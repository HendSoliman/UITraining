Status

Purpose

The Status dialog is used to display the status of MFC devices and components such Means of Transport, Connection, Transport Point. In Addition to that, it is possible to lock and release means of transports, connections and transport points.

 

      
Functions

The Status dialog is used to perform the following tasks:

  • View Status of MFC devices and components
  • Lock/Release MFC devices and components

 

      
Overview

MFC devices and components are presented in individual tabs.

Step

Action

Means of Transports

Displays the status of a means of transport.
Connections Displays the status of connections between transport points.
Transport Points Displays the status of transport points.
Group / Aisle Displays the status of an aisle.
Conveyor Elements Displays the status of conveyor elements
Multishuttle Level Displays the status of multishuttle levels.
