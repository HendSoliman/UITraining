Teach-In Step: Cover Restriction

Purpose

In this step it is specified how items of the given Packaging Variant can be covered by other items when being placed on a pallet.

Cover Restriction

The following table is a guide for selection of the correct cover restriction type:

Selection

Characteristics

Example

None

No restrictions need to be considered when placing other items on top of the teach item.

The item can carry others, irrespective how large they are and where they are placed.

 

Full Length

The teach item must be fully covered in length direction.

The borders on the short side are stable to carry other items.

Full Width

The teach item must be fully covered in the width direction.

The borders on the long side are stable to carry other items.

Full Length or Width

 

 

Full

The teach item must be fully covered.
