Repacking

Purpose

This Repacking screen supports spontaneous Repacking of Inventory Units onto other Load Units or inserts. After a Source Load Unit scan, the appropriate data displays. With a Destination Load Unit scan, the destination of subsequent booking is defined. If one of these two Load Units contained Inserts or further stacked Load Units, it is possible to deal with a single Insert/Load Unit. With Repacking, the desired quantity of selected Inventory Unit of Source is booked onto the selected Destination.

      
Screen

      
Fields

Field

Description

Source

Source Load Unit

Scan field for the Source Load Unit. Left section displays Source Load Unit data.

Inventory

All Inventory Units located directly on the scanned Source Load Unit. For selection, Client and SKU ID display. Selection change is only possible if multiple Inventory Units exist. If Inserts or stacked Load Units are located on scanned Source Load Unit, only Inventory Units located on selected Insert or selected stacked Load Unit are selectable. Selection can be changed in Load Unit View of Source.

Inventory Unit

Unique Identification of the Inventory Unit

Client

Client of the SKU of the Inventory Unit.

SKU

Stock Keeping Unit ID of the Inventory Unit.

Batch

Batch of the Inventory Unit.

Best Before Date

Best Before Date of the Inventory Unit.

Quantity

Quantity of the Inventory Unit in Base Quantity Units.

Unit

Base Quantity Unit of displayed quantity. Quantity to be repacked can be in Base or Pick Quantity Units.

Load Unit View

Graphical view of Inserts or stacked Load Units on Source Load Unit only displays if the related Load Unit Type is compartmentalized, i.e. configured with a layout for geometrical Stacking (LU type with grid sizes greater than 0). Load Units are depicted as follows:

- white: not yet stacked LU on raster

- red: LU is selected

- green: free and not selected

- yellow: occupied and not selected.

Target (Destination)

Target Load Unit

Scan field for the Destination Load Unit. Middle section displays Destination Load Unit data.

Load Unit Type

Enter/Select from drop down list the Load Unit Type of current Destination Load Unit.

 INFO 

Unpacking of a Dispatch Load Unit requests the creation of a new Dispatch Load Unit. The type will be defined by the type of the new Load Unit. In case of multiple DLU Types exists for the used Load Unit Type the system applies any without providing a specific selection logic. This is done for example if a Pick Container that is represented as a DLU as well is manually repacked in a new Load Unit.

Load Unit View

Graphical view of Inserts or stacked Load Units on Destination Load Unit is only displays if the related Load Unit Type is compartmentalized, i.e. configured with a layout for geometrical Stacking (LU type with grid sizes greater than 0). Load Units are depicted as follows:

- white: not yet stacked LU on raster

- red: LU is selected

- green: free and not selected

- yellow: occupied and not selected.
     
Special Buttons

Button

Description

Source

F2: Finish

Remove Source Load Unit from the display. A new Source Load Unit can be scanned for the next Unpack action.

 INFO 

The Load Unit gets an inbound transport automatically if required depending on configuration topic "itl-unpacking:UnpackingWorkstation.automatedPutaway".

Target

F3: New

Create a new Destination Load Unit of selected Load Unit Type. Only activated if Load Unit Type is selected from list.

The Load Unit is created on Workstation's Location immediately.

Target

F4: Print

Print a Load Unit Label for the Destination Load Unit.

Target

F5: Finish

Remove Destination Load Unit from the display. A new Destination Load Unit can be scanned for the next Unpack action.

 INFO 

The Load Unit gets an inbound transport automatically if required depending on configuration topic "itl-unpacking:UnpackingWorkstation.automatedPutaway".

Quantity based

Quantity

Unit

Enter the quantity of displayed Quantity Units to be repacked.

The quantity to be repacked can be selected in Base or Pick Quantity Units.

Quantity based

F6: Unpack

Book entered quantity for Repacking from the Inventory Unit of the Source Load Unit onto the Destination Load Unit or Insert.

 INFO 

Only activated if the entered quantity is less or equal the available quantity of selected Inventory Unit.
