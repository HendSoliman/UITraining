Create by Selection Detail

Purpose

This dialog is used to create a Permission for a dialog specific component (e.g. Button). The specific component has to be selected. This is done by the following steps:

  • In dialog Permission process Button "Create By Selection"

  • Select a specific component by holding the <Ctrl>-Key and clicking on it (e.g. a Button).

  • This detail dialog opens for creating a permission on the specific component
