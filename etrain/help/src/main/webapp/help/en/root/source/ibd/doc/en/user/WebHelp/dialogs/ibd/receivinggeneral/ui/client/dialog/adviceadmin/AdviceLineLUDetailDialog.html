Description

Advice Line

Advice Line associated to this Advice Line Load Unit.

Overview

Status

How far the Advice Line LU was processed:

  • pending (Advice Line Load Unit is completely prepared for processing and available for release)
  • receiving complete (receiving process of the Advice Line Load Unit finished, i.e. inventory creation was completed)
  • complete (Advice Line Load Unit is completely processed and the related Load Unit arrived at the final Storage Locations)
  • to confirm (Advice Line Load Unit is ready for confirmation message to the host system)
  • finished (All required tasks for the Advice Line Load Unit are finished. If required, the advice was confirmed to the host).

Arrival Date

Set by receiving management when the first inventory is created for this Advice Line LU.
Advised

Automatically set to enabled if Load Unit is created in an advice from the host system or in the Advices screen.
Disabled, if Load Unit is created while receiving goods.

Quantity
Quantity Target (LU)

Total inventory quantity announced by this Advice Line Load Unit. Expressed related to the 'Receiving Quantity Unit' of the Advice Line.
Quantity Current

Quantity of the so far created inventory for this Advice Line Load Unit. Expressed related to the 'receiving quantity unit' of the Advice Line.
Receiving Quantity Unit

Receiving quantity unit from the Advice Line.

Supply

Variation

Logistic Error Code

Errors related to logistic aspects of a Load Unit. Set during appropriate business processes and supports clarification at a Clarification Station.

  • wrong inventory (any problem with inventory on the Load Unit and must be clarified (at Clarification Station))
  • wrong load unit (Load Unit was not the expected one and must be clarified (at Clarification Station))
  • receiving clarification (Load Unit was sent to a Clarification Station by any receiving process)
  • picking clarification (Load Unit was sent to a Clarification Station by any picking process)
  • putaway clarification (Load Unit was sent to a Clarification Station by any putaway process)
  • load unit not empty (Load Unit not empty but expected to be)
  • sortation clarification (Load Unit was sent to a Clarification Station by any sortation process)
  • qa clarification (Load Unit was sent to a Clarification Station by any quality assurance process).
Logistics Error Reason

Reason of the logistic error.

Error Code

Transport error which occurred with last transport of the Load Unit.
Error Reason

Reason for last transport error of the Load Unit.

Handling
Receiving Workstation

Workstation used to create the advised inventory.
Receiving Location

Location where the advised Load Unit was created.

Packaging
Packaging SKU

Packing used. SKU for this packaging must exist.
Packaging Quantity

Quantity of the packaging SKU used for the inventory advised with this Advice Line LU. A packaging SKU with an undefined packaging quantity is possible. When submitted by the host, the packaging quantity equals the packaging quantity host. A user may overwrite this quantity.
Host Packaging Quantity

Host defined quantity of the packaging SKU used for the inventory advised with this Advice Line LU. Can be changed using the 'SKU Packaging' field.
Total Weight

For weight based checks of the delivery Load Unit.
Tare Weight

For weight based checks of the delivery Load Unit.

Load Unit
Load Unit Code

Load Unit ID that contains the advised inventory.
Load Unit Type

Load Unit type of the announced Load Unit.
Code Type

A Load Unit can be identified globally using the Load Unit code and Load Unit code type.

 INFO 

A typical Load Unit type is the NVE (German: Nummer der Versandeinheit - number of the delivery Load Unit) or the SSCC (Serial Shipping Container Code).
Load Unit Position

Position onto the Parent Load Unit. Defines the geometrical alignment of this Load Unit (insert/partition) on the Parent Load Unit. Set during geometrical stacking operation.
Full

Set if the Load Unit represents a Full Load Unit which will be stored with this flag subsequently.
Alternate Load Unit Code

Can be independently set. Defines a possible variant external ID of the Load Unit.

Parent Alternate Load Unit Code

Base Load Unit
Base Load Unit Code

ID of the lowest Load Unit of the stack, which contains the advised inventory and is the Load Unit for transport issues.
