Transfer Tables

Purpose

Each "Transfer Tables Dialog" shows all messages which are send and/or received to/from one host system.

      
Main Functions

Following tasks can be performed within the "Transfer Tables Dialog":

  • Change message(s) state to new (N)
  • Delete message(s)
  • Search messages
  • Copy message(s)
  • Edit an existing message

Specific Buttons

Button

Description

Reset Message

Set the state of the message(s) to new (N).

Delete Message

Deletes the message(s).

Copy Message(s)

A popup dialog appears. After selecting a table within the popup dialog, the selected message(s) are copied to another table.

Detail Dialog

Shows the detail dialog for the selected message

Search

At top of each transfer table dialog a text field for search purpose is available.

Put any text in this field and press return. Such messages which contain this search string are shown in this dialog as result.
