Picking Overview

Purpose

This screen displays information on main picking concerns.

      
Screen

      
Fields

Provides information based on open Picks, Pick Jobs and Pick Orders.

KPI

Description

Outstanding Picks

Number of Picks in status "pending", "active" or "released" per Picking Zone.

Late Pick Orders

Number of not yet completed Pick Orders having an already outdated latest end time.

Missing Stock

Number of Pick Order Lines in status "released" or "active" having a target quantity that exceeds the available quantity of the Source Location.

Missing Pick Container

Number of Pick Container having an outdated staging time and have not yet reached their destination.
     
Special Buttons

Provides information based on the outstanding workload.

KPI

Description

Pick Container

Number of open Pick Missions (Pick Container) in status "created", "pending", "released" or "active".

Picks

Number of open Picks in status "created", "pending", "released" or "active".
     
Exceptional Work

Provides information due to exceptional operations and situations.

KPI

Description

Exceptional Picks

Number of completed Picks with a Function Code different from "OK" grouped by Function Code.
     
Completed Work

Provides information based on the yet completed Picks and Pick Container.

KPI

Description

Picks

Number of completed Picks per hour for the last day.

Container

Number of completed Pick Missions (Pick Container) in status "finished" per hour for the last day.
