Detail Dialog Layout Status Transition

Purpose

This screen opens when creating, viewing, or modifying Layout Status Transitions.

 INFO 

For configuration purposes refer to the related topic in the Warehouse Administration Guide.
     
Screen

      
Fields

Field

Description

Allocation Type

Defines the geometrical allocation of the abstract Load Unit Type within the layout. Pre-configured values are:

single, double, single and a half, two in x-direction, two in y-direction, three in x-direction, 4 raster elements.

Capacity

Overall capacity for the given allocation type at the current state.

Position

Storage position of a Load Unit inside the Location on which it is stored. Actual occurrence of the position depends on the raster.

Depth

Depth of the related storage position inside the Location.

Raster Number

How an MFC currently occupies the Location with Load Units. 0 (zero) means no raster.

Status

Identification of the layout status. Combined with "next status", it defines the identification of the status transition.

Blocked

If the raster position is blocked.

Occupied

If the raster position is occupied for the given allocation type.

Next Status

Next state the Location will reach when a transition occurs.
