Clarify Multiple Position Bins

Purpose

Define the occupancy of a bin and create the appropriate UFO-Transports to the chosen Clarification Station.

      
Screen

      
Fields

Field

Description

ID

ID of the selected Source Location in the Dialog Clarification Bin

Short Name

Short Name of the Source Location

Error Code

Error Code of the Location (possible values are listed in Detail Dialog Location).

Error Reason

Error Reason according to the Error Code.
Destination Destination Location the UFO Load Unit is transported to

Load Unit Type

Load Unit Type of the created UFO Load Unit

Set Occupied Positions

Load Unit Type

Load Unit Type of the created UFO Load Unit

Source Position

Source Location Position that the Load Unit is booked on
Occupied Positions

List of all already occupied positions with the corresponding Load Unit Type.

 INFO 

It is important to provide a complete list with the occupancy of this BIN to avoid possible damage when clearing this BIN!
