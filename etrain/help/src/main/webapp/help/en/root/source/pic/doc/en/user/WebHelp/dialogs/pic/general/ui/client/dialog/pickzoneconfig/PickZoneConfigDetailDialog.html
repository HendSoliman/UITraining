Fields

Field

Description

Zone

Unique ID of the Pick Zone.

Overview

Pick Zone Type

Basic handling of picking processes and thus the related Pick Controller type:

  • Goods-to-Person (Zones of this type are related to principles described in Overview Goods-to-Person Picking)
  • Person-to-Goods (Zones of this type are related to principles described in Overview Route Based Picking).

Default Settings

Default Location

New Pick Container are created on this location.

Default Destination

Default Destination Location for the final transport of a Pick Container out of the Pick Zone. The Pick Controller confirms the Pick Container after transportation to this Destination Location. If not set, Pick Containers' destination is defined by the Destination Location of the Pick Order Line.

Default Load Unit Type

Default Load Unit Type for the Zone during creation of new Pick Container.

Default Dispatch Load Unit Type

Default Dispatch Load Unit Type for the Zone during creation of new Pick Container directly as Dispatch Load Unit.

Handling

Blocking Management

If disabled, the currently blocked Pick Missions will be released as well. All required Load Units are requested without consideration of the concurrent request of other Pick Missions (see "Blocked" field in Pick Mission).

DLU Picking Allowed

Whether picking into Dispatch Load Units is desired or not is defined globally in the Order Fulfillment and specifically defined per Order on behalf of the Service Qualifier. This setting allow for overruling and may forbid the use of Dispatch Load Units s in the whole Pick Zone.

Inventory Scan Required

If the Picker must always scan the Inventory Unit.

Scan of Put Location Required

If a Put Location scan is required when push away or put of a new Pick Container on the Put Location.

Zero Cross Check

If set, a (nearly) zero cross check must be done by the Picker if the Order Fulfillment requests it for the Inventory Unit.

Possible values are:

  • disabled The Zero Cross Check is disabled.

  • recount The user has to count a second time if there are any quantity differences.

  • push away After counting if there are any differences the load unit is conveyed to a clearing station.

Limit Reverse Picking

Minimum percentage of inventory to be picked from one Load Unit to get a recommendation from system for Reverse Picking.

Block Picking allowed

If block picking is allowed at all or not (see Service Qualifier "block" in Pick Mission).

Optimization

Pick Walk Setup

Pick Mission Assignment Rule

How Pick Missions are associated to a Pick Walk:

  • Urgency (only due to urgency of Pick Missions)
  • Shortest Path (with preferred optimization of shortest path selection; considering the sequence number of locations).

Max. Number of  Pick Container

Maximum number of possible Pick Containers for simultaneous processing. Used for clustering of Pick Missions for Multi Order Picking.

 INFO 

Person-to-Goods Picking only.

Pick Location Sequence Strategy

Pick route calculation depends on the configured pick face sequence across all aisles. Defines the method for considering these sequences.

  • Sequence only (Pick sequence is by the sequence number of the locations across all aisles)
  • Default (Pick sequence is by the sequence number of the Pick Order Lines).

Workload Size

Max. Number of Picks

Maximum number of Picks while processing a Pick Mission or a whole Pick Walk. If empty or 0, the number of Picks is unlimited. In case of Person-to-Goods Picking this is the number of stops or Pick Faces. For Goods-to-Person Picking, this restricts the maximum number of different Source Load Units within one Pick Mission.

Printing

Lists

List Target Name

Default target for list printing.

Picking List Type

Default Pick List Type of the Pick Zone:

  • Standard Pick list (simple textual Pick List)
  • Pick label list (list of Pick Labels ready to use for applying to each single Pick).

Number of Content Lists

Number of internal content lists to print for each Pick Container.

Labels

Label Target Name

Default target for label printing.

Pick Label Type

Default Pick Label Type of the Pick Zone:

  • Default Pick Label (The only provided Pick Label)

Number of Pick Labels

Number of Pick Labels to print for each Pick.

Print new Labels

If set, for each put Pick Container a new Label is printed automatically.
