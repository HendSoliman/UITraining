Locations

Purpose

The Location Administration screen provides all existing Locations of the system and is divided into three parts. The right parts list the Locations and the Slots. The left side provides a menu view of the warehouse structure. The user can navigate the menu and click an entry to change the content of the upper grid. Menu selection first resets the modifiers, then sets the corresponding fields of the grid modifiers and performs a select action.

 INFO 

Storage Locations (bins) basically support either discrete raster Layouts or dynamic slotting with use of Slots. Definition is done through the positioning method of the related Area.
