Dialog Overview

RF-Terminal Repacking

Functions
  • Repacking of a certain quantity of an Inventory Unit from one Load Unit to another Load Unit.
      
Dialog Overview

Dialog State

Description

Repacking Source

The dialog state "Repacking Source" is the initial action to be done. With this step the Source Load Unit is determined from which Inventory is to be repacked.

Select Inventory

In dialog state "Select Inventory" one Inventory Unit must be selected in case more than one exist on the Source Load Unit.

Repacking Target

In dialog state "Repacking Target" the Target Load Unit for the Repacking operation is determined.

Confirm Repacking

In dialog state "Confirm Repacking" the unpacking operation is confirmed (and booked) and the repacking quantity is determined and could be changed.
