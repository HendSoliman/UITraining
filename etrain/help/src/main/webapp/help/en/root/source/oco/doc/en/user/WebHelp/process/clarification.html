Clarification

Overview

Reasons for enabling an operator to clarify exceptional cases which can occur during transport of Load Units to storage locations:

  • During Putaway
  • destination bin of the Load Units Putaway transport is unexpectedly occupied with error message "bin occupied" with several reasons (e.g. Location occupied or Destination Location unreachable).
  • During Retrieval
  • source bin of the Load Units Retrieval Transport is unexpectedly free with error message "bin empty" with several reasons (e.g. Location free or Source Location unreachable).

The system supports many clarification issues. Clarification is supported by the following screens:

  • Load Unit Guide: displays basic data of a Load Unit to be clarified at a Clarification Station and provides clarification functionalities.
  • Initiate or finish a transport
  • Adjust the inventory quantity
  • Adjust the location
  • Unlock the Load Unit
  • Print Load Unit labels.

  • Clarification Bin: displays all locked Locations in error status and provides clarification functionalities.
  • Initiate inspections using clear trips
  • Control all current clear trips
  • Create UFO transports
  • Book bin to "empty"
  • Book bin to "clarified".

Description

Clarification Bin

Provides specific features for clarification of locked Locations.

Load Unit Guide

For use at Clarification Stations: provides different, mainly project- specific, information and clarification functionalities for a Load Unit with a set of specific handlers able to deal with the Load Unit and its specific situation; which depends on various predefined criteria.
