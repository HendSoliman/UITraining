New Pick Container

Description

This must be performed if additional Pick Containers are required at a Pickstation. It is the standard process when the Picking Screen is started.

      
Triggers and Actors

The following users must perform this after launching the Picking Screen and not all required Pick Containers are already booked on Put Locations and if a Pick Container is sent away and a new one is required.

  • Picker at Goods-to-Person Pickstation.
      
Prerequisites
  • Active Pick Mission assigned to the Pickstation (see Workstation Queue Entries).
  • Additional Pick Container is required for processing one of the current Pick Missions.
  • Pickstation's Put Location is available in status "free".
  • New (unknown) Pick Container is available at Pickstation.
      
Work Flow

Step

Description

1.

Open the Picking Screen.

2.

Destination Put displays.

3.

Scan the Pick Container.
"Enter" key and "OK" button activate with scan.

4.

Put the Pick Container onto the displayed Put Location.

Required scan of Put Location

2a.

If a Put Location scan is required, Destination Put with Scan displays.

5a.

Scan the required Put Location.
"Enter" key and "OK" button activate with scan.

6a.

Invalid Put Location is scanned.

7a.

Put Location Invalid Scan displays.

8a.

Proceed with step 5a.

Change Load Unit Type

3a.

Press "F10" or click "F10: Change Load Unit Type" to change the displayed Load Unit type of new Pick Container.

3b.

Select another Load Unit type. Confirm with "Enter" key or "OK" button. Or discard changes: "Esc" / "Cancel".

3c.

New Load Unit type displays for later creation of a new Pick Container. Proceed with original step.
     
Result

One or multiple new Pick Containers are booked onto the displayed Put Locations.

      
Required Resources
  • Picking Screen and in more detail the following screens:
  • Destination Put
  • Destination Put with Scan
