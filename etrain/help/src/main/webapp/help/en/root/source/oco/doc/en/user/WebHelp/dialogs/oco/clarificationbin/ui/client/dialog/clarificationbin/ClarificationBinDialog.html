Clarification Bin

Purpose

This screen is used to initiate inspection and clarification of locked Locations in error status. All Locations with an error status display. The table allows selection of Location(s) to initiate a Transport for the bins to the Clarification Station.
Transports to a Clarification Station are called "Clear Trips". All initiated Clear Trips display on the second tab in a Master-Detail view for Orders and related Order Lines. A Clear Trip Order can have one or more Order Lines.
While creating Transports, an Order Line is created for every Load Unit located on the selected locked Locations.

 INFO 

Specialty for multi-divided bins while creating UFO Transports:
For each unoccupied Location position, an UFO Load Unit is generated. When the Location Type uses more than one raster number, there are several possibilities to fill the unoccupied positions of the Location with UFO Load Units. Creating Transports for Load Units that do not match the position and size of the Load Units in the bin can cause damage to the actually stored Load Units or to the transport systems. This operation is not allowed for multi-raster bins.
For multi-raster bins with variable occupancies a special function is provided.

Special Buttons

Clarification Bin

Button

Description

Start Clarification

Initiates Clarification process for Location(s) and presents Destination Locations for selection for Clear Trips.

The only Locations listed have configured Area Access to the Locations (to be clarified) and are related to a Workstation.

 INFO 

For multi-selection, ensure that all Locations to be clarified have a valid Area Access as the first selected entry.

Start Repair process

If the bin is logically empty, clears only appropriate status fields of the Locations.

If the bin is logically not empty, must confirm an additional question prior to booking Load Units on the Location specified by the configuration topic "oco-clarifactionBin.EmptyLoadUnitLocationId".

Create UFO Transport

Presents screen to create an UFO Load Unit (unknown Load Units for transport purposes only) on the Location and an UFO Transport to the chosen Clarification Station. (See Dialog Create UFO Transport.)

 INFO 

Only available for bins in appropriate status.

Clarified

Books the Location empty. The Load Unit is booked on the Location specified by the configuration topic "oco-clarifactionBin.EmptyLoadUnitLocationId".

Clarify Multiple Position Bin
Opens a screen to define the occupancy of this bin and afterwards create UFO Load Units for this to the chosen Clarification Station.

 INFO 

It is strongly required to clarify the real occupancy of the bin manually.

Clear Trip

For button descriptions, see Orders.
