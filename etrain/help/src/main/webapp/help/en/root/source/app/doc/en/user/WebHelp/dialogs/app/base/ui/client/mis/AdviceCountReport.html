Advices

Brief Description

This KPI reports the number of advices and number of averaged advised pallets per advice in the processing queue differentiated by the dimension "advice type" and "status".

      
Screen

      
Input Parameters

Input Parameter

Description

Time Level

List information for the specified Time Level only and defines the time slice granularity which is to be displayed.

Start Time / End Time

List only data where the event time is within the specified range.

Advice Type

List information for the specified advice type only.

Status

List information for the specified status only.
     
Fields

Field

Description

Time Level

The displayed date is controlled by the selected Time Level which also defines the period of the time slice where the data is aggregated.

Advice Type

The type of the advice.

Status

The status of the advice.

Average number of advices

The average number of advices for the type and status

Average number of pallets

The average number of announced pallets
     
Default Configuration

Parameter

Value

Facts time slice

 5 min

Aggregation levels

 15 min, hour, day, week & month

Aggregation interval

 300 days
