Overview

Storage Related Resources

Purpose

This topic describes how to map a physical warehouse structure to the logical representation required by the Logistics Manager.

      
Overview

The following storage-related resources must be configured in the Logistics Manager.

 INFO 

The resource order in the following table considers dependencies. Resources should be created in this order.

 

Resource

Short Hint

Description

Domain Values

optional

Describes configuration of storage related dynamic domain values.

Load Unit Type

must be configured

Represent different physical properties, e.g. width and length of the loading device (Euro Pallet, Industry Pallet).

Layout

load defaults from demo system

Describes how load units of a specific allocation type are booked onto a location.

Location Type

must be configured

Groups locations with common physical attributes.

Pick Controller

optional

Configures how pick orders are delegated to an executing picking subsystem.

Dispo Zone

must be configured

Groups Locations with common settings for inventory selection (e.g. for picking, full load unit retrieval or replenishment).

Zone

optional

Facilities can be divided into several zones. Zones may be defined by physical or functional properties. Zones may overlap.

Storing Condition

optional

A location must be assigned to all Storing Conditions requested by (assigned to) the SKUs of the inventory to be stored.

Weight Limiter

optional

Holds the aggregated weight limits of all assigned locations.

Facility

must be configured

Top level of the hierarchy that groups locations.

Area

must be configured

Facility is divided into a number of areas based on their functions.

Aisle

must be configured

Areas may be divided to aisles. An aisle can be an aisle of a high-bay store or for example a grid of shipping lanes in a shipping area.

Level

optional

Aisles may be divided into levels.

Location

must be configured

Basic spatial unit, which can be addressed in a warehouse.

Area Access

must be configured

Represents the reachability of areas from other areas for load unit types.

Terminal

must be configured

Required if UI client has a specific business role or when equipment (e.g. weighing scale or printer) must be assigned to the terminal.

Workstation

must be configured

Working place for a user; provided with a LMS terminal and assigned to a Warehouse Location.
