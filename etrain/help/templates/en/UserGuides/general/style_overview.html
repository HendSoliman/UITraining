Dematic iQ Styles for Product Documentation

Purpose

This chapter gives an overview on the styles which are available for rendering text within Dematic IT Product Documentation.

Styles are declared in cascading style sheets for different output media.

Styles defined for Dematic IT HTML-based documentation mimic the Dematic styles available for the technical documentation.
Every style in the technical documentation has a corresponding style in the style sheet. Names between Word styles and CSS styles may, however, differ.

The table in xxx shows the correspondence between Word and CSS styles.
